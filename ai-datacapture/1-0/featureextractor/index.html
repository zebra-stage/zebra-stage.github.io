Feature Extractor

AI Data Capture SDK 1.0

Overview

The FeatureExtractor class is designed to generate a set of descriptors that are representative of items located within an image. This class provides functionalities to process both entire images and specific regions defined by bounding boxes, supporting various input formats and orientations.

Feature Extractor generates descriptors (embedding) on a given image.

Input: Model and Image

Output: Descriptor

Initialization

The Feature Extractor object is initialized via its configuration structure. There are 2 mandatory fields that need to be set:

  • MODEL_ARCHIVE_FILEPATH = fully qualified file path to the .tar or .crypt archive from which the feature extractor model is read

  • MODEL_FILENAME = file name of a localizer model (no extension)

    import com.zebra.AI.Vision.FeatureExtractor;

FeatureExtractor.Settings settings = new FeatureExtractor.Settings(MODEL_ARCHIVE_FILEPATH, MODEL_FILENAME);

// Initialize FeatureExtractor object
FeatureExtractor featureExtractor = new FeatureExtractor(settings);

Usage

Generates descriptor (embedding) on a given image.

IMAGE = Bitmap image

    float[] [descriptors](types.md#descriptor)= featureExtractor.generateSingleDescriptor(IMAGE)

IMAGE = Bitmap image

    float[][] [descriptors](types.md#descriptor)= featureExtractor.generateDescriptors(DETECTIONS, IMAGE)

FeatureExtractor() Constructor

Construct a new Feature Extractor object.

  • Method:
    • FeatureExtractor.FeatureExtractor (Settings settings) throws FileNotFoundException, IOException
  • Parameters:
    • settings - Settings to construct a Feature Extractor object.
  • Exceptions:
    • FileNotFoundException - Thrown if the model file is not found in the archive.
    • IOException - Thrown if there is an issue accessing the archive (e.g., corruption).

Dispose()

Releases all internal resources used by the feature extractor object. This method should be called manually to free resources.

  • Method: void FeatureExtractor.dispose()

generateSingleDescriptor (Bitmap bmp, int orientation)

CompletableFuture FeatureExtractor.generateSingleDescriptor (Bitmap bmp, int orientation) throws InvalidInputException

Description

Generate a descriptor for a whole image.

Parameters

  • bmp - The image to process.
  • orientation - The orientation angle of the image. The image is rotated by this angle (clockwise) before processing, and the result is rotated back (anticlockwise).

Exceptions

  • InvalidInputException

Throws InvalidInputException exception when Bitmap is passed as null or orientation is not equal to anyone of the following:

  • 0 degree
  • 90 degree
  • 180 degree
  • 270 degree

Return Value

A CompletableFuture containing a float array representing the descriptor.

generateSingleDescriptor (Bitmap bmp)

CompletableFuture FeatureExtractor.generateSingleDescriptor (Bitmap bmp) throws InvalidInputException

Description

The image to process for descriptor extraction.

Parameters

  • bmp - image to process for a descriptor.

Exceptions

  • InvalidInputException

Throws InvalidInputException exception when Bitmap is passed as null.

Return Value

A CompletableFuture containing a float array representing the descriptor of the input image.

generateDescriptors (BBox[ ] detections, Bitmap bmp, int orientation)

CompletableFuture FeatureExtractor.generateDescriptors (BBox[ ] detections, Bitmap bmp, int orientation) throws InvalidInputException

Description

Generate a set of descriptors for each of the items in an image determined by an array of bounding boxes.

Parameters

  • detections - array of bounding boxes determining the location of items in the image.
  • bmp - image where the items are present.
  • orientation - orientation angle which the image was taken from - the detection routine rotates the image by this angle (clockwise) before processing, then rotates the result (anticlockwise) to correct.

Exceptions

  • InvalidInputException

Throws InvalidInputException exception when Bitmap is passed as null or detections array is passed as null or orientation is not equal to anyone of the following:

  • 0 degree
  • 90 degree
  • 180 degree
  • 270 degree

Return Value

Array with a descriptor for each of the items defined by the bounding boxes.

generateDescriptors (BBox[ ] detections, Bitmap bmp)

CompletableFuture FeatureExtractor.generateDescriptors (BBox[ ] detections, Bitmap bmp) throws InvalidInputException
