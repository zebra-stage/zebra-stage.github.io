Text OCR Settings

AI Data Capture SDK 1.0

Overview

The Settings class is a nested class in TextOCR class and is designed to configure various parameters and settings required for performing Optical Character Recognition (OCR) using Zebra’s AI Vision technology. This class provides options to fine-tune the OCR process through various settings related to detection, recognition, and decoding.

APIs

Settings()

TextOCR.Settings.Settings ()

Description

Constructs a new Settings object with default values for all parameters.

Settings(String fn)

TextOCR.Settings.Settings (String fn)

Description

Constructs a new Settings object with a specified resource file path.

Parameters

  • model_file_location - The location of the resource file containing OCR models.

Settings(String fn, String dmn, String rmn)

TextOCR.Settings.Settings (String fn, String dmn, String rmn)

Description

Constructs a new Settings object with specified resource file, detector model name, and recognition model name.

Parameters

  • resource_filename - Location of the file with resources for the OCR (i.e. model).
  • detector_model_name - Name of the detector model in the archive file.
  • recognition_model_name - Name of the recognition model in the archive file.

TextOCR.Settings

boxThreshold

float TextOCR.Settings.boxThreshold

  • Description: Confidence threshold for the box calculation stage of detection.
  • Type: float

decoderType

int TextOCR.Settings.decoderType

  • Description: Specifies the decoding strategy used during the recognition stage.
  • Type: int

decodingCharacterConfidenceThreshold

float TextOCR.Settings.decodingCharacterConfidenceThreshold

  • Description: Minimum confidence required for recognizing individual characters during decoding.
  • Type: float

decodingMaxWordCombinations

int TextOCR.Settings.decodingMaxWordCombinations

  • Description: Maximum number of possible word outcomes that can be returned.
  • Type: int

decodingTopKCharacters

int TextOCR.Settings.decodingTopKCharacters

  • Description: Number of characters considered during TopK decoding.
  • Type: int

decodingTotalProbThreshold

float TextOCR.Settings.decodingTotalProbThreshold

  • Description: Cumulative probability threshold used in the total decoding process.
  • Type: float

decodingTopkIgnoreCutoff

int TextOCR.Settings.decodingTopkIgnoreCutoff

  • Description: Maximum number of characters considered in total decoding.
  • Type: int

detectionInferencerOptions

InferencerOptions TextOCR.Settings.detectionInferencerOptions = new InferencerOptions()

  • Description: Options for configuring the detection inferencer.

detectionModelName

String TextOCR.Settings.detectionModelName

  • Description: Name of the detector model in the OCR resource archive.
  • Type: String

flip

boolean TextOCR.Settings.flip

  • Description: If true, perform recognition twice: once in the regular orientation and once rotated by 180 degrees.
  • Type: boolean

grouping

int TextOCR.Settings.decodingTopkIgnoreCutoff

  • Description: Settings for grouping detected text elements.

heatmapThreshold

float TextOCR.Settings.heatmapThreshold

  • Description: Threshold for the heatmap stage of detection.
  • Type: float

minBoxArea

int TextOCR.Settings.minBoxArea

  • Description: Minimum area required for a detection box.
  • Type: int

minBoxSize

int TextOCR.Settings.minBoxSize

  • Description: Minimum size required for a detection box.
  • Type: int

minRatioForRotation

float TextOCR.Settings.minRatioForRotation

  • Description: Minimum height-to-width ratio for rotating word crops during recognition.
  • Type: float

recognitionInferencerOptions

InferencerOptions TextOCR.Settings.recognitionInferencerOptions = new InferencerOptions()

  • Description: Options for configuring the recognition inferencer.

recognitionModelName

String TextOCR.Settings.recognitionModelName

  • Description: Name of the recognition model within the OCR resource archive.
  • Type: string

resizeDetectorToImage

boolean TextOCR.Settings.resizeDetectorToImage

  • Description: If true, dynamically resize the detector to fit the image (not supported by SNPE).
  • Type: boolean

resourceName

String TextOCR.Settings.resourceName

  • Description: File path to the OCR resources, including models.
  • Type: string

tiling

TilerSettings TextOCR.Settings.tiling

  • Description: Settings for managing image tiling during OCR.

unclipRatio

float TextOCR.Settings.unclipRatio

  • Description: Factor for unclipping bounding boxes before the recognition stage.
