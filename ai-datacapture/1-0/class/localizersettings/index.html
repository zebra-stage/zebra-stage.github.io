Localizer Settings

AI Data Capture SDK 1.0

Overview

The Localizer.Settings class provides a comprehensive set of attributes and options for configuring the behavior of the Localizer. encapsulates the configuration parameters required for initializing the Localizer and provides settings for managing model execution, resource allocation, image processing, and result aggregation. These capabilities include adjustments for tiling, padding, model thresholds, and more, allowing for fine-tuned control over image processing and inference stages.

Methods

Settings()

Constructs a new Settings object with default configurations.

Settings (String resource_filename, String model_name, int model_thread)

Constructs a new Settings object with specific overrides.

Parameters

  • resource_filename - Location of the file with resources for the localizer (e.g. model and meta files with configuration values).
  • model_name - Name of the model in the compressed file.
  • model_thread - Number of threads that the inferencer may use to execute the model.
