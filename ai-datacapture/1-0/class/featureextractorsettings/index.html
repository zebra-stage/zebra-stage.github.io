Feature Extractor Settings

AI Data Capture SDK 1.0

Overview

The Settings class is a nested class in FeatureExtractor class and it provides configuration options for the feature extractor. It offers multiple constructors to initialize the settings with varying levels of detail, allowing flexibility in how the feature extractor is configured.

FeatureExtractor Types

Settings Class Members

Settings ()

FeatureExtractor.Settings.Settings ()

Description

Constructs a new Settings object with default values. This constructor is suitable for scenarios where default configuration is adequate.

Settings (String fn, String mn, int mt)

FeatureExtractor.Settings.Settings (String fn, String mn, int mt)

Description

Constructs a new Settings object with specified parameters. This constructor allows you to specify the file location, model name, and the number of threads, providing detailed control over the feature extractor’s execution environment.

Parameters

  • resource_filename - Location of the file with resources for the feature extractor (i.e., model).
  • detector_model_name - Name of the model in the compressed file.
  • recognition_model_name - Number of threads that the inferencer may use to execute the model.

Settings (String fn, String mn)

FeatureExtractor.Settings.Settings (String fn, String mn)

Description

Constructs a new Settings object with specified file location and model name. This constructor is useful when you need to specify the resource file and model name but do not need to change the default number of threads.

Parameters

  • resource_filename - Location of the file with resources for the feature extractor (i.e., model).
  • model_name - Name of the model in the compressed file.

Settings (String fn)

FeatureExtractor.Settings.Settings (String fn)

Description

Constructs a new Settings object with a specified file location. This constructor is ideal for scenarios where only the file location needs to be specified, and default values for other parameters are acceptable.

Parameters

  • resource_filename - Location of the file with resources for the feature extractor (i.e., model).

Settings Members Class Description

resourceName

String FeatureExtractor.Settings.resourceName

Description

Location of the file with resources for the feature extractor (i.e., model). This attribute defines where the necessary files for the feature extractor are located.

modelName

String FeatureExtractor.Settings.modelName

Description

Name of the model in the compressed file. This attribute specifies which model to use within the provided resources.

inferencerOptions

InferencerOptions FeatureExtractor.Settings.inferencerOptions = new InferencerOptions()

Description

Options for the inferencer. This attribute allows for additional configuration of the inferencer’s behavior.
