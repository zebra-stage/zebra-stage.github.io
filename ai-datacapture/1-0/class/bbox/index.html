BBox

AI Data Capture SDK 1.0

Overview

The BBox class is a core component of the com.zebra.AI.Vision package, designed to handle bounding boxes typically used in object detection tasks. Bounding boxes are rectangular boxes used to define the location of an object within an image. The BBox class encapsulates attributes such as class label (cls) and probability (prob), which are essential for identifying and evaluating detected objects.

APIs

The following is the list of APIs supported by BBox.

Return Type | API Name -----------------| -------------------- int | cls float | prob

API Detailed Description

cls

API Name

int BBox.cls

Description

The cls attribute represents the class label of the detected object. It is an integer value that corresponds to a specific category or class in the object detection model.

prob

API Name

float BBox.prob

Description

The prob attribute indicates the probability or confidence score associated with the detection of the object. It is a float value typically ranging from 0.0 to 1.0.
