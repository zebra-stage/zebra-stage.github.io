Localizer Settings

AI Data Capture SDK 1.0

Overview

The Localizer.Settings class provides a comprehensive set of attributes and options to configure the behavior of the localizer. This includes settings for tiling, padding, model thresholds, and more, allowing for fine-tuned control over the image processing and inference stages.

Localizer Types

Localizer.Settings

The Localizer.Settings class encapsulates the configuration parameters required to initialize the Localizer. It provides various settings for managing model execution, resource allocation, image processing, and result aggregation, allowing for fine-tuning of the model inference process.
