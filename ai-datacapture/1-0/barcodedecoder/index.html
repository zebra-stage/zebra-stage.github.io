Barcode Decoder

AI Data Capture SDK 1.0

Overview

The BarcodeDecoder class is designed to decode various types of barcodes detected in images. Initially, the Localizer class is used to identify the location of barcodes in captured images. The BarcodeDecoder class then decodes barcodes from entire images or specific regions defined by bounding boxes. It is capable of decoding a single barcode or multiple barcodes within a single image.

Common applications:

  • Retail and Inventory Management - Automate product identification and inventory management by decoding barcodes on shelves, labels, or packaging.
  • Mobile Applications - Integrate scanning capabilities for QR codes and other barcodes in mobile apps.
  • Logistics and Supply Chain - Track packages and shipments by decoding barcodes on boxes and pallets.
  • Healthcare - Scan barcodes on medical supplies and patient wristbands for inventory management and patient tracking.

Developer Guide

Step 1: Initialization

  1. Import the BarcodeDecoder class using com.zebra.ai.vision.BarcodeDecoder.

  2. Initialize BarcodeDecoder Settings: Create an instance of BarcodeDecoder.Settings, which will hold any configurations needed to set for the barcode decoding process.

  3. Configure Barcode Symbologies: Enable the desired barcode symbologies to recognize and set its parameters; refer to the Supported Barcode Symbologies.

  4. Initialize and load the model asynchronously - Declare a BarcodeDecoder object and use a CompletableFuture to load the model asynchronously. Ensure an Executor is available to manage asynchronous processing.

  5. Handle synchronous initialization - Implement the thenAccept() callback to use the decoder instance once it is available. 

    import com.zebra.ai.vision.BarcodeDecoder;

// Initialize BarcodeDecoder settings object
BarcodeDecoder.Settings settings = new BarcodeDecoder.Settings();

// Enable Code 128 and its parameters or other desired symbologies
settings.Symbology.CODE128.enable(true);
settings.Symbology.CODE128.setLength1(1);
settings.Symbology.CODE128.setLength2(10);

// Initialize BarcodeDecoder object
BarcodeDecoder decoder = null; 

// Executor = An executor thread in which the processing of API call happens, and the results are returned
CompletableFuture<BarcodeDecoder> futureObject = getBarcodeDecoder(settings, executor);

//Use the futureObject to implement thenAccept() callback of CompletableFuture.
futureObject.thenAccept (decoderInstance -> {
    //Use the decoder object returned here for decoding the barcodes
    decoder = decoderInstance;
});

Step 2: Decode

Select one of the following methods to decode:

Method 1: Decode Full Image

This method runs the decoder over the entire image, attempting to find and decode a barcode. Typically, this applies for decoding a single barcode.

  1. Prepare Inputs: Obtain a Bitmap image that contains the barcode. Use an Executor to manage the asynchronous decoding operation.
  2. Decode Barcode: Call the decode method on the BarcodeDecoder instance, passing the image and executor.
  3. Handle Results: Process the decoded barcode when the CompletableFuture completes.
  4. Dispose Decoder: After decoding is complete and the decoder instance is no longer needed, dispose of the decoder to release resources.

Code: // Decoding starts once the bitmap image is available. Note that the Decode API accepts Bitmap image.
// Input params: bitmap image and an executor thread object (in which the decode happens and the results are returned). CompletableFuture resultFutureObject = barcodeDecoder.decode(bitmap, executor);
resultFutureObject.thenAccept (results -> { //Process the returned result of decoded barcode. });

    // Once done with using the decoder object, dispose the object to release the resources and memory used for decoding
    decoder.dispose();

Method 2: Decode Regions of Image

This method runs the decoder over specific areas of an image, defined by bounding boxes. This is typically used when these bounding boxes are determined by a separate localization process, such as a Localizer. Typically applies when decoding multiple barcodes.

  1. Prepare Inputs: Obtain a Bitmap image containing the barcodes. Use an array of BBox (bounding boxes) indicating the regions of interest in the image where barcodes are expected. Use an Executor to manage asynchronous decoding operations.

  2. Decode Barcodes: Call the decode method on the BarcodeDecoder instance, passing the image, bounding boxes, and executor.

  3. Handle the Results: Process the decoded barcodes when the CompletableFuture completes.

  4. Dispose the Decoder: Once decoding is complete and no further use of the decoder instance is needed, dispose of the decoder to release resources.

    // Decoding starts once bounding boxes are available. 
// Input params: bitmap image, array of bounding boxes, an executor thread object (in which the decode happens and the results are returned) 
CompletableFuture<Result[]> resultFutureObject = barcodeDecoder.decode(bitmap, bboxes, executor);  
resultFutureObject.thenAccept (results -> { 
    //Process the returned result of decoded barcodes. 
});

// Once use of the decoder object is done, dispose the object to release the resources and memory used for decoding
decoder.dispose();

Methods

getBarcodeDecoder (Settings settings, Executor executor)

    CompletableFuture<BarcodeDecoder> getBarcodeDecoder (Settings settings, Executor executor) throws InvalidInputException, RuntimeException

Description: Asynchronously initializes and returns a BarcodeDecoder object.

Parameters:

  • Settings settings - Configurable options for the Barcode Decoder.
  • Executor executor - The executor used to create the BarcodeDecoder object.

Exceptions:

  • InvalidInputException - Thrown if the settings object is null.
  • IllegalStateException - Thrown if the AI Data Capture SDK is not initialized.

decode (Bitmap bmp)

    CompletableFuture<Result> BarcodeDecoder. decode (Bitmap bmp, Executor executor) throws InvalidInputException, IllegalStateException

Description: Asynchronously decodes a barcode from the provided bitmap image. Runs the decoding process in the specified executor thread.

Parameters:

  • Bitmap bmp - Image to decode.
  • Executor executor - The executor for running the decoding process.

Return Value - A CompletableFuture returing a String representing the decoded barcode.

Exceptions:

  • InvalidInputException - Thrown when the Bitmap is null.
  • IllegalStateException - Thrown if the previous decoding task is still in progress.

decode (Bitmap bmp, BBox[] detections, Executor executor)

    CompletableFuture<Result[]> BarcodeDecoder.decode (Bitmap bmp, BBox[] detections, Executor executor) throws InvalidInputException, IllegalStateException

Description: Decodes barcodes at specific locations (bounding boxes returned by Localizer) within the input image.

Parameters:

  • Bitmap bmp - Image to decode.
  • BBox[] detections - Bounding boxes indicating the barcode locations.
  • Executor executor - The executor for running the decoding process.

Return Value: A CompletableFuture returning an array of Result objects representing the decoded barcodes.

Exceptions:

  • InvalidInputException - Thrown when Bitmap or detections array is null.
  • IllegalStateException - Thrown if the previous decoding task is still in progress.

dispose()

    void dispose()

Description: Frees or releases resources. This method must be called manually to release all internal resources when a new BarcodeDecoder object is created.

BarcodeDecoder.Settings

The BarcodeDecoder.Settings class configures the barcode symbologies and related settings required for decoding barcodes from images. This class allows users to customize decoding parameters to optimize performance for specific use cases.

Class Members

Settings()

    BarcodeDecoder.Settings())

Description: Constructs a new Settings object with default values. Use this when no specific configuration is needed.

barcodeSettingMap

    Map<SymbologyType, Boolean> barcodeSettingMap = new HashMap<>();

Sample Code

Decode Barcodes from a Full Image

This sample code demonstrates how to detect and decodes barcodes from a full bitmap image:

  1. Initialize Settings Object - Create an instance of BarcodeDecoder.Settings to configure the decoder with default or custom settings.
  2. Define Executor - Use an Executor to handle asynchronous execution. This allows the decoding process to run in a separate thread, improving application responsiveness.
  3. Initialize BarcodeDecoder - Call getBarcodeDecoder() with the settings and executor to asynchronously create a BarcodeDecoder instance. This method returns a CompletableFuture.
  4. Handle CompletableFuture - Use the thenAccept() method to obtain and use the BarcodeDecoder instance once it is ready.
  5. Load Image - Load the image containing the barcodes to be decoded into a Bitmap object.
  6. Perform Decoding - Call the decode() method on the BarcodeDecoder instance, passing the Bitmap and Executor. This method returns a CompletableFuture with the decoding result.
  7. Process Results - Use the thenAccept() method on the resulting CompletableFuture to process the decoded barcode results.
  8. Dispose Resources - Call dispose() on the BarcodeDecoder instance to release resources and memory once decoding is complete.

Sample code:

    import com.zebra.ai.vision.BarcodeDecoder;
    import com.zebra.ai.vision.BarcodeDecoder.Settings;
    import android.graphics.Bitmap;

    // Initialize BarcodeDecoder settings object 
    BarcodeDecoder.Settings settings = new BarcodeDecoder.Settings();
    BarcodeDecoder  barcodeDecoder = null;

    // Initialize executor
    Executor executor = Executors.newFixedThreadPool(1);

    CompletableFuture<BarcodeDecoder> futureObject = getBarcodeDecoder(settings, executor); 

    // Use the futureObject to implement thenAccept() callback of CompletableFuture. 
    futureObject.thenAccept (decoderInstance -> { 
        //Use the decoder object returned here for the detection of barcodes 
        barcodeDecoder = decoderInstance; 
    });

    // Load your Bitmap image
    Bitmap image = ...; // Assume this is your input image

    // Perform decode on barcode
    CompletableFuture<Result> resultFutureObject = barcodeDecoder.decode(bitmap, executor);   
    resultFutureObject.thenAccept (result -> {  
        //Process the returned result (of type Result) of decoded barcodes.  
    }); 

    // Dispose resources
    // Once done using the decoder object, dispose it to release the resources and memory used for decoding.  
    barcodeDecoder.dispose();

Decode Barcodes from Regions of an Image

This sample code demonstrates how to detect barcodes using the Localizer class and decode them from specific regions of a bitmap image defined by bounding boxes.

Step 1: Barcode Detection:

  1. Set Model Path and Name: Define the path to the cryptographic model file and obtain the model name for barcode localization.
  2. Configure Localizer Settings: Create and configure a Localizer.Settings object with the model path and name. Optionally, set the input dimensions.
  3. Initialize Localizer: Asynchronously obtain a Localizer instance using a CompletableFuture.

Step 2: Barcode Decoding:

  1. Create BarcodeDecoder Settings: Instantiate a Settings object for the BarcodeDecoder and enable desired symbologies, such as UPC_A.
  2. Initialize BarcodeDecoder: Asynchronously obtain a BarcodeDecoder instance using a CompletableFuture.
  3. Detect Objects: Once a camera image is available, use the Localizer to detect objects. This method accepts a Bitmap image and returns bounding boxes of detected items.
  4. Decode Barcodes: Use the BarcodeDecoder to decode the detected barcodes from the bounding boxes.

Sample code: 

    import com.zebra.ai.vision.BarcodeDecoder;
    import com.zebra.ai.vision.BarcodeDecoder.Settings;
    import com.zebra.ai.vision.BBox;
    import android.graphics.Bitmap;
    import com.zebra.ai.vision.Localizer;

    // Initialize SDK
    AIVisionSDK.getInstance(Context).init(); // Context refers to application context object.

    // Barcode Detection
    String modelPath = "path_to_file/model.crypt";
    String modelName =  Localizer.Settings.getModelName(Localizer.Settings.MODEL_TYPE_BARCODE_LOCALIZER);
    Localizer.Settings settings = new Localizer(modelPath, modelName);

    // Optional: Set model input size parameters
    settings.inferencerOptions.defaultDims.height = 640;
    settings.inferencerOptions.defaultDims.width = 640;

    // Initialize Localizer object
    Localizer localizer = null;

    CompletableFuture<Localizer> futureObject = Localizer.getLocalizer(settings,executor);

    // Use the futureObject to implement thenAccept() callback of //CompletableFuture.
    futureObject.thenAccept ( localizerInstance -> {
        localizer = localizerInstance; // Use the localizer object for the detection of barcodes/shelves/products
    });

    // Create BarcodeDecoder object
    BarcodeDecoder barcodeDecoder = null;

    // Create settings for BarcodeDecoder
    Settings settings = new Settings();

    // Enable symbology settings    
    settings.barcodeSettingMap.put(BarcodeDecoder.Settings.SymbologyType.UPC_A, true); // Enable UPCA decoding

    // Initialize executor
    Executor executor = Executors.newFixedThreadPool(1);

    // Asynchronously get BarcodeDecoder instance
    CompletableFuture<BarcodeDecoder> futureDecoderObj = BarcodeDecoder.getBarcodeDecoder(settings, executor);

    // Use the futureObject to implement thenAccept() callback of CompletableFuture. 
    futureDecoderObj.thenAccept (decoderInstance -> { 
        barcodeDecoder = decoderInstance;  // Use the decoder object returned here for barcode detection
    });

    // Detect objects once the camera image is available
    BBox[] detections;

    // Input params: bitmap image and an executor thread object in which the detection happens
    // Results are returned in thenAccept() callback
    CompletableFuture <BBox[]> resultFutureObject = localizer.detect(Bitmap image , Executor executor); 
    resultFutureObject.thenAccept ( bboxes -> {
        detections = bboxes;  // Process the returned array of bounding boxes
    });

    // Once the bounding boxes are available, pass them to barcodeDecoder for decoding
    Bitmap image = ...; // //Load your Bitmap image here

    // Perform decode on barcode
    CompletableFuture<Result> resultFutureObject = barcodeDecoder.decode(bitmap, detections , executor);   
    resultFutureObject.thenAccept (results -> {  
        // Process the returned result (Result[]) of decoded barcodes
    }); 

    // Dispose resources
    // Once done using the decoder object, dispose it to release the resources and memory used for decoding.  
    barcodeDecoder.dispose();

Code 128

Description: Configures the Code 128 symbology, allowing for customization of parameters such as ISBT 128 handling, EAN 128 support, length restrictions, and security settings. Updates the barcodeSettingMap with Code 128 configurations.

Parameters:

  • initialStatus - Boolean to enable (true) or disable (false) Code 128. Default is true.
  • checkIsbtTable - Boolean to check the ISBT table.
  • enableEan128 - Boolean to support for EAN 128 barcodes.
  • enableIsbt128 - Boolean to support for ISBT 128 barcodes.
  • enablePlain - Boolean to decode plain Code 128 barcodes.
  • isbt128ConcatMode - Type Isbt128ContactMode to set ISBT 128 concatenation mode.
  • length1 - Integer specifying the minimum barcode length to decode.
  • length2 - Integer specifying the maximum barcode length to decode.
  • reducedQuietZone - Boolean to enable a reduced quiet zone.
  • redundancy - Boolean to enable redundancy checks for improved accuracy.
  • securityLevel - Type SecurityLevel to set the decoding security level, influencing the stringency of error checking.
  • ignoreFNC4 - Boolean to ignore FNC4 characters during decoding.

EAN8

Description: Configures parameters for EAN8 barcode decoding, including optional conversion to EAN13.

Parameters:

  • initialStatus - Boolean to enable (true) or disable (false) the symbology.
  • convertToEan13 - Boolean to convert EAN8 to EAN13.

I2OF5

Description: Configures the I2OF5 symbology and parameters such as EAN conversion, FEBRABAN handling, length restrictions, and security settings. Updates the barcodeSettingMap to include I2OF5 configurations.

Parameters:

  • initialstatus - Boolean to enable (true) or disable (false) the symbology. Default is false.

UPCA

Description: Configures parameters related to UPCA barcode decoding, including handling of the UPCA symbology.

Parameters:

  • initialStatus - Boolean to enable (true) or disable (false) the symbology.
  • reportCheckDigit - Boolean t report the check digit.
  • preamble - Integer to define preamble settings:
    • 0 - Preamble None, transmit no preamble
    • 1 - (Default) Preamble Sys Char, transmit System Character only
    • 2 - Preamble Country and Sys Char, transmit System Character and Country Code

UPCE

Description: Configures UPCE barcode decoding, including handling and conversion of UPCE symbology.

Parameters:

  • initialStatus - Boolean to enable (true) or disable (false) the symbology.
  • reportCheckDigit - Boolean to report the check digit.
  • preamble - Enum Preamble value to define the preamble settings.
  • convertToUPCA - Boolean to convert UPCE to UPCA.

UPCE1

Description: Configures UPCE1 barcode decoding, including handling and conversion settings.

Parameters:

  • initialStatus - Boolean to enable (true) or disable (false) the symbology.
  • reportCheckDigit - Boolean to report the check digit.
  • preamble - Enum Preamble value to define the preamble settings.
  • convertToUPCA - Boolean to convert UPCE1 to UPCA.
