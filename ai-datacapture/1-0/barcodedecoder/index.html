Barcode Decoder

AI Data Capture SDK 1.0

Overview

The BarcodeDecoder class is designed to decode different kinds of barcodes from image data. It provides methods to decode barcodes from Bitmap images.

BarcodeDecoder Settings

The BarcodeDecoder Settings class provides configuration settings necessary for customizing the barcode decoding process.

Initialization

    import com.zebra.AI.Vision.BarcodeDecoder;
    BarcodeDecoder.Settings settings = new BarcodeDecoder.Settings();
    // Initialize BarcodeDecoder object
    BarcodeDecoder barcodeDecoder = new BarcodeDecoder(settings);

Usage

Method 1: Decode Using Bounding Boxes

This method runs the decoder over specific areas of an image, defined by bounding boxes. This is typically used when these bounding boxes are determined by a separate localization process, such as a Localizer.

    String[] decodedBarcodes = barcodeDecoder.decode(IMAGE, DETECTIONS);
  • Parameters:
  • IMAGE - A Bitmap object representing the image from which barcodes need to be decoded.
  • DETECTIONS - A list of bounding boxes. Each bounding box is a detected region within the image that potentially contains a barcode.
  • Return Type:
    • A CompletableFuture object with an array of result types. See Result Types

Method 2: Decode Full Image

This method runs the decoder over the entire image, attempting to find and decode a barcode.

    String decodedBarcode = barcodeDecoder.decode(IMAGE);

  • Parameters:

    • IMAGE - A Bitmap object representing the full image to be scanned for a barcode.

  • Return Type:

    • A CompletableFuture of result.

APIs

BarcodeDecoder

Initializes a new instance of the BarcodeDecoder class using the provided settings. This constructor loads necessary resources and configurations for barcode decoding.

NOTE: Call the dispose() API for every object of BarcodeDecoder when it is no longer needed.

  • Constructor:
    • BarcodeDecoder (Settings settings) throws FileNotFoundException, IOException
  • Parameters:
    • Settings - Configurable options for the Barcode Decoder object.
  • Exceptions:
    • FileNotFoundException - Thrown if required resource files are missing or cannot be located.
    • IOException - Thrown if the archive is corrupted.

decode (Bitmap bmp)

Runs barcode decoding on the entire input Bitmap image.

  • Method:
    • CompletableFuture<Result> BarcodeDecoder.decode (Bitmap bmp) throws InvalidInputException
  • Parameters:
    • bmp (Bitmap bmp) - Image to decode
  • Return Value:
    • CompletableFuture with String as output. A string representing the decoded barcode.
  • Exceptions:
    • InvalidInputException - Thrown when Bitmap is passed as null.

decode (Bitmap bmp, BBox [ ] detections)

Runs barcode decoding on specific locations (Bounding Box returned by localizer) within the input image.

  • Method:
    • CompletableFuture<Result[ ]> BarcodeDecoder.decode (Bitmap bmp, BBox[ ] detections) throws InvalidInputException
  • Parameters:
    • bmp(Bitmap bmp) - Image to decode
    • detections - Bounding boxes indicating the barcode locations
  • Return Value:
    • A CompletableFuture with Result[] array representing the decoded barcodes.
  • Exceptions:
    • InvalidInputException - Thrown when Bitmap or detections array are passed as null.

dispose()

Frees or releases resources.

  • Constructor:
    • BarcodeDecoder.dispose()

NOTE: Needs to be called manually to release all internal resources. Call dispose() when the new object of BarcodeDecoder is created.

BarcodeDecoder Types

BarcodeDecoder.Result

The BarcodeDecoder.Result class provides a comprehensive representation of the decoded barcode data, including the symbology type and the bounding box coordinates.

Detailed Description

The BarcodeDecoder.Result class encapsulates the result of a barcode decoding operation. It provides information on the decoded data value, the type of symbology used, and the bounding box data that indicates the position and dimensions of the barcode within an image.

Result Class Member Description

symbologytype

int BarcodeDecoder.Result.symbologytype

Description

This integer member represents the type of symbology that the decoded barcode corresponds to. The symbology type is essential for understanding the format of the barcode, such as QR code, UPC, EAN, etc. Each symbology type is typically associated with a specific integer value that can be cross-referenced with a symbology type enumeration or reference guide.

value

String BarcodeDecoder.Result.value

Description

This string member contains the decoded value of the barcode. It is the actual data encoded within the barcode, such as a product number, URL, or any other encoded information. This value is extracted during the decoding process and is the primary output of the barcode decoding operation.

bboxData

BBox BarcodeDecoder.Result.bboxData

Description

This member holds the bounding box data of the decoded barcode. The BBox class typically contains information about the position and dimensions of the barcode in the image, such as coordinates of the bounding box corners. This is useful for identifying the exact location of the barcode within a scanned image.

BarcodeDecoder Types

BarcodeDecoder.Settings

The BarcodeDecoder.Settings class provides the necessary configuration options required for decoding barcodes using the Zebra AI Vision Barcode Decoder. This class allows users to customize decoding parameters to optimize performance for specific use cases.

Description

The BarcodeDecoder.Settings class is essential for configuring the parameters used in barcode decoding tasks. It provides flexibility to adjust settings such as the types of barcodes to decode, additional margin for detection accuracy, and the resource file location for configuration overrides.

Settings Class Member Description

Settings()

BarcodeDecoder.Settings.Settings ()

Description

Constructs a new Settings object with default values. Use this when no specific configuration is needed.

Settings(String fn)

BarcodeDecoder.Settings.Settings (String fn)

Description

Construct a new Settings object with overrides.

Parameters

  • resource_filename - A string specifying the location of the resource file for custom configuration.

Settings Members

resourceName

Optional BarcodeDecoder.Settings.resourceName

Description

Represents the file path to the configuration resources for the decoder. This is optional and used for loading specific configuration settings from a file.

barcodeTypes

int BarcodeDecoder.Settings.barcodeTypes

Description

Specifies the types of barcodes the decoder will attempt to decode.

extraMarginPercent

int BarcodeDecoder.Settings.extraMarginPercent

Description

Increase bbox dimensions by a given percentage, which usually improves results (default 10%)
