Localizer

AI Data Capture SDK 1.0

Overview

The Localizer class is designed to detect and localize objects within images. It processes input images and outputs bounding boxes that identify the locations of detected objects or barcodes. This capability is particularly useful for applications that require object recognition and localization within an image.

Configuration of the Localizer class is facilitated by the Settings class, allowing for specification and customization of model parameters.

Supported detection models:

  • Barcode Localizer
  • Shelf Localizer


Bounding Box Explained

A Bounding Box is a rectangular frame that encloses a detected object within an image, defining its position and dimensions. Each bounding box provides:

  • Coordinates - Specifies the object's location, width and height.
  • Probability (Confidence) - Represents the confidence score of the detection.
  • Class - Displays the label of the detected object.

Barcode Localizer

The Barcode Localizer detects barcodes in images. Supported barcode types are: linear barcode (e.g. UPC nad EAN), 2D barcode, Data Matrix, QR Code, Aztec, PDF417, DotCode, postal code, MaxiCode, Composite Code, Han Xin Code, MicroPDF417, Micro QR Code, GridMatrix, and ArUco. After detection, the Barcode Decoder can be used to decode the barcodes.

Input Parameters:

  • The input size can be changed during runtime initialization. Default input size: 640x640
  • Larger sizes improve detection of smaller barcodes but increase processing time. This achieves better results for images with smaller more distant barcodes.
  • Smaller sizes enhance speed but may reduce accuracy. This achieves better results for images with larger barcodes.

Output Parameters: Bounding boxes for the detected barcodes.

Shelf Localizer

The Shelf Localizer model detects and identifies objects on retail shelves, including products, shelf labels, peg labels, and the shelves themselves.

Input Parameters:

  • Input size can be adjusted during initialization based on the proximity of the camera to the shelf.
  • Smaller sizes are suitable for close-up images, capturing details of partial shelves and providing more precise detection of smaller objects and labels. Default size: 192x192 RBG image.
  • Larger sizes capture full shelves from a distance, improving accuracy but increasing inference time. Default size: 832x832 RGB image.

Output Parameters: Bounding boxes for the detected objects.

Initialization

To initialize a Localizer object, specify the following mandatory fields:

  • MODEL_FILEPATH - Fully qualified file path to the to the model file, e.g., /folder1/file.tar.crypt
  • MODEL_FILENAME - String representing the localizer model:
    • model for Barcode Localizer.
    • localizer for Shelf Localizer.

Optional configuration of model input size (height and width) is available in inferencerOptions. Sizes must be powers of two (e.g., 640). Adjusting the input size impacts both accuracy and inference time.

  • Smaller input sizes execute faster but may reduce accuracy

  • Larger input sizes may improve accuracy particularly when detecting smaller or distant objects, but take longer to inference. Excessively large input sizes may lead to out-of-memory errors, potentially causing the application to crash at run-time. 

    import com.zebra.ai.vision.Localizer;

// Initialize settings with mandatory fields

// For Barcode Detection
Localizer.Settings settings = new Localizer.Settings("path/to/model.crypt", "model");

// For Product Detection
Localizer.Settings settings = new Localizer.Settings("path/to/model.crypt", "localizer"); 

// Optional model input size parameters
settings.inferencerOptions.defaultDims.height = 640;
settings.inferencerOptions.defaultDims.width = 640;

// Initialize Localizer object
Localizer localizer = new Localizer(settings);

// Detect objects on a given image
Bitmap image = //... obtain Bitmap image
CompletableFuture <BBox[]> future = localizer.detect(image); 
// Return type is com.zebra.ai.vision.BBox

For more advanced options, see Settings class.

Usage

Use Localizer to detect objects in an image and obtain bounding boxes for each detected object.

    CompletableFuture <BBox[]> future = localizer.detect(IMAGE); // Return type is com.zebra.AI.Vision.BBox

Methods

Constructor

Localizer (Settings settings) throws FileNotFoundException, IOException

Description: Creates a new Localizer object with specified settings and loads the detection model.

Parameters:

  • settings (Settings) - Configuration settings for initialization.

Exceptions:

  • FileNotFoundException if the model file is not found.
  • IOException if the model is corrupted.


detect (Bitmap bmp, int orientation)

CompletableFuture<BBox> detect (Bitmap bmp, int orientation) throws InvalidInputException

Description: Runs the model on an input image to calculate bounding boxes, with optional image rotation for processing. Optionally including a depth map. The total number of supplied channels (primary image + depth) must match the model's input channel requirements.

Parameters:

  • bmp (Bitmap) - Input image for detection.
  • orientation (int) - Orientation angle for image rotation during processing. The image rotates clockwise by this angle for processing, then applies an anticlockwise rotation to the results for correction. Values?: Orientation.DEG_0, Orientation.DEG_90, Orientation.DEG_180, Orientation.DEG_270.

Return Value: CompletableFuture<BBox[]> with detected objects, locations, probabilities and classes.

Exceptions:

  • InvalidInputException for invalid Bitmap (e.g., null) or orientation.


detect (Bitmap bmp)

CompletableFuture<BBox> detect (Bitmap bmp) throws InvalidInputException

Description: Runs the model on an input image to calculate bounding boxes, with optional image rotation for processing. Optionally including a depth map. The total number of input channels (primary image + depth) must match the model's required input channels.

  • bmp (Bitmap) - Input image for detection.

Return Value: CompletableFuture <BBox[ ]> with detected objects, locations, probabilities and classes.
