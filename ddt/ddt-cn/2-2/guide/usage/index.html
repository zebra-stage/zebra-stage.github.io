使用设备诊断工具

Device Diagnostic Tool 2.2

概述

设备诊断工具支持两种操作模式：

  1. 管理员模式 - 管理员可以访问应用程序设置，修改测试配置并运行每个单独的测试。这是默认模式。
  2. 用户模式 - 用户仅限于运行测试；无法访问应用设置或测试配置。要设置用户模式，请禁用 admin_mode 请参考 配置 部分。

启动应用程序

使用以下一种方法运行设备诊断工具：

  1. 使用用户界面手动操作 - 用户启动设备诊断工具并手动运行测试。请参考 用户界面 部分。
  2. 使用EMM（企业移动管理）系统远程进行 - 在后台启动设备诊断工具，并使用以下命令生成日志:
    adb shell am broadcast -n "com.symbol.selfdiagnostics/com.symbol.selfdiagnostics.SESReceiver"

仅应使用一种方法来运行设备诊断工具。如果同时使用两种方法，则可能会发生意外行为。

用户界面

首次启动应用程序时，主屏幕显示可运行的测试： 主屏幕
如果已经进行过测试，则在进行了最后一项测试时，在测试名称后面加上测试日期。

在主屏幕上点击 运行测试 以执行所有测试。测试执行基于在 配置测试 屏幕中选择的测试。 启动测试后，将提示用户在以下各项的测试执行过程中执行其他操作：扫描器测试、按键测试、触摸屏测试和音频测试。建议在设备正常使用时执行所有测试，即不要将其放在支架上或通过USB连接到计算机。测试完成后，将显示结果:
   合格 - 测试已执行并通过。结果符合测试标准。
   不合格 - 测试已执行且不符合测试标准。

   信息 - 检索并显示的数据。

或者，在管理员模式下，可以通过点击各个测试类别然后点击 运行测试来执行。
测试失败 和 测试超时 结果显示为相同的红色危险图标。

扫描器测试

提示用户扫描条形码。结果：

  • 扫描器测试 - 显示条形码数据
  • 标签类型 - 显示条形码类型或扫描的解码器

注： 在TC55上执行扫描器测试时，用户必须长按扫描器按键才能对其进行检测。

按键测试

提示用户按设备上的物理按键：扫描键（左侧或右侧）、push-to-talk按键、音量增大按键和音量减小按键。结果：

  • 按键测试 - 测试成功， 测试失败，或 测试超时

注： 如果重新映射了物理按键，则按键测试将无法正常工作。

触摸屏测试

提示用户触摸屏幕上的每个网格框

  • 触摸屏测试 - 测试成功， 测试失败，或 测试超时

蓝牙测试

检查蓝牙无线电是否可用并返回蓝牙相关信息。结果：

  • 名称 - 显示蓝牙名称
  • 无线电功率循环 - 测试成功 或 测试失败。在进行此项测试之前，已保存无线电的状态。
  • 功能正常/非正常 - 功能正常 或 功能不正常
  • 可发现/可连接 - 可连接, 可发现, 或 无

Wifi测试

检查WiFi无线电的操作并返回WiFi相关信息。结果：

  • MAC地址 - 有效 或 无效
  • 网络测试 - 显示 已连接 或 未连接 。如果WiFi未连接到任何网络，则会发生故障。在进行此项测试之前，已保存无线电的状态。如果测试成功，则显示以下值:
    • 强度 - 显示信号强度
    • Essid - 显示ESSID； 在Android O及更高版本上， 由于安卓的限制，必须在设备设置中启用该位置。如果未启用设备位置，则返回值为 未知的ssid。
    • ip - 显示IP地址
    • bssid - 有效 或 无效
    • 速度 - 显示连接速度
  • 无线电功率循环 - 测试成功 或 测试失败

电池测试

检查电池状态并返回电池相关信息。结果：

  • 部件编号 - 显示部件编号
  • 序列号 - 显示序列号
  • 生产日期 - 显示电池的生产日期
  • 停用状态 - 显示电池的运行状况为：
    • 良好 - 电池处于良好运行状态
    • 停用 - 电池已达到指定的阈值（当前为400个充电周期），表明应尽快更换新电池。
    • 未知 - 表明检索电池状况信息时出现问题
  • 电压 - 显示电压
  • 电流 - 显示电流
  • 温度 - 显示温度

WWAN（无线广域网）测试

检查WWAN无线电的操作并返回相关的WWAN信息。结果：

  • Sim 状态 - 飞行模式必须禁用。显示以下一个值：
    • 存在 - SIM卡存在
    • 不存在 - SIM卡不存在
  • 语音状态 - 显示以下一个值：
    • 语言在使用中
    • 语音服务中断
    • 仅限紧急情况使用语音
    • 语音电源关闭
    • 不明语音
  • 数据状态 - 显示以下一个值：
    • 数据已连接
    • 数据断开
    • 数据连接中
    • 数据挂起
    • 未知数据
  • WAN类型 - 显示网络类型，例如: LTE，2G，3G，4G 或 无法使用
  • 信号强度 - 显示以下之一：
    • 不适用 - 没有SIM卡
    • 未知 - 设备可能处于飞行模式
    • 信号强度 - 以dBm（分贝毫瓦）为单位
  • 电话号码 - 有效 或 无效，具体取决于服务提供商是否公开了电话号码。
  • 设备ID（标识号） - 有效 或 无效，具体取决于是否公开了设备ID（标识号）

音频测试

检查设备麦克风和扬声器是否正常运行。结果：

  • 音频测试 - 测试成功 或 测试失败
