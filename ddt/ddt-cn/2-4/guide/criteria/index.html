测试标准

设备诊断工具 2.4 版本

概述

设备诊断工具根据特定条件测试设备硬件功能的运行情况，并得出以下结果：

  • 合格 - 如果符合条件
  • 不符合 - 如果不符合条件
  • 信息 - 显示信息性数据（合格/不合格不适用）
测试 需要用户干预 属性/操作 合格/不合格标准
扫描器 是 扫描条形码 成功扫描条形码时为合格。
按钮 是 按下按钮：
  - 扫描触发器(向左或向右)
  - 一键通
  - 调高音量
  - 调低音量		 当每个按钮都能检测到按钮操作时为合格。
触摸屏 是 触控网格 在屏幕上的每个网格框中都检测到触控时为合格。
蓝牙 无 返回信息
  - 名称
  - 无线电功率循环
  - 功能正常/
非正常
  - 可连接/
可发现		 如果蓝牙无线电在电源恢复时持续存在并成功返回信息，则为合格。
WLAN（无线局域网） 无 返回信息：
  - MAC 地址
  - 网络测试
  - 信号强度 (i)
  - IP 地址 (i)
  - BSSID (i)
  - 速度 (i)		 在网络测试成功时通过。
WWAN 无 返回信息：
  - Sim 状态
  - 语音状态
  - 数据状态
  - WAN 类型 (i)
  - 信号强度 (i)
  - 电话号码 (i)
  - 设备 ID (i)		 存在 SIM 卡时即可合格。在非 WAN 设备上自动合格。
电池 无 返回信息：
  - 部件编号
  - 序列号
  - 生产日期
  - 停用状态
  - 电压 (i)
  - 电流 (i)
  - 温度 (i)
  - 电量 (i)
  - 电流容量 (i)		 如果电池运行状况良好并且电池信息可以成功检索则为合格。注：电流容量仅适用于 PowerPrecision+ 电池。
音频 是 录音并播放文件 需要用户手动干预。用户手动录制音频，然后收听录音。测试是否通过取决于用户的判断。此测试应使用设备扬声器 - 不允许使用耳机。此测试应在背景噪音最小的环境中进行，以成功进行测试。回放期间将设备设置为最大音量以获得最佳效果。
注意：如果设备没有麦克风，例如 MC18，则无法进行测试。


红色文字表示未显示的敏感信息。而显示“有效”或“无效”是为了表明是否以正确的格式检测到信息。“有效”表示已检索到适当的值。“无效”表示检索到的值为零、空或与预期的 MAC 地址模式不匹配（仅适用于 MAC 地址）。

(i) 表示被视为额外数据的信息数据，在 configuration.xml 文件部分中 show_extra_data 都设置为“真”时显示。

关于每项测试的说明，请参见配置测试部分。
