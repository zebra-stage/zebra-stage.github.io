    • 3. Tips and Advice

    1. As a best practice, it is recommended to not make calls to the API from the UI thread.
    2. Use the command line for quick printer utilities without having to write an entire application.

    4. Known Issues

    1. The iMZ unit ships in a line mode configuration. Because of this, the SDK will identify the unit as "Not a Link-OS printer". To force the SDK to identify the unit as a Link-OS printer, use the device.languages command to set the iMZ to "hybrid_xml_zpl" mode.
    2. When using the Link-OS Multiplatform SDK Command Line Tool to set the weblink information on a printer, if the printer is connected via TCP using the DNS name of the printer, then an error may be indicated when in fact the printers weblink information was configured properly. To avoid this issue connect to the printer using the IP Address.
