Overview

Browser Print allows cloud-based applications to print via a browser to local network or USB connected printers.

Share this page link via email.


Requirements

  • Browser Print installed on your client computer.
    • Windows 7, Windows 10 or MAC OS X
  • One of the supported browsers:
    • Chrome v56 or later
    • Internet Explorer v11 or later
    • Safari

Instructions

  1. Download and install Browser Print on your computer.
  2. Add a network or USB connected printer in Browser Print.

How to Access the Demo

Click the link below. You may be prompted to add the site to the Browser Print list of accepted hosts.

http://weblink.westeurope.cloudapp.azure.com/BrowserPrint/Sample/
