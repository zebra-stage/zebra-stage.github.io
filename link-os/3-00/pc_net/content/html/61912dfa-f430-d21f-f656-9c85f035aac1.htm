Setting Properties

The Setting type exposes the following members.

Properties
  Name Description
Access Gets or sets a string that describes the access permissions for the setting. RW is returned for settings that have read and write permissions. R is returned for settings that are read-only W is returned for settings that are write-only
Archive Gets or sets if this setting can be applied when loading a backup
Clone Gets or sets if this setting can be applied when loading a profile
IsReadOnly Returns true if the setting does not have write access.
IsWriteOnly Returns true if the setting does not have read access.
Range Gets or sets a string that describes the acceptable range of values for this setting.
Type Gets or sets a string describing the data type of the setting.
Value Gets or sets the setting's value.
