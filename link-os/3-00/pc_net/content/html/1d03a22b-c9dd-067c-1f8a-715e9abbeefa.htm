Remarks
Will retrieve the following file extensions: .FNT, .ZPL, .GRF, .DAT, .BAS, .STO, .PNG, .LBL, .TTF, .PCX, .BMP, .IMG, .TTE, .WML, .CSV, .HTM, .BAE, .TXT.
See ConnectionBuilder for the format of connectionString.
