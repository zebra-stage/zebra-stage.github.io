Remarks
This constructor will use the specified timeouts for Read for both channels. The timeout is a maximum of maxTimeoutForRead milliseconds for any data to be received. If no more data is available after timeToWaitForMoreData milliseconds the read operation is assumed to be complete. If you wish to specify different timeouts for each channel, use MultichannelBluetoothConnection(String, Int32, Int32, Int32, Int32)}.
