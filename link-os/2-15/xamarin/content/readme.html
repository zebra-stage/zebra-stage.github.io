Link-OS Multiplatform SDK
(for Windows .NET PCs)

Release Notes

PC (.NET)

Build: 2.15.2634

1. Installation

The SDK is supported on the following devices:

  • Zebra™ ET5 (Windows™)
  • Lenovo™ Windows™ 10 Tablet
  • Various Windows™ 10 Desktop Machines
  • Android devices running version 4.4 (API 19) or higher
  • iOS devices running version 8 or higher

The SDK and Zebra Utilities are supported on the following printers:

  • Mobile - iMZ™, QLn™ series, RW™ series, P4T™/RP4T™, ZQ500 series, ZQ300 series
  • Desktop - G-Series™, HC100™, TLP/LP 2824 Plus™, ZD400 series, ZD500 series, ZD600 series
  • High-Performance/Midrange - Xi4™, PAX4™ series, ZM400™/ZM600™, ZT200 series, ZT400 series, ZT500/ZT600 series, ZE500 series.
  • RFID - R110PAX4™, RZ400™/RZ600™, ZD500R series, ZT400R series, ZE500R series.
  • Kiosk - KR403™.

2. Requirements

  • Microsoft™ .NET Framework 4.7 (or higher)
  • Microsoft™ Visual Studio 2017 (or higher)

3. Known Issues

  1. When connected via a DriverPrinterConnection, the Open() call will not immediately fail if the printer is offline/unreachable. An error will be thrown when attempting to read/write data.
  2. DriverPrinterConnection performance/throughput may be poor. Zebra recommends using a UsbConnection where possible to avoid this issue.
  3. When using PrintStation or other Developer Demos on Windows tablet, the use of a mouse is recommended to ensure user has full access to all features
  4. Bluetooth connections may become unstable after a failed read/write operation, or following a command which triggers a reboot. If this problem persists, it may require the user to unpair/repair the printer in the device settings.
