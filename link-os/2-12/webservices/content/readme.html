1. Installation

The SDK is supported on the following devices:

  • PC operating systems using Java Runtime Environment (JRE) v1.6 and 1.7

The SDK and Zebra Utilities are supported on the following printers:

  • Mobile - MZ™, iMZ™, QL Plus™, QLn™ series, RW™ series, P4T™/RP4T™, ZQ500 series.
  • Desktop - G-Series™, HC100™, TLP/LP 2824 Plus™, ZD500 series.
  • High-Performance/Midrange - Xi4™, XiIIIPlus™, PAX4™ series, ZM400™/ZM600™, S4M™, 105SL™, ZT200 series, ZT400 series, ZE500 series.
  • RFID - RXi™, R110PAX4™, RZ400™/RZ600™, ZD500R series, ZT400R series, ZE500R series.
  • Kiosk - KR403™.

2. Requirements

  • JRE v1.6 and 1.7

3. Tips and Advice

  1. As a best practice, it is recommended to not make calls to the API from the UI thread.
  2. Use the command line for quick printer utilities without having to write an entire application.

4. Known Issues

  1. The iMZ unit ships in a line mode configuration. Because of this, the SDK will identify the unit as "Not a Link-OS printer". To force the SDK to identify the unit as a Link-OS printer, use the device.languages command to set the iMZ to "hybrid_xml_zpl" mode.
