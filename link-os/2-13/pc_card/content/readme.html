Release Notes

Desktop Java - Card

Build: 2.12.3968

1. Installation

The SDK is supported on the following devices:

  • PC operating systems using Java Runtime Environment (JRE) v1.6, v1.7, or v1.8

The SDK is supported on the following printers:

  • Card - ZXP™ Series 1, ZXP™ Series 3, ZXP™ Series 7, ZXP™ Series 8, ZXP™ Series 9

3. Tips and Advice

  1. As a best practice, it is recommended to not make calls to the API from the UI thread.
  2. Use the command line for quick printer utilities without having to write an entire application.
