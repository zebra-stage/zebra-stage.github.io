                                        <p></p><h2 id="overview" class="anchor"><a class="heading-anchor" href="#overview"><span></span></a>Overview</h2>

<p><strong>Managed configurations are part of a specification developed by Google</strong> and the Android community for remotely configuring installed applications and devices through any Enterprise Mobility Management (EMM) system that supports it. Originally known as "App Restrictions" because of their limited initial scope, managed configurations (MCs) are now being used by Zebra to remotely configure a variety of settings, including those for device hardware, software and security. </p>

<p><strong>The features of a given app that are manageable using Managed Configurations are defined in its <em>schema</em></strong>. The Identity Guardian schema is downloaded when visiting Google Play through the EMM console and selecting Identity Guardian for administration. The schema defines the features available for consumption by the EMM, and provides the information necessary to present the app's management UI within the EMM console.† This data-driven UI method allows delivery of new features and their corresponding UI attributes as soon as they become available, and without the need to download a new <code>.exe</code>. For more information about configuring Zebra devices with managed configurations, see <a href="/oemconfig">Zebra OEMConfig</a>. </p>

<p><i><font size="2" color="DimGrey"> † The Identity Guardian management UI varies slightly depending on the EMM system in use.</font></i> </p>

<p>Identity Guardian offers multiple setting categories, each with a corresponding bundle as described in the table in each subsection below. Specific parameters for each configuration are contained in the tables that follow. These settings specify the method of use (shared device vs personally assigned), device enrollment, and user authentication. User authentication specifies the comparison source (e.g. barcode, facial biomaetrics) and authentication methods (e.g. facial biometrics, PIN).</p>
