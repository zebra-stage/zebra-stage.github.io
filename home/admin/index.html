Administration & Design

Manage device security, configure and deploy device settings and software to a fleet of Zebra devices, regardless of location.

Android 11+ Migration

Moving to Android 11, 13 or beyond? There are changes to the platforms that run most Zebra devices. These guides explain what's changing and why, when each change is happening and how to adapt.

App Signature Tools

A command-line utility to get a DER file or Base64 string for a certificate from an APK, useful for app certification in Zebra Access Manager.

Data Encryption Tool

A tool to encrypt sensitive data shared between apps.

Device Central

A centralized solution for tracking Bluetooth peripherals, pairing and unpairing, and viewing detailed peripheral information across an enterprise.

Go to section:

Device Diagnostic Tool

Instantly test and diagnose device system operability to determine system health.

Device Licensing

Learn about what is included with Zebra Professional and Enterprise devices and how to purchase an MDNA Enterprise license for access to Enterprise-grade features.

Go to section:

Device Tracker

A centralized cloud-based solution to track, find and prevent misplaced devices within a facility.


Go to section:

Enterprise Home Screen

A secure Android launcher that gives admins a simple UI for controlling user access to apps, settings and files on Android devices without the need for custom code.

Go to section:

LifeGuard

Delivers OS updates and security patches, which include fixes from Zebra and Google, to Zebra Android devices.


Go to section:

MX

Zebra’s Mobility DNA - an added layer of enterprise-class security and manageability for Android devices.


Go to section:

OEMConfig

An administrative tool for configuring Zebra Android devices using Android Managed Configurations.


Go to section:

RxLogger

A GUI-based tool for capturing log information from system and application events on Zebra mobile computing devices.


Go to section:

StageNow

A desktop tool that provides an easy way to stage, set up and mass-deploy Zebra's Android-based mobile computers.

Go to section:

USB Host Manager

An Android app for Zebra devices that allows administrators to create rules for controlling which USB peripherals can be used on the device.


Go to section:

Wireless Insights

Enriches our ecosystem providers' dashboard by providing device analytics and voice data, giving insight over the entire infrastructure to proactively identify issues and uncover root causes.

Go to section:

Zebra Data Service

A guide to the Zebra Data Service, a background service running on all Zebra devices that collects and uploads analytics data.


Go to section:

Zebra DNA Cloud

An administrative solution for managing and configuring Zebra devices and apps. Similar to Zebra StageNow, it leverages the power of Zebra MX and can work with an Enterprise Mobility Management (EMM) system.
