WWC Setup

Wireless Workstation Connect 1.0

Overview

Zebra Wireless Workstation Connect (WWC) can be installed on any Zebra device running Android, and is required only for displaying the sender's device screen on the monitor connected to the receiving device and using its other connected peripherals.

Requirements

Hardware

  • Receiver(s):
    • One or more Zebra ZEC500-series devices to "receive" casting session(s) sent from mobile handhelds or tablets
    • Peripherals: External monitor (connected to ZEC500); optional keyboard and mouse
  • Sender(s):
    • One or more supported Zebra mobile handhelds or tablets running Android 14 (or later) to "send" a casting session request
    • Wireless Workstation Connect app downloaded from Zebra Support Portal or Google Play installed on sending device †
    • Optional: Zebra Workstation Connect app installed on sending device if discrete desktop is desired on receiver screen. Otherwise, receiver mirrors sender.

Configuration

  • Receiver (ZEC500):
    • P2P enabled for casting (disabled by default).
      To enable P2P Casting: Go to Settings > Connected Devices > Connection preferences > Cast receiver > Use cast receiver
      For StageNow options, see Configuration Processes below
    • DHCP option 119* (Domain Search) enabled (enabled by default)
  • Sender (handheld or tablet):
    • Operation of the auto dis/connect feature requires "Bluetooth Scanning" to be enabled (device defaults vary).
      See Settings > Location > Location Services on the device.

IMPORTANT: DO NOT install the Wireless Workstation Connect app on the ZEC500.

* If enabling DHCP Option 119 is required, it must be set using Android APIs on a device in managed or device-owner mode when configured. Also, Option 119 cannot be applied to SSIDs configured using the Zebra Wi-Fi Manager CSP (component of MX).

Configuration Processes

Receiver Process 1

Use StageNow to configure a ZEC500 as a casting receiver:

  1. Open Zebra StageNow and create a Profile that uses the MX Wi-Fi CSP to Turn on Wi-Fi Casting (off by default).
  2. Follow the StageNow instructions to the Publish section and output the resulting barcode.
  3. Attach the ZEC500 to a USB or Bluetooth scanner.
  4. On the ZEC500, locate and run the StageNow app and scan the barcode created in Step 2 to initiate staging.

Upon successful staging, the ZEC500 is ready to receive a casting session from a supported Zebra device running WWC.

Receiver Process 2

Receiver Process 3

Use this process to manually configure a ZEC500 as a casting receiver from a second device without the use of a scanner.

Configure a ZEC500 for casting without using a scanner:

  1. Download the .bin file to a PC, laptop, mobile device, etc.
  2. Transfer the downloaded .bin file to the ZEC500 using Android Quick Share, USB transfer or any available means.
  3. Run StageNow on the ZEC500.
  4. Tap the "BROWSE" button in StageNow.
  5. Navigate to the .bin file and select it to initiate staging.

Upon successful staging, the ZEC500 is ready to receive.

Sender Process 1

To configure a mobile device as a casting sender:

  1. Verify that the target "sending" device is among the supported Zebra devices and is running Android 14 (or later).
  2. Download and install the Wireless Workstation Connect app from the Zebra Support Portal or Google Play. †
  3. Open WWC app on mobile device and tap "Show Nearby Devices" button. A list of nearby casting receivers appears.
  4. Tap "Connect" for the desired receiver to cast contents of mobile device to receiver's monitor.

The device begins casting in Mirror Mode.

Sender Process 2

To configure a mobile device for casting in Desktop Mode:

  1. Tap "Disconnect" button in the WWC app on mobile device (if connected).
  2. Download and install Zebra Workstation Connect‡, following all prompts. ZWC delivers Android Desktop Mode capability.
  3. Open the WWC app on the mobile device and tap "Show Nearby Devices" button. A list of nearby casting receivers appears.
  4. Tap "Connect" for the desired receiver to cast contents of mobile device to the receiver's monitor.

The device begins casting in Desktop Mode.

Revert to Mirror Mode

To switch back to Mirror Mode from Desktop Mode:

  1. Tap "Disconnect" button in the WWC app on mobile device (if connected).
  2. Uninstall the Zebra Workstation Connect app from the device.
  3. On the device, go to Settings > System > Developer options > Force desktop mode and tap to switch toggle off.
  4. Launch the WWC app and connect to a device as in Steps 3 and 4, above.

The device begins casting in Mirror Mode.

† When installing from Google Play, some apps display the message: "This app won't work for your device." Such apps must be installed from outside the Google Play infrastructure.

‡ ZWC on Zebra Professional Series devices** requires purchase of an MDNA Enterprise License.
