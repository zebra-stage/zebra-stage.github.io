Enterprise Keyboard 配置

DataWedge 7.4

概述

Enterprise Keyboard 配置允许在 DataWedge 配置文件中将 Enterprise Keyboard（无论是标准 [默认] 布局还是自定义布局）用作与应用程序关联的 IME（输入方法编辑器），而无需修改该应用程序。这样，Enterprise Keyboard 就可以显示为屏幕上的键盘，供用户输入文本.Enterprise Keyboard 需在 Android 设置中列为虚拟键盘才能使用。使用 Keyboard Designer 工具可以生成自定义的 Enterprise Keyboard。

图 1.Enterprise Keyboard - 标准（默认）布局

Enterprise Keyboard 配置可以从 DataWedge 配置文件屏幕访问：


如果存在以下情况之一，则无法访问（灰显）和配置 Enterprise Keyboard 配置：

  • Enterprise Keyboard 不是设备上 Android 系统中的默认 IME。
  • 在 /enterprise/device/settings/ekb/config 目录中找不到 *.encrypted 文件。
  • 设备上的 Enterprise Keyboard 低于 3.0 版，不支持自定义布局。

图 4.无法访问 Enterprise Keyboard

启用/禁用

启用/禁用与 DataWedge 配置文件一起使用的 Enterprise Keyboard。如果启用，则在前台显示的活动/应用程序中，当进行配置文件或活动切换/更改时，Enterprise Keyboard 的选定布局会自动出现在关联的应用程序中。如果禁用，Enterprise Keyboard 不会显示在关联的应用程序中。

选择布局

为键盘选择布局。可用布局填充自设备上 /enterprise/device/settings/ekb/config 目录中自定义生成的 *.encrypted 文件（例如，EKBCustomLayouts.encrypted）。如果该文件不存在，则无法访问（灰显）和配置 Enterprise Keyboard 配置。布局组名称基于 *.encrypted 文件名。有关更多信息，请参阅 Enterprise Keyboard。
