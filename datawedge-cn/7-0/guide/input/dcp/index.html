DCP 输入

DataWedge 7.0

概述

Data Capture Plus（以前称为“Data Capture Panel”）支持将设备屏幕的某些区域指定为扫描触发器。通过轻触这样的屏幕区域，DataWedge 会对扫描器按钮按下或其他硬件触发器作出响应。DCP 默认处于禁用状态。

DataWedge 配置文件配置屏幕允许应用程序用户在加载特定配置文件后，在屏幕上配置 DCP 的外观。如果用户选中此选项以启用 DCP，下面显示的五个参数会出现在首选项屏幕上，并且您可根据需要进行配置。

注意：如果在活动配置文件中禁用了扫描器，则不会显示 DCP。

用于设置扫描触发器的 Data Capture Plus 选项

停靠按钮开

设置浮动 DCP 按钮的初始停靠位置。用户在运行时进行的更改将保存到活动配置文件中。

停靠选项：

  • 仅右侧
  • 仅左侧
  • 任一侧

以最小化模式显示的 Data Capture Plus

启动模式

设置 DCP 的启动模式。如果配置为作为按钮启动，则可以在运行时通过拖动来更改 DCP 模式，但将无法在配置文件中更改启动状态。

启动模式选项：

  • 按钮模式（浮动按钮）
  • 全屏模式
  • 仅按钮模式

以最大化模式显示的 Data Capture Plus

按钮最高位置

设置按钮位置上限，以占总屏幕高度的百分比表示。例如，在垂直测量高度为四英寸的屏幕上，如果将此值设置为 75 (%)，则 DCP 按钮的上边缘无法来到距离屏幕顶部不到一英寸的位置。

底部最低位置

设置按钮位置下限，以占总屏幕高度的百分比表示。例如，在垂直测量高度为四英寸的屏幕上，如果将此值设置为 25 (%)，则 DCP 按钮的下边缘无法来到距离屏幕底部不到一英寸的位置。

拖动检测时间

在触发扫描器操作之前，DCP 应在屏幕轻触后应等待的等待时间（以毫秒计）。将 DCP 按钮拖动到新位置时，这可帮助阻止意外触发事件发生。

注意：使用摄像头作为扫描器时，快速触摸并释放 DCP 有时会启动取景器。要退出，请按“后退”按钮。

注意：如果配置为作为按钮启动，则可以在运行时通过拖动来更改 DCP 模式，但将无法在配置文件中更改启动状态。但是，对垂直位置的运行时更改和设备屏幕的停靠侧将保存到活动配置文件中。

使用 DCP 进行扫描

要使用 DCP 扫描条码：

1.在 DCP 已启用的情况下，轻触并按住屏幕上为 DCP 指定的区域。当按住时，扫描光束（或摄像头取景器）将处于活动状态。

2.将扫描光束或摄像头十字线对准要扫描的条码。DCP 将使用在当前配置文件的条码输入插件中配置的首选项。

3.松开手指以停止扫描或关闭摄像头取景器。

注意：使用摄像头作为扫描器时，快速触摸并释放 DCP 控件有时会启动取景器。要退出，请按“后退”按钮。
