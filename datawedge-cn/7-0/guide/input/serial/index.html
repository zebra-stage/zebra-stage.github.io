串行输入

DataWedge 7.0

概述

串行输入选项用于指定扫描器或其他设备（连接到将用于获取数据的串行端口）的通信参数。在 DataWedge UI 中，所选内容可用的串行端口数会因主机设备上的串行端口数而异。

串行端口配置

仅适用于运行 Android N 及更高版本的设备。Pre-N 版本仅提供启用/禁用功能。

在 Zebra VC80x 的串行端口 1 上启用输入；还会显示第二个串行端口（已禁用）。

波特率 - 指定已连接串行设备的调制速率。

数据位 - 指定串行帧中的数据位数（每字节数据位）。

奇偶校验 - 指定使用下列其中一个值的奇偶校验位：

  • 无：无奇偶校验检查
  • 奇数：设置奇偶校验位，使位集的计数为奇数
  • 偶数：设置奇偶校验位，使位集的计数为偶数
  • 标记：将奇偶校验位设置为 1
  • 空格：将奇偶校验位设置为 0

停止位 - 指定要使用的停止位数。

轻触设置会显示更改其值的对话框。

注意：

  • 串行端口配置仅在运行 Android N 及更高版本的设备上可用。对于之前的版本，只有启用/禁用选项可用于串行端口。
  • 有关特定设备的所需串行通信设置的更多信息，请参阅所连接外围设备的附带文档。
  • 串行端口设置还可以通过编程方式进行配置。请参阅 SET_CONFIG API 部分。
