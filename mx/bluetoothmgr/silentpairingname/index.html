Used to enter the name of a Bluetooth device with which to allow device pairing without requiring user confirmation. On devices running Android 9.2 (and older), only the Name field is required to create a Silent Pairing Rule but can be combined with other fields (e.g. Class of Device and/or Upper Address Part) to improve security.

On devices with MX 9.3 and later, the Name MUST be combined with the Upper Address Part (UAP) of the device's MAC address to create Silent Pairing Rules. This change was implemented for compliance with a Google security mandate.

Input value rules:

  • String from 1-255 characters containing Bluetooth device name. For example: "Zebra Printer_01"
