The LifeGuard Over the Air Manager (formerly FOTA Manager or FotaMgr) controls the Firmware Over The Air (FOTA) Client on the device. This client allows administrators to perform OS updates on Zebra devices without a physical connection. The FOTA Client app comes preinstalled on supported devices and is preconfigured to communicate with the Zebra update server. Learn more about Zebra LifeGuard.

SUPPORT NOTES

  • This CSP is displayed as "FOTA Manager" (FotaMgr) on some devices with older MX versions.
  • StageNow support for a given CSP or feature sometimes lags behind its support of the MX version that introduced it.
  • For Zebra Professional-series devices: This CSP requires purchase of a Mobility DNA Enterprise License.
  • On QC8956-platform devices: This CSP requires Android 7.x Nougat or later with LifeGuard Update 19 or later.

Main Functionality

  • Set the State (active/inactive) of the over-the-air update client on the device
  • Enable/Disable the FOTA user interface on the device
  • Enable/Disable automatic OS updates whenever they become available
