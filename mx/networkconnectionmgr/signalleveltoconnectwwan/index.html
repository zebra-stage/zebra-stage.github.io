Used to enter the minimum signal level (in dBm) for connection to a cellular (WWAN) network. Values range from -156 dBm (weak) to -31 dBm (strong).

NOTE: The signal-level ranges of this parameter encompass those of 4G and 5G networks as well as the broader ranges of 2G and 3G. Values closer to 0 (zero) are strongest.

Parm input value rules:

  • String between "-156" (weak) and "-31" (strong) representing WWAN signal strength in dBm
