Used to enter a signal strength low threshold on the monitored SIM slot, below which the device switches to the non-preferred SIM subscription. A value of "0" (default) is ignored, causing no change to the current setting.

Parm input value rules:

  • String between "-75" (strong) and "-125" (weak) representing WLAN signal strength in dBm
  • Entering "0" causes no change to the current setting, if any
