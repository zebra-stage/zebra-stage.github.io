Used to enter a signal strength (in dBm) between -75 (strong) and -125 (weak), below which the device switches to the non-preferred SIM subscription. A value of "0" (default) is ignored, causing no change to any current value.

Parm input value rules:

  • String between "-75" (strong) and "-125" (weak) representing WLAN signal strength in dBm
  • Entering "0" causes no change to current value, if any
