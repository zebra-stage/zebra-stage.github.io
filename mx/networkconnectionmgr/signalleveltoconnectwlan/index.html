Used to enter the minimum signal level (in dBm) for connection to a Wi-Fi (WLAN) network. Values range from -90 dBm (weak) to -30 dBm (strong). A value of "0" (default) is ignored, causing no change to any current value.

Parm input value rules:

  • String between "-90" (weak) and "-30" (strong) representing WLAN signal strength in dBm
  • Entering "0" causes no change to current value, if any
