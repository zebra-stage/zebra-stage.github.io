Used to enter a signal strength (in dBm) between -125 (weak) and -75 (strong), above which the device switches back to the preferred SIM subscription. A value of "0" (default) is ignored, causing no change to any current value.

Parm input value rules:

  • String between "-125" (weak) and "-75" (strong) representing WLAN signal strength in dBm
  • Entering "0" causes no change to current value, if any
