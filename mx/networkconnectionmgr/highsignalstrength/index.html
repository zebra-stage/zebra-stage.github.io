Used to enter a signal strength high threshold on the monitored SIM slot, above which the device switches back to the preferred SIM subscription. A value of "0" (default) is ignored, causing no change to any current value.

Parm input value rules:

  • String between "-125" (weak) and "-75" (strong) representing WLAN signal strength in dBm
  • Entering "0" causes no change to current value, if any
