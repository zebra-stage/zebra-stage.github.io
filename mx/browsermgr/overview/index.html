The Browser Manager (BrowserMgr) permits the configuration of password retention, default home page and other settings on the Android Open Source Project (AOSP) browser, which is the default browser on many Zebra Android devices and has been modified for this purpose.

This CSP is INOPERABLE on devices with OSX 4.0 and later.
     BrowserMgr was deprecated with MX 10.0; it is no longer being advanced or supported.

Main Functionality

  • Set the Default Home Page
  • Remember Passwords enable/disable
  • Save Form Data enable/disable
