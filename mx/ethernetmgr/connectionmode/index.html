Used to select which wireless radio(s) to enable/disable when a wired Ethernet connection is established. This feature allows Wi-Fi and wired Ethernet connections to co-exist on devices running Android 14 (or later) with MX 15.1 (or later).

While enabled, Connection Mode blocks settings changes by any other means, including the Android Settings panel.

Platform Notes

  • This feature requires EthernetConfig 14.36 (or later) on the device, delivered with the LifeGuard updates shown below.
  • Supported ONLY on the following platforms with these (or later) platform-specific LifeGuard updates:
    • 2290 - 14-12-17.00-UN-U35
    • 4490 - 14-15-22.00-UG-U129
    • 6375 - 14-09-18.00-UG-U285
    • 6490 - 14-35-10.00-UG-U76

See Zebra device models by platform

Important Notes

  • Connection Mode requires EthernetConfig 14.36 (or later).
    To display version on the device, go to Settings > Network & Internet > Ethernet > menu > Settings
  • This parameter configures the device according to Device Policy Manager (DPM) settings established by the administrator upon device enrollment; while enabled, it blocks settings changes by any means, including the Android Settings panel. Previous settings are restored when wired Ethernet is disconnected from the device.
  • Casting Mode enables Wi-Fi Direct connections (e.g. for using Zebra Wireless Workstation Connect) and permits wired Ethernet to carry only data.
  • While in Casting Mode, the Miracast feature becomes inoperable if the DPM policy with the "Disallow config tethering" restriction is enabled before initiating the casting feature. To ensure proper casting functionality, enable the "Disallow Config Tethering" restriction after casting is activated.
  • Connection Mode does not prevent manual settings changes by the device user, which can potentially put the device in an undefined state.
  • On WLAN-only devices, Ethernet and Wi-Fi icons appear in the status bar when Ethernet is connected.
  • On WLAN/WWAN-enabled devices, only the Ethernet icon is shown; Wi-Fi and Mobile icons are not visible.
