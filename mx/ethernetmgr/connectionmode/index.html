Used to select which wireless radio(s) to enable/disable when a wired Ethernet connection is established. This feature allows Wi-Fi and wired Ethernet connections to co-exist on devices with MX 15.1 (or later) and Android 14 (or later). All communications are restored to their previous settings when wired Ethernet is disconnected from the device.

Important Notes

  • Casting Mode enables Wi-Fi Direct connections (e.g. for using Zebra Wireless Workstation Connect) and permits wired Ethernet to carry only data.
  • When using Zebra Defaults or Casting Mode, Wi-Fi state changes with Ethernet connection follow Data Protection Management (DPM) policy if the "Disallow config Wi-Fi" restriction is enabled.
  • While in Casting Mode, the Miracast feature follows DPM policy if the "Disallow config tethering" restriction is enabled.
  • Connection Modes do not prevent manual settings changes by the device user.
  • On WLAN-only devices, Ethernet and Wi-Fi icons appear in the status bar when Ethernet is connected.
  • On WLAN/WWAN-enabled devices, only the Ethernet icon is shown; Wi-Fi and Mobile icons are not visible.
