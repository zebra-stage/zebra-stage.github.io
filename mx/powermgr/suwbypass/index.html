Controls whether to bypass the Android "setup wizard" (also known as the "Welcome Screen") following any Enterprise Reset. Applies only on devices with Google Mobile Services (GMS). Once applied, this setting persists on the device and is effective following all subsequent Enterprise Resets until it is explicitly changed using PowerMgr or cleared by a Factory Reset.

This feature behaves differently on devices running Android 13 (and later) than on those with earlier Android versions. Enhanced behavior is described below, and is implemented in the platform updates and MX versions shown in the table.

Setup Wizard Bypass Enhancements

  • The number of Google acceptance screens (each of which requires manual interaction) is reduced to three (from as many as 10), after which StageNow launches on the device, allowing any remaining barcodes to be scanned to complete device staging.
  • The number of Google screens is similarly reduced when scanning an enrollment barcode for DNA Cloud and some EMM systems, after which StageNow launches on the device and any remaining barcodes can be scanned.
  • After applying the Power Manager Setup Wizard Bypass feature on a device, the SUW is completely skipped following any future Enterprise Reset. When the device reboots, persisted configuration profiles (if any) are applied and the Android Home Screen is displayed with no manual intervention required.
  • Setup Wizard Bypass can be applied using Zebra DNA Cloud and StageNow, as well as EMM systems with Zebra OEMConfig.
