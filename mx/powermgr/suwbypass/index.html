Controls whether to bypass the Android "setup wizard" (which begins with the "Welcome" or "Hi there" screen) following an Enterprise Reset (ER). Applies to devices with Google Mobile Services (GMS) only. Once applied, this setting persists on the device and the SUW is skipped following all subsequent ERs until the setting is explicitly set to "False" using PowerMgr or cleared by a Factory Reset.

When the device reboots following an Enterprise Reset, persisted configuration profiles (if any) are reapplied and the Android Home Screen is displayed with no manual intervention required. Setup Wizard Bypass is applied only through barcodes or .bin files created with Zebra DNA Cloud or StageNow, or with an EMM system using Zebra OEMConfig.

Behavior of this feature varies by Android version. For a full description of all behaviors, please see the StageNow Wipe-a-Device guide.
