This is the On/Off switch for Doze mode, a set of energy-saving Android features designed to help maximize usage while a device is running on battery power. To conserve power, Doze mode prevents apps from accessing network services and delays tasks and notifications except during scheduled maintenance windows. While on battery, a device enters Doze mode when its screen is off and the system detects that the device has remained stationary for a while. Learn more from the Android community.

By default, all apps are subject to Doze mode. However, for apps that must be available at all times, such as those for real-time communications, it's possible to exempt app(s) from Doze mode restrictions using App Manager. Go there now.
