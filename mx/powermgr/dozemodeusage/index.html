This is the On/Off switch for Doze mode, a set of system-wide energy-saving Android features designed to help maximize usage while a device is running on battery power. Encompassing the whole device, Doze mode conserves power by preventing most apps from accessing network services and delaying tasks and notifications except during scheduled maintenance windows. While on battery, a device enters Doze mode when its screen is off and the device hasn't been moved in a while.

By default, all apps are subject to Doze mode. However, for apps that must be available at all times, such as those for real-time communications, it's possible to exclude individual app(s) from power restrictions of Doze mode using App Manager. Go there now.

Doze mode behavior varies by Android version. Learn more from the Android community.
