Used to enter the path and filename of an on-device .jpeg or .png image or .mp4 video to display on the screen while the device is in screen-saver mode.

On devices running Android 14 (or later) with MX 15.3 (or later): Entering the name of an on-device folder (with no individual file name) displays all media in that folder in alphabetical sequence, advancing every 20 seconds until the screen saver is exited.

Parm value input rules:

  • Accepts a string of characters corresponding to a valid on-device folder path and file name
  • Supports .jpeg and .png image files and .mp4 video files only
  • Devices with Android 14+ and MX 15.3+: File name is optional; displays all files in specified folder
