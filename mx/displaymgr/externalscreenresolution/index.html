Used to select screen resolution on the secondary (external) display when a device is connected to an external monitor through a Zebra Workstation Cradle.

Notes

  • This feature requires "Force Desktop Mode" to be enabled on the device (Settings > Developer Options or StageNow > DisplayMgr > Secondary Display Mode > Mirror Device Screen).
  • This setting has no effect on devices with the Zebra Workstation Connect app installed and Desktop Mode configured.
  • If applying this setting to a cradled device, changes in resolution occur next time the device is inserted into the cradle.
  • If a device is cradled without an external monitor connection, the setting is applied to the device. When removed from the cradle, device resolution returns to its previous setting.
  • The default setting for external screen resolution is 1920 x 1200.

Upon configuring this setting, if font size or display size is changed, on removing device from cradle, font size remains same as the value selected before docking device to cradle. Display size falls back to default (medium) after removing device from cradle.

When this setting is configured, device resolution falls back to default only when device is removed from workstation cradle. If device is retained in workstation cradle and HDMI cable is removed (external monitor disconnected), configured resolution on the device does not fall back to default.

This setting is applied globally when device is in COPE mode.
