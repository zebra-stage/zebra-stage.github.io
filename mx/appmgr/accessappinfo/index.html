Used to allow blocking of access to "App Info" for some or all device apps without blocking access to other parts of the Android Settings panel. When App Info is disabled for an app, the device user is prevented from quitting (using "Force Stop"), uninstalling, clearing data or otherwise changing the app's configuration through the App Info screen. After applying this feature, App Info (and app uninstallation) remain accessible from the "Recent Apps" list and from any app-related notification.

To prevent app removal by a device user, apply one of the Android Device Policy Manager APIs shown below. About DPM.

  • setUninstallBlocked() blocks uninstallation of the specified package name
  • UserManager.DISALLOW_UNINSTALL_APPS blocks uninstallation of all packages

To block all or part of the Android Settings panel, see Access Manager.
