Used to enter one or more apps (by Package Name) to be added to the Battery Optimization "exemption whitelist." While running, apps on this list prevent the device from entering power-saving modes. To enter multiple Package Names, separate with commas. Package Name(s) must be specified exactly, and can be acquired from the application developer, or by extraction from the the .apk file or from the device file system. Doze Mode is enabled/disabled using the Power Manager > Doze Mode Usage parameter.

DO NOT USE THIS PARAMETER if the desired behavior is to ALLOW Doze Mode. On devices running Android 7.x and later, all apps by default allow the device to enter Doze Mode to maximize device usage while on battery; no further action is required. Designating one or more apps using this parameter prevents the device from entering Doze Mode whenever those app(s) are in the foreground.

Also see:

  • Using Doze and App Standby | Differences between modes and how and when to use them
  • Power Management with Doze | Describes exactly how devices behave in Doze Mode

Input value rules:

  • String containing one or more Package Names
  • Separate multiple names with commas. For example: "com.myCompany.myPackage,com.myCompany.myOtherPackage"
    If a name is specified incorrectly in a list, all names following that name are ignored.
