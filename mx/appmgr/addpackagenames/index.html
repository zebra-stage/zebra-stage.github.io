Used to enter one or more apps (by Package Name) to be subject to the restrictions of Android Doze mode energy saving features. This is the default setting for most apps on GMS devices running Android 7.x and later. Therefore, use of this parameter is required only for app(s) that were previously excluded from Doze mode restrictions using the Exclude App from Doze parameter. Doze mode itself is controlled using the Power Manager > Doze mode Usage parameter.

Also see:

  • Using Doze and App Standby | Differences between these power-saving modes and how and when to use them
  • Power Management with Doze | Android power-saving features and relevant behavior of apps and devices

Input value rules:

  • String containing one or more Package Names
  • Separate multiple names with commas. For example: "com.myCompany.myPackage,com.myCompany.myOtherPackage"
  • If a name is specified incorrectly in a list, all names following that name are ignored. Package Name(s) can be acquired from the application developer, by extraction from the .apk file or from the device file system.
