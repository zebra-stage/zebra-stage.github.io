Used to enter one or more apps to be added (by Package Name) to the Battery Optimization "exemption whitelist." Apps on this list are exempt from restrictions of the Android Doze mode energy saving features. To enter multiple Package Names, separate with commas. Package Name(s) must be specified exactly, and can be acquired from the application developer, or by extraction from the .apk file or from the device file system. Doze mode is enabled/disabled using the Power Manager > Doze mode Usage parameter.

USE THIS PARAMETER if the desired behavior is to PREVENT the app from becoming unavailable when the device is in Doze mode. On GMS devices running Android 7.x and later, all apps by default are subject to Doze mode. Exempting one or more apps using this parameter allows the app(s) to use network services and perform all tasks at all times, potentially consuming more battery power.

Also see:

  • Using Doze and App Standby | Differences between modes and how and when to use them
  • Power Management with Doze | Describes exactly how devices behave in Doze mode

Input value rules:

  • String containing one or more Package Names
  • Separate multiple names with commas. For example: "com.myCompany.myPackage,com.myCompany.myOtherPackage"
    If a name is specified incorrectly in a list, all names following that name are ignored.
