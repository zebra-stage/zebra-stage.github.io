Used to enter the name of app package(s) on the device to which to grant or deny permission for the selected feature.

Input value rules:

  • String of at least 1 character containing the full path and name(s) of device-resident app package(s)
  • Separate multiple names using commas (i.e. packagename1,packagename2,packagename3)
  • If specifying a third-party launcher, enter single package name only
