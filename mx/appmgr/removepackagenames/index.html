Used to enter one or more apps (by Package Name) to be "unrestricted" by the power-saving features of Android Doze mode and other limitations imposed by battery optimization. On GMS devices running Android 7.x and later, all apps by default are subject to Doze mode. Use of Doze mode on the device is controlled using the Power Manager > Doze Mode Usage parameter.

USE THIS PARAMETER if the desired behavior is to allow the app(s) to remain available when the device is in Doze mode, and to access network services and perform all tasks at all times. Such apps are likely to consume more power than those left in their default state.

Input value rules:

  • String containing one or more Package Names
  • Separate multiple names with commas. For example: "com.myCompany.myPackage,com.myCompany.myOtherPackage"
  • If a name is specified incorrectly in a list, all names following that name are ignored. Package Name(s) can be acquired from the application developer, by extraction from the .apk file or from the device file system.
