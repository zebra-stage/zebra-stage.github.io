Used to enter one or more apps (by Package Name) to be removed from the Battery Optimization "exemption whitelist." Apps removed from the exemption list are subject to the restrictions of Android Doze mode energy saving features (the default for most apps). To enter multiple Package Names, separate with commas. Package Name(s) must be specified exactly, and can be acquired from the application developer, or by extraction from the .apk file or from the device file system. Doze mode is enabled/disabled using the Power Manager > Doze mode Usage parameter.

USE THIS PARAMETER if the desired behavior is to ALLOW the app to help conserve power when the device is in Doze mode. On GMS devices running Android 7.x and later, most apps by default are subject to Doze mode; use of this paramter is required only for app(s) previously exempted.

Input value rules:

  • String containing one or more Package Names
  • Separate multiple names with commas. For example: "com.myCompany.myPackage,com.myCompany.myOtherPackage"
    If a name is specified incorrectly in a list, all names following that name are ignored.
