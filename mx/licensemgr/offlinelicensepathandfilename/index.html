Used to enter the on-device path of lservrc, the Software License File that must exist in the device file system and contain the exact contents acquired from the Zebra Software License Server. This parameter applies the license to the device.

Parm value input rules:

  • String containing a valid path of lservrc, a Software License File in the device file system
  • Must contain at least 1 character
  • Example: /sdcard/Android/data/com.zebra.licensemgrservice/lservrc
