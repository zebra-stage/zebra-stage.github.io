Used to specify a User Name, which must correspond with a user account configured on the AppGallery server. The User Name (with corresponding Password and Organization Key) enables AppGallery on the device to access private apps and collections made available by the organization. AppGallery can otherwise access only public-facing marketplace apps.

Parm value input rules:

  • String containing the User Name as it appears in the Admin screen
  • The minimum length is 1 character
