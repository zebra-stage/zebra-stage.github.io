Controls whether the device user is permitted to change the state (turn on/off) of "Do Not Disturb" (DND) mode on the device, which toggles app notification sounds and messages. When Enabled (option 1), the user can turn notifications on or off using the Notification pull-down or Android Settings panel. When Disabled, the user can view the DND state but cannot change it.

This parameter does NOT change the feature state (turn on/off); it controls the device user's ability to do so.
