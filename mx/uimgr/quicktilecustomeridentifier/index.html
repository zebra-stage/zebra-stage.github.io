Used to enter the name of the custom Quick Setting tile to show/hide on the device.

Parm value input rules:

  • String containing a valid name of a custom Quick Tile on the device

Platform Notes

  • For ET40 and other 6375-platform devices, this parameter is supported ONLY with the following Android-specific OSX versions:
    • Android 13 - QCT6375.130.13.33.28 (or later)
    • Android 14 - QCT6375.140.14.11.8 (or later)
      See all Zebra devices by platform.
