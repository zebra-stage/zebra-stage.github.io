Controls whether the device user is permitted to change the state (turn on/off) of the Large-screen Taskbar, which appears at the bottom of the screen on supported devices, and displays recent and pinned apps, plus a link to the app library.

Supported Devices

  • CC600 (8 inch, 10.1 inch)
  • ET5x (8 inch, 10.1 inch)
  • L10 (8 inch, 10.1 inch)
  • ET60, ET65 (10.1-inch)
  • ET40, ET45 (10.1-inch only)

This parameter does NOT change the feature state (on/off); it controls only the device user's ability to do so.
