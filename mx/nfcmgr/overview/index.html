The NFC Manager (NfcMgr) controls near field communications options on the device, including power supplied to the NFC radio and the state and behavior of features to improve performance and/or to avoid interoperability issues. NFC is a short-range wireless specification that enables two devices that are touching or within close proximity to communicate using magnetic field induction.

Advanced NFC functionality is supported only on Zebra devices.

Supported Devices

NFC Manager is supported only on NFC-equipped Zebra devices, including (but not limited to):

  • All Zebra 2290-platform devices
  • All Zebra 4490-platform devices
  • All Zebra 5430-platform devices*
  • All Zebra 6375-platform devices
  • All Zebra 6490-platform devices
  • Some SDM660-platform devices, including:
    • CC600/CC6000
    • EC50/EC55
    • MC20
    • MC3300X
    • MC9300
    • TC21X/TC26X
    • TC52X/TC57X
    • TC77
    • TC72X
    • TC8300
    • WT6000

* NFC is optional on some models.

See all Zebra device models by platform

To inquire about NFC support on a Zebra device not listed above, please contact Zebra support or search the Zebra Knowledge Center.

Main Functionality

  • Enable/Disable:
    • NFC Data Exchange Format (NDEF)
    • Tap-to-pair functionality
    • NFC radio and its ability to communicate
    • Polling mode
    • Card Emulation mode
    • Low-power Card Detection (LPCD)
    • Peer-to-Peer mode (DEPRECATED)
    • CPU boost mode (DEPRECATED)
    • NFC usage when the Android "lock screen" is displayed (DEPRECATED)
  • Select type A, B, F, or V tags for use
  • Select communication speed for Types A and B cards
  • Reset device to default NFC settings
  • Supports the following card technologies:
    • ISO 14443 Types A and B (ISO/IEC 14443A, 14443B)
    • ISO 15693
    • FeliCa (JIS X6319-4)
    • Type V (all that comply with ISO/IEC 15693)
