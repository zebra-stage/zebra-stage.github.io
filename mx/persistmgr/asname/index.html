Used to enter a Name to be assigned to a Request XML document when it is made Persistent and/or to identify an existing Persistent Request XML document on which to perform a Persist Action. When identifying an existing document, the value in the "Persist as Name" field is used in conjunction with the values of "Persist As Version" and "Persist As Order" (if previously specified). Failure to submit exact (and correct) values results in no action and returns an error in the Result XML.

Usage Notes

  • The value specified in Persist As Name must be unique on the device. If a submitted document contains a Persist As Name value that already exists on the device, the existing document is overwritten by the newer one.

See the List Persistent Documents section of the Overview above for more information about this parameter.
