Permits selection of the GPS Location Mode for determining the device location, allowing for scanrios in which location accuracy can be sacrified in favor of longer battery life. This parameter permits selection of Android's three location modes:

  • High Accuracy mode suppliments information from the built-in GPS receiver with Wi-Fi and cellular data to calculate the location with the highest degree of precision. This option draws the most energy.
  • Battery saving mode uses only Wi-Fi and cellular data to determine location, but doesn't work with Google Maps or similar apps. This option draws the least energy.
  • Device-only mode uses only GPS data; subject to signal loss in some scenarios.
