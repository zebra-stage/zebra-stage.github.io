Used to select "pass-through antenna" settings on the device for use with external antenna(s) connected to the ANT1, GPS and ANT2 ports on an ET6x vehicle dock.

Options that include "None" configure the device to use the corresponding internal antenna, even if an external antenna is connected to an antenna port on the dock and "Use External Antenna" (option 2) is chosen in the Antenna Selection parameter.

Learn more about configuring external antennas in the ET6x Product Reference Guide.
