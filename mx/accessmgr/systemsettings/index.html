Controls features of the Android Settings panel that a device user is permitted to access. This parameter takes precedence over the "Quick Settings" parameter of UI Manager. If Reduced Settings Access is enabled, subsequent attempts enable Quick Settings result in failure.

Usage Notes

  • This parameter is the Zebra-recommended method of limiting or preventing user access to the Android System Settings panel. Blocking access by any other means could lead to unwanted system access or behavior.
  • On devices running Android 11 (or later) with OSX 11.5.21 (or later), the Reduced Settings option also enables Accessibility features.
  • CAUTION: Even with the Settings panel blocked, it is still possible to uninstall an app from the Android Launcher/Home Screen.
  • To prevent app removal by the device user, apply one of these Android Device Policy Manager (DPM) APIs:
    • setUninstallBlocked() blocks uninstallation of the specified package name
    • UserManager.DISALLOW_UNINSTALL_APPS blocks uninstallation of all packages
      Learn more about Android Device Policy Manager

Platform Notes

  • Supported* ONLY on the following platforms with these (or later) OSX versions:
    • 4490 - QCT4490.130.13.22.16
    • 6375 - QCT6375.130.13.28.24
    • 6490 - QCT6490.130.13.29.27
    • SDM660 - SDM660.130.13.26.27

See Zebra device models by platform

* Option 5 of this parameter is supported ONLY on devices running Android 13 or later and ONLY with the platform-specific OSX versions listed above.
