Controls whether the device user can access tethering and Wi-Fi hotspot settings through the Android Settings panel or Quick Settings control. If tethering and/or hotspot features are disabled on the device, setting this parameter to "Disabled" (option 2) prevents the user from enabling them.

This parameter controls only the device user's ability to change tethering and hotspot state (on/off) and settings.
    Direct control and configuration of those features is done through Wi-Fi settings.
