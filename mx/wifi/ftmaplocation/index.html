Used to enter the Fine Timing Measurement (FTM) AP Location Source. Option 1 selects the Location Configuration Information (LCI) and/or Location Civic Report (LCR) supplied by the AP. With option 0, an XML file must be created in advance, deployed to the device and referenced using the FTM Deployment File parameter.

LCI - Location Configuration Information includes latitude, longitude, and altitude of the AP providing the report.

LCR - Location Civic Report includes location reference, shape, map image sub-elements and other civic information of the AP providing the report.

For more information, please refer to IEEE.org.
