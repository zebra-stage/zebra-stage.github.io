Used to control Radio Resource Management, a feature of the 802.11k specification that analyzes access point traffic and estimates the optimal connection for maximizing wireless communication throughput. When disabled (option 0), the device generally connects to the AP with the strongest signal. When enabled using "802.11k Lite" (option 1), the Wi-Fi client can request Neighbor Reports, from which compatible APs return a Beacon Table* containing information† about nearby APs to help manage network connections effectively while roaming. This can be especially useful for environments where multiple networks are present.

Attempts to use this parameter on a device that doesn't support it returns a value of "0" or "99" (depending on the MX version) in the relevant section of the Result XML file.

* Requires Android 14 and MX 14.2+; not supported on all platforms.

† Beacon Table includes SSID, MAC address, signal strength, supported data rates, security settings and other information.
