Controls whether Wi-Fi hotspot settings can be configured on the device.

Platform and SKU Support

Hotspot-on-device functionality is supported as follows:

  • 6375, SDM660:
    • WLAN+WWAN SKUs only
  • 4490, 5430, 6490, 6690:
    • WLAN-only and WLAN+WWAN SKUs
  • 5430, 6490 WLAN-only SKUs:
    • 2.4GHz band only
  • 5430, 6375, 6490, SDM660 WLAN+WWAN SKUs:
    • 2.4GHz and 5GHz bands only
  • 6690 SKUs:
    • 2.4GHz, 5GHz and 6GHz bands
