The Remote Scanner Manager (RemoteScannerMgr) is used to monitor, configure and apply firmware updates to supported Zebra Bluetooth scanners from a Zebra device running Android 5.1 (Lollipop) or later.

Main Functionality

  • Update the firmware of a connected scanner
  • Perform the following functions on a paired scanner:
    • Apply a Scan123 config file (downloaded separately; learn more)
    • Cycle the power on the scanner
    • Initiate a device-location LED/sound sequence
    • Disconnect the scanner
    • Unpair the scanner
  • Enter the serial number of a device on which to perform an action

Supported Devices

Remote Scanner Manager supports these Bluetooth scanners:

  • CS6080
  • DS2278
  • DS3678*
  • DS8178
  • LI3678
  • RS507
  • RS5100
  • RS6000
  • RS6100

Scanners in BOLD are supported only with SDM660-platform devices in SSI mode running Android 11 or later.

* Support for the DS3678 requires MX 6.3 or later on the host device.
