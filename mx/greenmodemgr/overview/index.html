Green Mode Manager (GreenModeMgr) allows devices to be configured to help minimize power consumption. This is done by scheduling power to be supplied to the parts of the device that use the most energy only when those parts are expected to be active, and/or by shutting off components when device sensors indicate certain conditions.

Main Functionality

  • Set schedules for supplying power to:
    • WLAN (Wi-Fi) radio
    • WWAN (cellular) radio
    • Device display and backlight
  • Set device to turn off display:
    • While charging
    • At a specified time
    • When idle for a specified period
    • When placed face-down for a specified period
    • When a low-light condition is detected for a specified period
  • Automatically turn off WWAN radio whenever device connects to a Wi-Fi network
  • Automatically dim display backlight if a low-light condition exists for a specified period
  • Automatically dim display backlight if Adaptive Brightness is disabled in the Android Settings panel

Green Mode Manager is NOT supported on SDM660-platform devices.
