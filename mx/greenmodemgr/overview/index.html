Green Mode Manager (GreenModeMgr) allows devices to be configured to help minimize power consumption. This is done by scheduling power to be supplied to the parts of the device that use the most energy only when those parts are expected to be active.

Main Functionality

  • Set schedules for supplying power to:
    • WLAN (Wi-Fi) radio
    • WWAN (cellular) radio
  • Turn off display while device is charging

This CSP is NOT supported on SDM660-platform devices.
