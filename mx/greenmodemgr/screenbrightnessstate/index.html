Controls whether to automatically dim screen brightness when ambient light sensors on the device detect a low-light condition to the level specified in the Auto-dim Percentage parameter (below).

Behavior Notes

Behavior of this parameter varies based on the Adaptive Brightness setting in the Android system.

  • If Adaptive Brightness is ENABLED when parameter is applied:
    • Brightness is immediately reduced by the specified percentage (or to 30 percent, whichever is greater)
    • Screen brightness adjusts according to ambient light conditions
  • If Adaptive Brightness is DISABLED when parameter is applied:
    • Brightness is immediately reduced by the specified percentage (or to 30 percent, whichever is greater)
    • Brightness remains at the same level unless changed by the device user or a new setting is applied
