Used to select the mode of operation for Profile0, the default scanning Profile that (by default) automatically takes effect on the device for any app running in the foreground that has not been explicitly associated with another DataWedge Profile. On devices equipped with only a camera (and no dedicated scanner), Profile0 designates the camera for scanning, preventing its use for image capture.

To allow DataWedge usage only by apps explicitly associated with a DataWedge Profile, select "Disable" (Option 2).
