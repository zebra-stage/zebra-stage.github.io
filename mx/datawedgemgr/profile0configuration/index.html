Used to select the mode of operation for Profile0, the default scanning profile on most devices for unassociated apps. When enabled, Profile0 automatically takes effect for any app running in the foreground that has not been explicitly associated with another DataWedge profile.

Notes

  • When enabled, Profile0 is automatically activated for use by foreground apps not associated with another DataWedge profile.
  • While Profile0 is enabled on most devices by default, out-of-the-box behavior varies by device model and software build.
  • On devices equipped with only a camera (and no dedicated scanner), Profile0 designates the camera for scanning, preventing its use for image capture.
  • To allow DataWedge usage only by apps explicitly associated with a specific DataWedge profile, select "Disable" (Option 2).
