Used to select the operational state of Profile0, the default DataWedge profile that enables out-of-the-box scanning on Zebra devices without further configuration. By default, Profile0 automatically takes effect for any app running in the foreground that has not been explicitly associated with another DataWedge profile. Application of this parameter enables an administrator to force Profile0 to remain enabled or disabled following any subsequent action, including intent, profile import or Enterprise Reset, or to restore its factory-default behavior.

Notes:

  • While Profile0 is enabled on most devices by default, out-of-the-box behavior varies by device model and installed software build.
  • To limit DataWedge usage to one or more specific apps, select "Disable" (Option 2) and explicitly associate those app(s) with a DataWedge profile.
