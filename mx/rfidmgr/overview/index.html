The RFID Manager (RfidMgr) is used to update, manage and configure Zebra RFID peripherals.

During the 4-5 minute firmware update, status is displayed in the Notification bar.

Supported on Zebra MC333R and MC339R devices only.

Main Functionality

  • Update device firmware
  • Reset the radio
  • Reset device to factory-default settings
  • Export device settings for troubleshooting
  • Enable/disable configuration of advanced options:
    • Channel hopping
    • Channel masking
    • Country of operation
    • Query select
    • Query session
    • Query target
    • RFID transmit power level
