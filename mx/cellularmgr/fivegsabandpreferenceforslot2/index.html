Used to enter the preferred 5G Standaone band(s) to use in SIM slot 2 on the device. Select band(s) from the available range below.

Parm value input rules:

  • String from 0–255 characters containing only integers and commas
  • Use commas to separate multiple values. Example: "1,2,7,108"
  • 5G SA band range = 1–110
