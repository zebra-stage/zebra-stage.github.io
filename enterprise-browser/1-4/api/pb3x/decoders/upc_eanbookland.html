﻿ upc_eanbookland

upc_eanbookland META Tag

The upc_eanbookland META Tag is an action tag used to enable or disable decoding of upc_ean-bookland barcodes.

Syntax

upc_eanbookland (META Tag) Syntax

<META HTTP-Equiv="scanner" content="upc_eanbookland:[parameter]">

Parameters

Items listed in this section indicate parameters, or attributes which can be set.

Name Possible Values Description
Default Value
upc_eanbookland:[Value] true/false Enables/Disables Bookland code decoding. Device specific
Copy parameters template to clipboard: META Tags Javascript

Examples

The following example enables the scanner to decode upc_ean-bookland barcodes:

<META HTTP-Equiv="scanner" Content="all_decoders:enabled">
<META HTTP-Equiv="scanner" Content="upc_eanbookland:true">
<META HTTP-Equiv="scanner" Content="enabled">
Copy example to clipboard

Above example can also be written as shown below:

<META HTTP-Equiv="scanner" Content="all_decoders:enabled;upc_eanbookland:true;enabled">
Copy example to clipboard

or

<META HTTP-Equiv="scanner-all_decoders" Content="enabled">
<META HTTP-Equiv="scanner-upc_eanbookland" Content="true">
<META HTTP-Equiv="scanner-enabled" Content="SCN1">
Copy example to clipboard

The following example disables the scanner from decoding upc_ean-bookland barcodes:

<META HTTP-Equiv="scanner" Content="all_decoders:enabled">
<META HTTP-Equiv="scanner" Content="upc_ean-bookland:false">
<META HTTP-Equiv="scanner" Content="enabled">
Copy example to clipboard

Additional Information

Supported Platforms Windows CE, Windows Mobile, Windows Mobile SE
Persistence This tag is persistent.
Minimum Requirements Scanner or Imager module and device supports decoding of upc_ean-bookland barcodes.
