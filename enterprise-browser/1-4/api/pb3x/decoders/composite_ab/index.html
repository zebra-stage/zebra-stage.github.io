Composite_ab Meta Tag

PocketBrowser 3.x API

The Composite_ab Meta Tag is an action tag used to enable or disable the Composit_ab decoder.

Composit_ab (META Tag) Syntax 
<META HTTP-Equiv="scanner" content="Composit_ab:[parameter>

Parameters

Items listed in this section indicate parameters, or attributes which can be set.

Name Possible Values Description
Default Value
Composit_ab:[Value] enabled/disabled Enables/disables Composit_ab decoder. N/A
Copy parameters template to clipboard: META Tags Javascript

Examples

The following example enables the scanner to read only Composit_ab labels:

<META HTTP-Equiv="scanner" Content="all_decoders:disabled">
<META HTTP-Equiv="scanner" Content="Composit_ab:enabled">
<META HTTP-Equiv="scanner" Content="enabled">
Copy example to clipboard

Above example can also be written as shown below:

<META HTTP-Equiv="scanner" Content="all_decoders:disabled;Composit_ab:enabled;enabled">
Copy example to clipboard

or

<META HTTP-Equiv="scanner-all_decoders" Content="disabled">
<META HTTP-Equiv="scanner-Composit_ab" Content="enabled">
<META HTTP-Equiv="scanner-enabled" Content="SCN1">
Copy example to clipboard

The following example enables the scanner to read all labels except Code11:

<META HTTP-Equiv="scanner" Content="all_decoders:enabled">
<META HTTP-Equiv="scanner" Content="Composit_ab:disabled">
<META HTTP-Equiv="scanner" Content="enabled">

Additional Information

Supported Platforms Windows CE, Windows Mobile, Windows Mobile SE
Persistence This tag is persistent.
Min. Requirements Scanner or Imager module and device supports Code11.
