About Showcase App

Showcase App

Overview

The Zebra Showcase App is designed for any user to easily learn and experience key features and capabilities of Zebra Mobility DNA, a set of tools and features for enhancing productivity, performance and security of Zebra Android devices. The initial Beta release of the Showcase App includes 16 Data Capture demos ranging from simple/multi-barcode scanning to OCR and image capture. All demos are tutorial-based, offering users an easy way to immediately learn and explore the diverse features with minimal effort. The Showcase App is built on Zebra's Enterprise Browser that enables quick and easy updates of the app and demos.

Learn about Zebra Showcase App

Requirements

The Showcase App beta release applies to the following Zebra Android™ platforms (see the respective device product page in Zebra support portal for LifeGuard updates):

  • Android 11 with LifeGuard Update 11-35-05.00-RG-U00 or later on SD660-platform devices, such as TC52, TC57, TC72, and TC77.
  • Android 11 with LifeGuard Update 11-21-27.00-RG-U00-STD-ATH-04 or later on SD6490-platform devices, such as TC53, TC58, TC73, and TC78.

For more information on the devices supported on each platform, see Zebra Platform Devices.

Installation

The app launcher for Showcase App (Beta) is pre-installed on all supported Zebra Android devices. It can be updated to the final released version towards the end of 2023.

To install the Showcase App:

  1. Connect the device to the internet (required for the first time download).
  2. From the home screen, tap on the Zebra Showcase icon.
  3. Tap Download.
  4. In the EULA screen, tap Agree.
  5. The Showcase App is downloaded and installed. A tutorial is displayed providing an overview how to use the app. Follow the tutorial or tap Close
  6. The Showcase App home screen is displayed.

Notes:

  • A predefined DataWedge profile named "Zebra Showcase" is dedicated for use with Zebra Showcase App.
  • After the download, all demos can be experienced offline without an internet connection.

User Guide

Showcase App is tutorial-based. After launching the app for the first time, the user is presented with a tutorial on how to use the app. Subsequent attempts to launch the app displays the app home screen, which displays Favorites followed by the demo categories and the quantity of demos within each category.

Tap on the options menu at the top left:

The menu from the home screen provides access to:

  • Home - navigates to the home screen.
  • Help - displays the tutorial on how to use the app.
  • Send Feedback - opens a form to submit feedback on the app or requests for additional demos.
  • Scan An Invite - for future use; currently disabled and not functional
  • Notification Settings - enables/disables notification settings for new releases of demos or demo updates, and controls decode feedback notifications (audio, LED, haptic).
  • About - displays app version information.

Demo

Each demo includes the following options from the top menu:

  • View info - Provides information about the demo and feature, including devices requirements, benefits and related settings.
  • Demo Tutorial - Provides a walk-through on how to use the demo.
  • Settings - Provides setting options that can be configured for the demo. Options menu from a demo

Favorites

Add a demo to Favorites for quick access. In the View info screen of the demo, tap on the star at the top right. This fills the star with yellow color and adds the demo to Favorites.
