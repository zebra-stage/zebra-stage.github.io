About Showcase App

Showcase App

Overview

The Zebra Showcase App is designed for any user to easily learn and experience key features and capabilities of Zebra Mobility DNA, a set of tools and features for enhancing productivity, performance and security of Zebra Android devices. The Showcase App includes Data Capture demos ranging from simple/multi-barcode scanning to OCR and image capture. All demos are tutorial-based, offering users an easy way to immediately learn and explore the diverse features with minimal effort. The Showcase App is built on Zebra's Enterprise Browser that enables quick and easy updates of the app and demos.

Requirements

The Showcase App beta release supports all Zebra Android™ platforms running Android 13 or higher. There is support for some devices running Android 11, which applies to the following Zebra Android platforms (see the respective device product page in the Zebra support portal for LifeGuard updates):

  • Android 11 with LifeGuard Update 11-35-05.00-RG-U00 or later on SD660-platform devices, such as TC52, TC57, TC72, and TC77.
  • Android 11 with LifeGuard Update 11-21-27.00-RG-U00-STD-ATH-04 or later on SD6490-platform devices, such as TC53, TC58, TC73, and TC78.

For more information on the devices supported on each platform, see Zebra Platform Devices.

Installation

The app launcher for Showcase App is pre-installed on all supported Zebra Android devices.

To install the Showcase App:

  1. Connect the device to the internet. This is required for the first time download.
  2. From the home screen, tap on the Zebra Showcase icon.
  3. Tap Download to initiate the installation process.
  4. In the End User License Agreement (EULA) screen, tap Agree.
  5. Once the app is downloaded and installed, it will automatically close, and the device will return to the home screen.

When launching the Showcase App for the first time, a tutorial appears providing an overview on how to use the app. Either follow the tutorial or tap Close to proceed to the main app interface:

Notes:

  • A predefined DataWedge profile named "Zebra Showcase" is dedicated for use with Zebra Showcase App.
  • After the download, all demos can be experienced offline without an internet connection.

User Guide

The Showcase App is tutorial-based. After launching the app for the first time, the user is presented with a tutorial on how to use the app. Subsequent attempts to launch the app displays the app home screen, which displays Favorites followed by the demo categories and the quantity of demos within each category.

Tap on the options menu at the top left:

The menu from the home screen provides access to:

  • Home - navigates to the home screen.
  • Help - displays the tutorial on how to use the app.
  • Send Feedback - opens a form to submit feedback on the app or requests for additional demos.
  • Scan An Invite - for future use; currently disabled and not functional
  • Notification Settings - enables/disables notification settings for new releases of demos or demo updates, and controls decode feedback notifications (audio, LED, haptic).
  • About - displays app version information.

Demo

Each demo includes the following options from the top menu:

  • View info - Provides information about the demo, its features, benefits and links to additional information.
  • Demo Tutorial - Provides a walk-through on how to use the demo.
  • Settings - Provides demo-specific setting options that can be configured. Options menu from a demo

Favorites

Add a demo to Favorites for quick access. In the View info screen of the demo, tap on the star at the top right. This fills the star with yellow color and adds the demo to Favorites.

Troubleshooting

If the full list of demo applications is not visible, try the following steps:

  1. Uninstall Enterprise Browser.
  2. Open the Showcase Info page and clear the app's memory.
