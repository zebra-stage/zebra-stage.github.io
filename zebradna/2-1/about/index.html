About Zebra DNA Cloud

Zebra DNA Cloud 2.1

See what's new in zDNA Cloud 2.1

Overview

Introducing Zebra DNA Cloud (zDNA Cloud), an administrative solution for managing and configuring Zebra devices and apps. Like a cloud-based version of StageNow, zDNA Cloud also performs licensing and other administrative tasks, and will eventually do much more. Also like StageNow, zDNA Cloud leverages the power of Zebra's Mobility Extensions (MX), is controlled through a simple, intuitive UI, and can stand alone or work with a company's existing Enterprise Mobility Management (EMM) system.

The zDNA dashboard provides a high-level system overview, including a count of "Active" devices that have communicated with the system within the past 24 hours.
Click image to enlarge; ESC to exit.

Over time, Zebra DNA Cloud will be enhanced to provide far more administrative functionality than its stand-alone predecessor, offering small- and medium-sized businesses a cohesive, unified user experience for managing enterprise devices and apps throughout their usable life cycle.

Planned Enhancements:

  • Versioning support to allow zDNA Cloud to store more than version of app packages
  • LifeGuard Over the Air integration to enhance beyond deployment of upgrades and security patches
  • A tiered solution with free and fee-based feature levels to suit any budget and need
  • Training and tutorial videos to ensure that organizations can make full use of all zDNA Cloud capabilities

New in zDNA Cloud 2.1

  • Security enhancements:
    • Implements 30-minute timeout for inactive user sessions
    • Improved end-point and data-access security rules
    • Updated data catalogue helps improve privacy and compliance
  • Removes Wi-Fi requirement for initial setup; allows device enrollment over Ethernet, WAN or Wi-Fi
  • Enables "Update zDNA Client" function for licensed and unlicensed devices
  • "Discard" function now labeled "Apply Later" on Profile pages:
    • Device Settings Wizard
    • Device Settings Advanced
    • New Setup for Zebra Collection, My Collection
