About Zebra DNA Cloud

Zebra DNA Cloud 3.1

What's new in zDNA Cloud 3.1

Overview

Zebra DNA Cloud (zDNA Cloud) is an administrative solution for managing and configuring Zebra devices and apps. Like a cloud-based version of Zebra StageNow, zDNA Cloud also performs licensing and other administrative tasks, and will eventually do much more. Also like StageNow, zDNA Cloud leverages the power of Zebra's Mobility Extensions (MX), is controlled through a simple, intuitive UI, and can stand alone or work with a company's existing Enterprise Mobility Management (EMM) system.

The zDNA dashboard provides a high-level system overview, including a count of devices active within the last 24 hours.

New in zDNA Cloud 3.1

  • Zebra Collection now includes latest versions of:
    • Enterprise Browser (3.4)
    • Enterprise Home Screen (6.0)
    • Enterprise Keyboard (4.2)
  • Automatically replaces package names of older Zebra apps with the new version, when applicable for Zebra app installation*
  • Improved support for device populations with mixed device ownership (in organizations where the DNA Cloud client is NOT the only DO)
  • Adds search functionality for OEM Config > Device Settings > Advanced > Managed Configurations

* Previously, it was necessary to manually uninstall older versions of Zebra apps if the newer version came with a new package name. DNA Cloud now performs such operations automatically.
