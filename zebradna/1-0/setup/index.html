Zebra DNA Cloud Setup

PRE-RELEASE DOCUMENTATION

Overview

This section covers the Zebra DNA Cloud setup process and the first few steps for its use. For details about zDNA Cloud usage, see the Usage Guide.

Setup Synopsys

  1. Configure the organization's network
  2. Register with Zebra.com
  3. Create at least one "New Device Setup" Profile
  4. Enroll one or more devices using the New Device Setup Profile
  5. Apply license(s) to device(s)

System Requirements

Admin Console

  • Computer with Windows 10 or later
  • Google Chrome (v101.0 or later)
  • An account on Zebra.com (Register now)
  • Firewall or network proxy server configured as required (see below)

Supported Devices

  • Zebra device(s) running Android 11 or later
  • LifeGuard update as indicated in Usage Guide

Pre-release edition supports a maximum of 50 devices

I. Configure Network

The Zebra DNA client app communicates with the ports and domain names listed below.

The client network must be configured as follows:

Using Wildcards

  • Open an HTTPS port (443 by default) between device client and app servers
  • Allow these domain names through the firewall or proxy server:
    • connectivitycheck.gstatic.com
    • *.googleapis.com
    • *.firebaseio.com
    • *.cloudfunctions.net
    • [ProjectID].firebaseapp.com †
    • [WebPortal] †

NOT Using Wildcards

If wildcards are not supported or desired, add the following domain names to the "allow" list:

  • connectivitycheck.gstatic.com
  • www.googleapis.com
  • firestore.googleapis.com
  • cloudfunctions.googleapis.com
  • us-central1-[ProjectID].cloudfunctions.net †
  • [ProjectID].firebaseio.com †
  • [ProjectID]-default-rtdb.firebaseio.com †
  • s-usc1c-nss-XXXX.firebaseio.com‡
  • [ProjectID].firebaseapp.com †
  • [WebPortal] †

† Company-specific [ProjectID] section and [WebPortal] domain names are supplied by Zebra during on-boarding.

‡ "XXXX" represents a dynamic range of numbers from 0000–9999 controlled by Google that can change at any time. To retrieve the latest URLs associated with s-usc1c-nss-XXXX.firebaseio.com, create a script to regularly look up the s-usc1c-nss-XXXX.firebaseio.com URL using https://[ProjectID].firebaseio.com/.settings/owner.json and update the firewall rules accordingly.

III. Enroll Device(s)

The "New Device Setup" section is primarily for creating the minimum settings required to allow a device to communicate with the internet for further configuration and management (or for hand-off to an EMM). This process also downloads the zDNA client app to the device and enrolls it in the zDNA Cloud system. To stage a device with these settings, create a new Enrollment Profile and use the device to scan the barcode that appears at the end of the process.

Repeat this process as many times as needed for different device types, Wi-Fi configurations, office locations or other purposes.
Zebra recommends downloading barcode PDFs for later reference.

Create an Enrollment Profile:

After logging in as a Tenant Owner or Administrative User, proceed to Step 1. Click image to enlarge; ESC to exit.

1. In the New Device Setup section, click the "Set Up Device" button: Click image to enlarge; ESC to exit.

2. Enter the Wi-Fi settings for the device in its intended location.
Click "Next" button when finished. Click image to enlarge; ESC to exit.

3. Leave "LifeGuard Over the Air" box CHECKED (default) to enroll device(s) in zDNA Cloud system.

Uncheck box ONLY if Zebra's system is NOT to be used to send updates and security patches to the device.
Click "Next" button when finished. Click image to enlarge; ESC to exit.

4. Enter the following information as indicated:
    a. Enrollment Profile Name - use auto-generated name or enter custom
    b. Administrator message - appears on barcode page and Profile listings
    c. Device Group Association - helps organize by model, function, dept. etc.
Click "Next" button when finished. Click image to enlarge; ESC to exit.

5. Review settings. If necessary, click "Edit" to modify any setting(s).
Click "Next" button when finished. Click image to enlarge; ESC to exit.

6. With the StageNow app running on the device, scan the generated barcode to configure and enroll the device.
Zebra recommends downloading the barcode (PDF) for later reference.
Then click "Done" button. Click image to enlarge; ESC to exit.

Initial device staging and enrollment are complete.
