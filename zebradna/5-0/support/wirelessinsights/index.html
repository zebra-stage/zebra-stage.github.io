Using Wireless Insights

Zebra DNA Cloud 5.0

Overview

The Wireless Insights service provides real-time performance data for active Zebra Android devices, including Wi-Fi and cellular connection events, signal strength and service quality by carrier, device and/or device group. The service is part of the Zebra Collection in My Apps section and is free to use on supported devices without additional licensing.

The Wireless Insights dashboard shows real-time Wi-Fi (top row) and cellular performance for active devices.
Click image to enlarge; ESC to exit.

Requirements

  • Wi-Fi performance: Any Zebra device running Android 13 or later
  • Cellular performance: A TC26 or any 6490-platform device running Android 13 or later

See a list of Zebra devices by platform

Using Wireless Insights

  1. Click "Support Tools" From the DNA Cloud Dashboard to expose available modules: Click image to enlarge; ESC to exit.

  2. Click "Wireless Insights" to view the interactive Wireless Insights Dashboard.
    Real-time data is shown for currently active devices over the last 24 hours; Wi-Fi on top, cellular below: Click image to enlarge; ESC to exit.

  3. Click the date selector to analyze wireless data for a particular date and/or time range (max. = 30 days): Click image to enlarge; ESC to exit.

  4. Click on a graph section to display the number of effected devices: Click image to enlarge; ESC to exit.

  5. Double-click a graph section (as in image above) to display more information about the effected devices: Click image to enlarge; ESC to exit.

  6. Scroll down from Dashboard to view reports for the Device Group(s) selected in previous step.
    Click arrows to sort and/or filter reports as desired: Click image to enlarge; ESC to exit.

  7. Click Wi-Fi or Cellular in a group row to view additional data for that group.
    Within that Group, click Wi-Fi or Cellular in a device row to view data for that device (as in the image below). Click image to enlarge; ESC to exit.

  8. Clicking on Wi-Fi or Cellular for an individual device (from screen above) displays performance stats for that device: Click image to enlarge; ESC to exit.

  9. Click tabs to view different reports.
    Click on "breadcrumbs" to navigate to previous screens. Click image to enlarge; ESC to exit.

  10. Click "Export" from the Dashboard to archive data or perform further analysis: Click image to enlarge; ESC to exit.

Enabling Wireless Insights

Wireless Insights requires an agent to be present on all devices from which wireless data is to be extracted. There's also a one-time configuration change in the "My Services" section of the DNA Cloud console. Details below.

To Set Up Wireless Insights

  1. In the DNA Cloud console, click on the User Account (upper-right corner) and select "My Services" from the drop-down menu.
    A screen appears similar to the image below.
    In the Wireless Insights Service Info box, click "Enable" button (if not already enabled). Click image to enlarge; ESC to exit.

  2. Click <- Back in the upper-left corner to return to the DNA console's main screen.

  3. From the main screen, click My Apps to display the Zebra Collection screen:
    Click image to enlarge; ESC to exit.

  4. Click "Setup" button in "Wireless Insights" tile of the Zebra Collection.

  5. See Setting up a Zebra App for further instructions.
