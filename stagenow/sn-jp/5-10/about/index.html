StageNow について

StageNow 5.10

StageNow 5.10の新機能

概要

このガイドではステージング管理者を対象に、StageNow を使用して XML ベース プロファイルを作成し、それにより Zebra デバイスを設定する方法を説明します。

前提条件

StageNow を使用する前に、ステージング ワークステーションに次のソフトウェアをインストールする必要があります。

  • Windows 7、10 Pro (英語、中国語、または日本語)
  • Windows 版 Java
  • サポートされている Web ブラウザ:
    • Firefox 29 以降
    • Google Chrome 35 以降
    • Internet Explorer 9 以降

StageNow のインストールと使用に関する 要件と手順の全リストを参照してください。

このガイドのセクション

  • StageNow のインストール: StageNow ステージング ソリューションのインストール、アンインストール、およびアップグレードの手順を、システム要件を含めて説明します。

  • ご使用の前に: ユーザー、ユース ケース、サポート対象デバイスに関する情報など、StageNow ソリューションについて説明します。

  • プロファイル: StageNow ウィザードに関する情報が含まれています。これにより、ステージング管理者は、エンタープライズ デバイスのソフトウェア設定とインストール シナリオを定義できます。

  • プロファイル ウィザード: プロファイル作成ウィザードの使用方法について説明します。

  • 設定: ステージング管理者がプロファイル作成で使用する設定の構成と管理について記載しています。

  • 設定タイプ: 設定の作成時に使用可能なパラメータと値がリストされます。

  • ステージング ガイド (ランディング ページ): ステージング ガイドへのリンクがあります。各ガイドでは、管理者およびステージング オペレータを対象に提供されるステージング方法を説明し、またステージング メディアを使用して Zebra デバイスの設定と展開を行うためのプロファイルの作成に関する情報を記載しています。

  • トラブルシューティング: StageNow ツールで発生する可能性があるエラーについて説明し、解決策を示します。

StageNow 5.10 の新機能

  • MX 11.9 機能をサポート (下記を参照)
  • デバイスのインストール用に最新の EHS apk を選択できるようにします

MX 11.9 のサポートにより、次の主要な機能と拡張機能が追加されます。

  • Access Manager は次の機能を追加します。
    • 一部の Zebra アプリにアプリ ログを読み取る権限を付与します。
  • Cellular Manager は次の機能を追加します。
    • 着信および/または発信を拒否する
    • デバイスからの緊急通話後、着信拒否を無効にする
  • Enterprise Keyboard Manager は次の機能を追加します。
    • 管理者が Enterprise Keyboard で使用するカスタム言語を入力できるようにします。
  • Network Connection Manager は、管理者が携帯電話 (WWAN) および Wi-Fi (WLAN) ネットワークへの接続に関する設定を行うことができる new CSP で、デバイスが常に最適なネットワークを使用できるようにします。
  • Wi-Fi は次の機能を追加します。
    • 管理者が、デバイスの Fine Timing Measurement 位置の更新頻度の設定である FTM 位置更新モードを選択できるようにします。

MX 11.x は、Android 11 以降を実行しているデバイスでのみサポートされます。

バージョン履歴

StageNow 5.9 で追加

  • MX 11.8 および MX 11.7 の機能をサポート (下記を参照)
  • MX 11.8 までのすべての MX バージョンの拡張 XML (eXML) バーコードを作成するようになりました
  • Android 13 を実行している Zebra デバイスをサポート

• 次に示す主な特徴と拡張機能を追加する MX 11.8 をサポートします。

  • Clock は次の機能を追加します。
    • 管理者は、デバイスの時計の時刻と選択した NTP サーバーの時刻との最大差を指定して、同期しきい値を設定できます。
  • Touch Manager は次の機能を追加します。
    • キーの操作が長押しとみなされるために必要な長押しの遅延を設定します。

次に示す主な特徴と拡張機能を追加する MX 11.7 をサポートします。

  • Access Manager は次の機能を追加します。
    • アクセシビリティ機能への システム設定へのアクセスを制御します。
  • Cellular Manager は次の機能を追加します。
    • ネットワーク モードのリストから選択し、優先ネットワーク タイプを SIM スロットに割り当てます。
  • UI Manager は次の機能を追加します。
    • デバイスの音量上下ボタンの使用を制御して、TalkBack を切り替えます。これは、視覚障害者を支援するためにデバイスの単語を音声化します。

MX 11.x は、Android 11 以降を実行しているデバイスでのみサポートされます。

StageNow 5.8 で追加

Android 13 以降のデバイスのステージングをサポートするようになりました

StageNow 5.8 では、Android 13 以降を実行している Zebra デバイスをステージングするためのバーコードと .bin ファイルを生成する機能が導入されています。新しい方法では、公開されている仕様に基づいた拡張 XML (eXML) フレームワークが採用されています。これは、Android 13 を実行している Zebra デバイスでサポートされる唯一のバーコードベースのステージング方法です。以下のビデオでは、詳細について説明しています。

• 新しいガイド: デバイスの所有権 を EMM システムに移行する

• PDF およびリニア バーコード、NFC および USB ステージング メソッドの eXML 出力の生成の詳細はこちら
(Batch Manager、Camera Manager、または License Manager ではサポートされていません)

次に示す主な特徴と拡張機能を追加する MX 11.6 をサポートします。

  • Audio Manager は次の機能を追加します。
    • ミュート/バイブレーションの使用を制御します。これにより、デバイス ユーザーが機能をオンまたはオフにする機能が制限されます。
  • Battery Manager は次の機能を追加します。
    • 充電モード を設定し、デバイスを完全に充電する日時をスケジュールします。
    • デバイスのバッテリーの 最大充電率を指定します。
  • UI Manager は次の機能を追加します。
    • 外部キーボードの Num Lock キーを構成し、ショートカット アクションを割り当てます。
  • USB Manager は次の機能を追加します。
    • デバイス ユーザーにプロンプトを表示せずに周辺機器を接続するためのルール、周辺機器へのアクセス許可をアプリに付与するためのルール、および接続時にアプリを自動起動するためのルールを使用して USB ホスト モードを構成します。
  • Wi-Fi は次の機能を追加します。
    • Android 設定パネルでネットワークを構成するときに、サーバー証明書を必須にするかどうか、およびデバイス ユーザーが検証をバイパスすることを許可するかどうかを制御します。
    • システム証明書またはユーザーがインストールした証明書を使用する証明書アクションを選択します。

次に示す主な特徴と拡張機能を追加する MX 11.5 をサポートします。

  • Access Manager は次の機能を追加します。
    • 通知リスナーをバインドする権限を制御します。通知リスナーとは、通知に関連するシステム通信を受信する Android サービスです。
  • Bluetooth Manager は次の機能を追加します。
    • [リモート デバイス アクション] パラメータで作成されたサイレント ペアリング ルールをデバイスが評価するときに考慮するデフォルトの周辺機器のセットを選択します。
  • DevAdmin は次の機能を追加します。
    • デバイス所有者がアプリをインストール、アンインストール、またはアップグレードするときのインストール通知を制御します。
  • Enterprise Keyboard Manager は次の機能を追加します。
    • 音声入力を選択するための [マイク] タブを表示します。
    • EKB が呼び出されるたびに、[マイク] タブを [優先] タブとして選択します。
    • ソフト入力パネルが外部モニターに表示されているときのキーボードの幅とキーボードの高さを制御します。
  • Hosts Manager は次の機能を追加します。
    • アプリ、デバイス ユーザー、管理者などでデバイスを簡単に識別できるよう、デバイス名を設定します。
  • SD Card Manager は次の機能を追加します。
    • SD カード ポップアップを制御します。これにより、デバイス ユーザーは、デバイスに新しく挿入された SD カードを操作するように求められます。
  • Settings Manager は次の機能を追加します。
    • デバイス ユーザーが Web ページの表示に使用するブラウザ アプリを変更できるかどうかを制御します。
  • Touch Manager は次の機能を追加します。
    • デバイス タイプを 6375 プラットフォームに設定し、ET40 や ET45 などの 6375 プラットフォーム デバイスのタッチ モードを設定します。
  • USB Manager は次の機能を追加します。
    • デバイスが USB ホスト モードのときにすべての USB 周辺機器を使用できるようにするか、管理者がカスタム規則リスト パラメータを使用して作成した規則で定義されている周辺機器の使用を制限するかを制御します。
  • Wi-Fi は次の機能を追加します。
    • アゼルバイジャン、バングラデシュ、ガーナ、ジブラルタル、ガーンジー島、コソボ、キルギスタン、モルドバ、セネガル、ウガンダの国コードを選択します。
    • RF 帯域の選択を設定して、デバイスで 2.4 GHz および 6.0 GHz 帯域が動作できるようにします。
    • RSSI ローミングしきい値と RSSI デルタ値を選択して、ローミング中の Wi-Fi ユーザー エクスペリエンスを向上させます。
  • UI Manager は次の機能を追加します。
    • デバイス上のデフォルトのブラウザ アプリのロール アクションを選択します。
  • Wireless Manager は次の機能を追加します。
    • 機内モード タイマーを制御し、デバイスの機内モードが自動的にオフになるまでの時間 (分単位) を設定します。

v5.7 で追加

MX 11.3 のサポート、これにより、次の主要な機能と拡張機能が追加されます。

  • Key Mapping Manager は次の機能を追加します。
    • ダブル トリガ モードを割り当て、設定してアクティブにします。このモードでは、指定された期間中にスキャン トリガを 2 回押すたびにキー コードを送信できます。
  • Power Key Manager は次の機能を追加します。
    • デバイスの電源ボタンを長押しした後に表示されるメニューの [電源オフ] ボタンの表示/非表示を切り替えます。
  • Settings Manager は次の機能を追加します。
    • デバイス ユーザーが、通話の発信や受信など、特定の役割に使用されるアプリを変更するかどうかを制御します。
  • Touch Manager は、Zebra の TC53、TC58、および WS50 デバイスのサポートを追加します。
  • UI Manager は次の機能を追加します。
    • デバイスの近接センサーのオン/オフを切り替えます。
    • デバイスでのコールを開始および受信するために使用されるデフォルトのダイヤラ アプリを選択できます。
  • Wi-Fi は次の機能を追加します。
    • デバイス通信に使用する 6.0GHz 帯域を選択します。
    • 通信に使用する 6.0GHz チャネルを選択します。
    • すべてのデバイス通信にグローバル プロキシ サーバーを使用するかどうかを制御します。

以前の MX バージョンの次の機能もサポートしています。

  • ファイル マネージャは次の機能を追加します。
    • ファイルをアプリケーションのデバイスに展開し、その特定のアプリケーションにファイルへのアクセス権を付与します。

MX 11.4 は、Android 11 以降を実行しているデバイスでのみサポートされます。

    • サーバーへのカスタム DHCP オプションの送信を有効/無効にします
    • 指定された DHCP オプションを送信するか、または無効にします
    • 指定したカスタム DHCP オプションを使用して値を送信します
  • License Manager を強化して、次の新しい動作を追加しました。
    • デバイスからライセンスを返すアクティベーション ID を指定します
    • 1 つまたはすべてのライセンスを返したり、ライセンス ソースを削除したりするための、サーバーにわかりやすい名前を指定します
    • 1 つまたはすべてのライセンスを返すために使用するライセンス ソース タイプを選択します
  • Settings Manager を強化して、次の新しい動作を追加しました。
    • Android システム設定にアクセスするためのスライド ドロワを有効/無効にします
  • UI Manager を強化して、次の新しい動作を追加しました。
    • 分割画面モードを有効/無効にします
    • 非通知モードを有効/無効にします
    • マルチユーザー モードを有効/無効にします
  • Wi-Fi を強化して、次の新しい動作を追加しました。
    • MAC アドレスのランダム化を有効/無効にします
    • コール許可制御を有効/無効にします
    • ホットスポット状態 (アクティブ/非アクティブ) のユーザー コントロールを有効/無効にします
  • 新しい AutoTrigger Manager CSP を使用して、自動スキャン トリガを設定します。これにより、スキャン ターゲットがデバイス センサの近くに来るとスキャンが開始されます。現時点では、Zebra PS20 Personal Shopper デバイスのみがサポートされています。

  • 新しい Fota Manager CSP により、デバイス上の Firmware Over The Air (FOTA) クライアントを制御し、物理的な接続がなくても、管理者は Zebra デバイスで OS の更新を実行できます。FOTA Client アプリはサポート対象デバイスにプリインストールされており、Zebra アップデート サーバーと通信するように設定されています。

互換性に関する注意

  • StageNow 3.0 をサポートする Zebra Android デバイスの全リストを参照してください

  • Android Nougat および Marshmallow を実行している Zebra デバイスには StageNow クライアントが付属しており、StageNow ツールが提供するすべてのデバイス設定オプションをサポートします。

  • 特定のデバイスでサポートされる MX 機能は、デバイス BSP 内の Android、OSX、および MX のバージョンによって異なります。組織内のデバイスでサポートされている機能を確認するには、MX 機能マトリックスを参照してください。

  • 無料の StageNow アプリケーションは、必要なソフトウェアがなく、他のコンポーネントを購入、検索、ライセンス、またはインストールする必要のないすべての Windows 7 PC にダウンロードしてインストールできます。
