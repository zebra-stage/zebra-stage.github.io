StageNow のインストール

StageNow 5.6

概要

このセクションでは、Windows 用 StageNow のインストール、アンインストール、およびアップグレードの手順について説明します。

要件

StageNow デスクトップ ツールを実行するための条件は以下のとおりです。

  • Microsoft Windows 7 または Windows 10 Pro を搭載した PC
  • 2GB 以上の RAM
  • 6GB 利用可能なストレージ (.NET Framework がまだインストールされていない場合)
  • StageNow インストーラ (StageNow ダウンロード ページから入手可能)
  • Microsoft .NET Framework 4.5 以降 (StageNow インストーラに付属)
  • Java for Windows は Java ダウンロード ページから入手できます
  • PDF リーダー (ステージング バーコードの表示および出力用)
  • Wi-Fi アクセス (Wi-Fi ホットスポット ステージング用 (サポートされている場合))

Windows 10 は Wi-Fi ホットスポット ステージングをサポートしていません。

StageNow のインストール

StageNow デスクトップ ツールをインストールすると、Microsoft.NET Framework 4.5 (必要な場合)、および関連するステージング データベースとサーバー コンポーネントも PC にインストールされます。

アップグレードについては、StageNow Workstation ツールのアップグレードを参照してください。

StageNow をインストールするには:

  1. StageNow ダウンロード ページにアクセスして、セットアップ パッケージ staging_solution.[バージョン番号].exe をダウンロードします。

  2. セットアップ パッケージ ファイルをダブルクリックします。

  3. まだ .NET Framework 4.5 がインストールされていない場合は、インストールのプロセスによって .NET Framework 4.5 のインストールを要求するメッセージが表示されます。

  4. [インストール] を選択します。Framework のインストールが完了すると、StageNow のインストールの準備中であることを示すウィンドウがウィザードに表示されます。

  5. [次へ] を選択して続行します。

  6. 使用許諾契約の条項に同意し、[次へ] を選択します。

  7. [ユーザー名] と [会社名] を入力し、Workstation ツールの使用を許可するユーザーを指定します。

  8. [次へ] を選択します。

  9. 管理者のパスワードを入力し、[次へ] を選択します。
    注: すべての StageNow プロファイルとデータがこのパスワードに関連付けられます。

  10. [次へ] を選択してデフォルトのフォルダを使用するか、先に [参照] を選択して別のインストール フォルダを選択します。

  11. [次へ] を選択して現在の設定を使用するか、[戻る] を選択して設定を変更します。ウィザードに、インストール用のソフトウェアを設定中であることが示されます。そうなればインストールはほぼ完了です。

  12. 必要に応じて、チェックボックスで、デスクトップにショートカットを作成するか、StageNow Workstation ツールを起動するかを選択して、[次へ] を選択します。

  13. ステージング素材は PDF ファイルに表示されるため、PDF リーダーをダウンロードするか、すでにリーダーがインストールされている場合はこの作業をスキップするように求めるメッセージがウィザードに表示されます。適切なオプションを選択して [完了] を選択します。

これで、StageNow Workstation ツールを使用する準備ができました。

StageNow のアップグレード

ホスト PC に古いバージョン StageNow Workstation ツールが存在する場合は、アップグレードすることが可能です。

重要: アップグレードする前に、ホスト PC 上にある作業途中のプロファイルをすべて完了してください。これを怠ると、予期しない動作や望ましくない動作が発生する可能性があります。

StageNow Workstation ツールをアップグレードする方法:

  1. セットアップ パッケージ staging_solution.[バージョン番号].exe をダウンロードします。

  2. セットアップ パッケージ ファイルをダブルクリックします。InstallShield で既存のバージョンが検出され、次のメッセージが表示されます。

  3. アップグレードを続行する場合は [はい]、既存のバージョンを保持してアップグレードを中止する場合は [いいえ] を選択します。[はい] を選択すると、既存の StageNow データ (設定やプロファイルなど) を保持するか破棄するかを確認するメッセージが表示されます。

  4. 目的のオプションを選択して、アップグレードを続行します。注: 既存のデータを保持することを選択した場合、既存のパスワードが保持されるため、インストール プロセスで管理者パスワードが要求されません。

  5. インストール手順の残りの部分については、「StageNow Workstation ツールのインストール」を参照してください。

StageNow のアンインストール

StageNow Workstation ツールをアンインストールする方法:

  1. [コントロールパネル] > [プログラム] > [プログラムと機能] の順に選択します。

  2. [StageNow] をダブルクリックします。インストール ウィザードが開きます。

  3. ウィザードの手順に従って、アンインストールを完了します。

  4. [完了] を選択してアンインストールを完了します。

オンデバイス ステージング ソフトウェア

StageNow オンデバイス クライアント ソフトウェアは、Android KitKat 以降が動作するすべての Zebra デバイスをサポートしています。これらのデバイスは通常、StageNow クライアントがプレインストールされた状態で出荷されており、すべての StageNow デバイス構成オプションをサポートします。

一部の古い KitKat デバイスには StageNow クライアントが入っていませんが、代わりに Rapid Deployment (RD) クライアントが付属しています。ステージングでは、これらのデバイスが StageNow で生成された「RD バーコード」をスキャンし、RD クライアントをネットワークに接続して、StageNow クライアントと、KitKat デバイス用の MX 5.0 を自動インストールします。すると、StageNow Client を使用して、StageNow プロファイルのバーコードをスキャンして設定できるようになります。OS の制約により、このようなデバイスでは StageNow の機能が制限される場合があります。

以下の方法をご確認ください。

  • RD バーコードを生成する
  • 特定のデバイスでサポートされている機能を確認する
