StageNow 故障排除

StageNow 3.0

本部分讨论在 StageNow 工具中可能出现的错误，以及可能的解决方案。

Wi-Fi 热点故障排除

在 StageNow 工具中打开 Wi-Fi 热点时,可能会出现以下错误。

以下是您可以采取的解决问题措施。

禁用和重新启用 Wi-Fi 适配器

  1. 右键单击 Windows 主机右下角的“网络”图标，然后选择“打开网络和共享中心”以打开以下屏幕。

  2. 在左窗格中，选择“更改适配器设置”。

  3. 打开命令提示符并输入“netsh wlan show drivers”以查找 Wi-Fi 驱动程序适配器名称。

  4. 在“网络连接”窗口中右键单击网络适配器，然后选择“禁用”。

  5. 再次右键单击，然后选择“启用”。几分钟后，尝试使用 StageNow 工具。

添加不受支持的 Wi-Fi 热点

注意：与以下内容背离的结果可视为与 Wi-Fi 热点有关的 Windows 驱动程序/软件问题。

  1. 以管理员身份打开命令提示符。

  2. 输入“netsh wlan show drivers”并找到下面以绿色显示的概述结果消息。

    如果系统不支持 Wi-Fi 热点，则显示以下消息。

  3. 在命令提示符中输入以下命令："netsh wlan set hostednetwork mode=allow ssid=key=""

    注意：HotspotName 是 Wi-Fi 热点的名称。

    HotspotPassword 是 Wi-Fi 热点密码，必须至少为 5 个字符。

    成功创建 Wi-Fi 热点后，将显示以下结果。

  4. 输入“netsh wlan start hostednetwork”以启动热点。

  5. 输入“netsh wlan stop hostednetwork”以停止热点。
