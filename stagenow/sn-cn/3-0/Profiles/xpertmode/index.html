Xpert 模式

StageNow 3.0

Xpert 模式允许通过配置文件向导和设置类型提供的选项和流程进行增强控制以构建自定义配置文件。

在此模式下，管理员可以按任何顺序使用任何设置和/或 StageNow 工具中提供的任何向导。例如，管理员可以在“专家模式”向导内使用“配置设备”向导，但包括其他设置，以及标准向导中提供的那些设置。通过这种方式，专家管理员可以利用 StageNow 提供的灵活性来使用现有向导中未涵盖的功能。请注意，这并不适用于 Xpert 向导的 RD 部分；它仍会限制管理员选择某些设置。

注意：
在 StageNow 工具配置部分内创建的所有设置都存储在部署条码、音频文件或 NFC 标签中，而在部署部分中创建的设置驻留在部署服务器。

要创建 Xpert 模式配置文件：

  1. 选择“新建配置文件”。

  2. 从下拉菜单中选择 MX 版本。

  3. 选择 Xpert 模式向导，然后选择“创建”。

  4. 输入配置文件名称，然后选择“开始”。

    注意：创建配置文件期间，窗口右上角指示配置文件的状态，例如“不完整”。

  5. 在“添加/编辑”窗口，选择“设置”选项卡以将设置添加到 Xpert 模式配置文件，和/或“向导”选项卡以将配置文件向导添加到 Xpert 模式配置文件。

  6. 选择“配置”或“部署”选项卡以选择配置文件向导中要包括在 Xpert 模式下的部分。

  7. 选择每个设置和/或向导旁边的加号图标以添加到 Xpert 模式配置文件，按所需顺序执行。此操作将每一项目移动到右侧窗格（“配置”或“部署”）。

  8. 如在创建部署配置文件中所述，选择“添加”以在配置文件向导中移动，并创建新的 Xpert 模式。

更新 Xpert 模式配置文件

在创建配置文件期间，随时选择“添加/编辑”按钮可返回到“添加/编辑设置”和向导窗口，然后添加、修改或删除配置文件中的设置或向导。完成后选择“更新”。

注意：
打开在 StageNow 2.1 中创建的现有 Xpert 模式配置文件将配置文件导入之前所述已更新的 StageNow 2.2 Xpert 模式用户界面。

管理企业主页屏幕 (EHS)

企业主页屏幕定义哪些应用程序显示在企业设备上，因此只有某些功能可供用户使用。StageNow 2.2 使您可以在 Zebra 企业设备上通过管理企业主页屏幕子向导轻松配置企业主页屏幕。只需在 Xpert 模式配置文件中包括此子向导，并按照屏幕上的步骤进行操作。有关信息 EHS 配置文件和可用作 EHS 分发部分的 EHS APK，请参阅 EHS 用户指南。

在其他设置和子向导之中，包括作为 Xpert 模式配置文件任何步骤的 EHS 子向导。以下示例仅为简单起见使用 EHS 子向导。使用此子向导之前，在工作站上获取并提取 EHS 分布，其中包括 EHS_L 和 EHS_R APK。

要通过 Xpert 模式向导配置 EHS：

  1. 选择“新建配置文件”。

  2. 从下拉菜单选择 MX 4.4 或 MX 5.0 版本。

  3. 选择 Xpert 模式向导，然后选择“创建”。

  4. 输入配置文件名称，然后选择“开始”。

  5. 在“添加/编辑”窗口中，选择“向导”选项卡。

  6. 选择“管理企业主页屏幕”旁边的加号。

  7. 选择“添加”。

  8. 在“下载 EHS 配置”窗口中，选择“是”以将 EHS 配置文件下载到设备进行部署。有关创建此文件的更多信息，请参阅 EHS 文档。

  9. 浏览至并从工作站选择文件，然后选择“继续”。为易于使用，预配置设备路径。

  10. 选择 APK 的类型并进行安装，然后选择“继续”。有关设备和 EHS APK 映射信息，请参阅 EHS 文档。

  11. 选择 EHS_L（或 EHS_R）APK 的位置以推送至设备，然后选择“继续”。

  12. 在设备上安装推送的 APK，确保该路径正确，然后选择“继续”。

  13. 选择“继续”在设备上将 EHS 设置为默认启动屏幕。

  14. 请参阅审核以继续创建 EHS 配置文件。

企业主页屏幕子向导与其他设置和向导一起使用

企业主页屏幕配置文件中的一些配置参数可以强制设备重新启动以应用它们。使用其他设置和向导创建 Xpert 模式配置文件时，管理员在应用配置时应避免此中途重新启动。设备重新启动后，StageNow 客户端和 MX 框架无法恢复部署过程。如果需要，我们建议您在向导的最后一步重新启动设备。

以下项目强制重新启动设备，并列出实现相同结果的等效 StageNow 设置。使用 EHS 向导与其他设置一起应用配置文件时，这些设置避免不受控制的重新启动：

  • 在快速设置面板中禁用 Wi-Fi 和蓝牙开/关：使用 Wireless Manager 管理蓝牙和使用 Wi-Fi Manager 管理 Wi-Fi。

  • 在快速设置面板中禁用飞行模式开/关：使用 Settings Manager 管理飞行模式。

  • 清除最近的应用程序列表：使用 Power Manager 重新启动（使其成为配置文件中的最后一步）。
