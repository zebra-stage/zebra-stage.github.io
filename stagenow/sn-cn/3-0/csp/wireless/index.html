关于 SmartDocs


“SmartDocs”栏可以自定义此页面，仅显示在特定 Zebra 设备上存在的功能。

  1. 使用下拉列表选择设备上的 OSX、MX 和/或 Android 版本，以及或者设备型号，它突出显示了设备中支持的功能。根据所做的选择，不受支持的功能会变灰。
  2. SmartDocs Nav 在查看单个 CSP 的功能时发挥作用。要使用它，请单击一个 CSP（例如“App Manager”），然后在 SmartDocs 栏中进行至少一项选择。不支持的功能将变为灰色。单击前进和后退 SmartNav 按钮可滚动到该 CSP 下一个或上一个支持的功能（SmartNav 在功能矩阵上不受支持）。
  3. 要清除所有选择，请按 SmartDocs 按钮。

在 Android“设置”面板或通过 ADB、EMDK 或 MX CSP 查询设备，可以找到设备的 OSX、MX 和 Android 版本信息。更多信息。

  • OSX 层提供了对基本操作系统的扩展，以实施 Android 未提供的功能。根 OSX 版本号始终与它扩展的 Android 版本的根号码匹配。更多信息。
  • MX 管理系统为设备上的特权和无特权 Android API 提供了一个统一的接口。某些功能需要对 MX 和 OSX 版本进行具体的配对。更多信息。
  • Android 版本有时足以确定功能集。其他情况则需要一个特定的 MX 版本加上 Android 的最低版本。
  • 设备型号可指明在设备上提供 MX 功能的哪个子集。

概述

Wireless Manager (WirelessMgr) 控制设备上所有无线电的使用，这些无线电因设备而异，可能包括蓝牙、GPS、NFC 和 WWAN 无线电。通过控制设备的无线电，WirelessMgr 允许管理员阻止设备用户访问给定无线电的任意功能或他们所连接的设备或网络的功能，从而提供了设备用户无法渗透的安全措施。

WirelessMgr 还可控制 GPS 定位模式以帮助节约电池寿命，在某些设备上还能控制蓝牙无线电的状态（活动/非活动）。

一定要注意区分“状态”和“使用”这两个词。无线电的“状态”是指该无线电是打开（可用于通信）还是关闭（非活动）。“使用”确定在设备上是否已启用无线电。只有当“使用”已打开（已启用）时，设备用户或管理员才能控制无线电的状态。

Android 设备通常配有各种无线通信接口。这些接口包括蓝牙、无线广域网（WWAN/蜂窝数据）、近场通信 (NFC) 等双向接口以及（仅接收）全球定位系统 (GPS) 等单向接口。能够发射信号的无线电被不必要地打开时，可能严重影响电池寿命。在某些情况下可能还必须关闭有些无线电，例如在空中旅行期间或在危险环境中。

主要功能

  • 启用/禁用用户对设备无线电的访问：
    • 蓝牙
    • 近场通信 (NFC)
    • 全球定位系统 (GPS)
    • 无线广域网（WWAN/蜂窝数据）
  • 控制蓝牙无线电的状态
  • 选择内部或外部蓝牙/Wi-Fi 天线
  • 选择 GPS 定位模式以控制电池使用：
    • 高精度
    • 省电
    • 仅设备

天线选择

用于在设备上用于蓝牙和 Wi-Fi (WLAN) 通信的内置天线和外部天线之间选择。此功能仅适用于 Zebra VC80x 和类似配备的车载数据终端。

参数名称：AntennaSelection

选项 名称 说明需要
0 不更改 此值（或 XML 中缺少此参数）导致不会更改设置；任何以前选择的设置都将保留。

MX：7.1+
1 使用内部天线 设置设备使用其内置蓝牙/Wi-Fi 天线。

MX：7.1+
2 使用外部天线 设置设备使用（可选的）外部蓝牙/Wi-Fi 天线。

MX：7.1+

蓝牙启用/禁用

对于已打开蓝牙无线电的 KitKat 设备，它控制设备上的蓝牙功能是否处于活动状态。

在 Jelly Bean 设备上，这是蓝牙无线电的开/关键，它控制在设备上是否能使用蓝牙。如果已启用，则 Jelly Bean 设备用户可以配对和连接其他蓝牙设备，以及更改蓝牙状态（活动/非活动）。

参数名称：蓝牙

选项 名称 说明需要
0 不更改 此值（或 XML 中缺少此参数）导致不会更改蓝牙无线的状态；任何以前选择的设置都将保留。

OSX：4.2+

MX：4.2+
1 启用 在 KitKat 设备上，将蓝牙无线电（如果已打开）置于活动状态。在 Jelly Bean 设备上，打开蓝牙无线电，允许在设备上使用它以及更改它的状态。

OSX：4.2+

MX：4.2+
2 禁用 在 KitKat 设备上，将蓝牙无线电（如果已打开）置于非活动状态。在 Jelly Bean 设备，关闭 Bluetooth 无线电，阻止在设备上使用它以及更改它的状态。

OSX：4.2+

MX：4.2+

蓝牙打开/关闭

蓝牙无线电的开/关键，它控制该无线电的状态（活动/非活动）是否能够被设备用户更改。打开蓝牙无线电，使设备能够与其他蓝牙设备配对，以及连接以前配对的设备。

注意：此参数仅在运行 KitKat 和更高版本的设备上可用。

参数名称：BluetoothState

选项 名称 说明需要
0 不更改 此值（或 XML 中缺少此参数）导致不会更改蓝牙无线电状态；任何以前选择的设置都将保留。

MX：4.3+

Android API：5+
1 打开 打开蓝牙无线电，允许使用和更改其状态。

MX：4.3+

Android API：5+
2 关闭 关闭蓝牙无线电，阻止所有使用。

MX：4.3+

Android API：5+

GPS 定位模式

允许选择用于确定设备位置的 GPS 定位模式，允许出现为了获得更长的电池使用时间而牺牲定位精度的情形。此参数使用 Android 的三种定位模式：

  • 高精度模式将结合使用来自内置 GPS 接收器的信息以及 Wi-Fi 和蜂窝数据，按最高的精确度计算位置。此选项耗能最多。

  • 省电模式仅使用 Wi-Fi 和蜂窝数据来确定位置，但是不能用于 Google 地图和类似应用程序。此选项耗能最少。

  • 仅设备模式仅使用 GPS 数据，但是在某些设置中会遇到信号丢失。

参数名称：GPSLocationMode

选项 名称 说明需要
0 不更改 此值（或 XML 中缺少此参数）导致不会更改 GPS 定位模式；任何以前选择的设置都将保留。

MX：6.1+

Android API：1+
1 高精度 使用 GPS、 Wi-Fi 和蜂窝网络信息，按最高的准确度确定设备位置。

MX：6.1+

Android API：1+
2 省电 仅使用 Wi-Fi 和蜂窝信息确定设备位置，节约电池电量。

MX：6.1+

Android API：1+
3 仅设备 仅使用 GPS 无线电确定位置，在有些设置中会遇到信号丢失。

MX：6.1+

Android API：1+

示例

允许对蓝牙状态进行更改

注意：在 JellyBean 设备上，此 XML 通过打开蓝牙无线电来更改它的状态。

<wap-provisioningdoc><characteristic type="WirelessMgr" version="4.3" ><parm name="Bluetooth" value="1"/></characteristic></wap-provisioningdoc>

禁止对蓝牙状态进行更改

注意：在 JellyBean 设备上，此 XML 通过关闭蓝牙无线电来更改它的状态。

<wap-provisioningdoc><characteristic type="WirelessMgr" version="4.3" ><parm name="Bluetooth" value="2"/></characteristic></wap-provisioningdoc>

打开所有无线电（对于 JellyBean 设备）

<wap-provisioningdoc><characteristic type="WirelessMgr" version="4.3" ><parm name="Bluetooth" value="1"/><parm name="NFCState" value="1"/><parm name="GPSState" value="1"/><parm name="WWANState" value="1"/></characteristic></wap-provisioningdoc>

关闭所有无线电（对于 JellyBean 设备）

<wap-provisioningdoc><characteristic type="WirelessMgr" version="4.3" ><parm name="Bluetooth" value="2"/><parm name="NFCState" value="2"/><parm name="GPSState" value="2"/><parm name="WWANState" value="2"/></characteristic></wap-provisioningdoc>

打开所有无线电（对于 KitKat 设备）

<wap-provisioningdoc><characteristic type="WirelessMgr" version="4.3" ><parm name="BluetoothState" value="1"/><parm name="NFCState" value="1"/><parm name="GPSState" value="1"/><parm name="WWANState" value="1"/></characteristic></wap-provisioningdoc>

关闭所有无线电（对于 KitKat 设备）

<wap-provisioningdoc><characteristic type="WirelessMgr" version="4.3" ><parm name="BluetoothState" value="2"/><parm name="NFCState" value="2"/><parm name="GPSState" value="2"/><parm name="WWANState" value="2"/></characteristic></wap-provisioningdoc>

为蓝牙和 Wi-Fi 选择外部天线

<wap-provisioningdoc><characteristic type="WirelessMgr"><parm name="AntennaSelection" value="2" /></characteristic></wap-provisioningdoc>
