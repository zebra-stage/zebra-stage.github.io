关于 SmartDocs


“SmartDocs”栏可以自定义此页面，仅显示在特定 Zebra 设备上存在的功能。

  1. 使用下拉列表选择设备上的 OSX、MX 和/或 Android 版本，以及或者设备型号，它突出显示了设备中支持的功能。根据所做的选择，不受支持的功能会变灰。
  2. SmartDocs Nav 在查看单个 CSP 的功能时发挥作用。要使用它，请单击一个 CSP（例如“App Manager”），然后在 SmartDocs 栏中进行至少一项选择。不支持的功能将变为灰色。单击前进和后退 SmartNav 按钮可滚动到该 CSP 下一个或上一个支持的功能（SmartNav 在功能矩阵上不受支持）。
  3. 要清除所有选择，请按 SmartDocs 按钮。

在 Android“设置”面板或通过 ADB、EMDK 或 MX CSP 查询设备，可以找到设备的 OSX、MX 和 Android 版本信息。更多信息。

  • OSX 层提供了对基本操作系统的扩展，以实施 Android 未提供的功能。根 OSX 版本号始终与它扩展的 Android 版本的根号码匹配。更多信息。
  • MX 管理系统为设备上的特权和无特权 Android API 提供了一个统一的接口。某些功能需要对 MX 和 OSX 版本进行具体的配对。更多信息。
  • Android 版本有时足以确定功能集。其他情况则需要一个特定的 MX 版本加上 Android 的最低版本。
  • 设备型号可指明在设备上提供 MX 功能的哪个子集。

概述

Browser Manager 允许配置 Android 开源项目 (AOSP) 浏览器上的密码保留、默认主页和其他设置，AOSP 浏览器是许多 Zebra Android 设备上的默认浏览器，已针对此用途进行了修改。

主要功能

  • 设置默认主页
  • 记住密码启用/禁用
  • 保存表单数据启用/禁用

设置默认主页

用于为设备上的默认 Android Web 浏览器设置统一资源定位符 (URL) 形式的默认主页。每次在没有用户明确请求的 URL 的情况下打开默认浏览器时，它将使用此参数提供的 URL。应该为对应于默认网页的完整有效 URL（包括协议），如“http://www.google.com”。这可用于启动一个特定的基于 Web 的应用程序，或者将用户定向到内部公司网站或门户作为浏览的起点。指定空值（零长度）（或 XML 中缺少此参数）将导致没有更改；任何以前指定的默认主页将被保留。

参数值输入规则：

  • 0 到 2000 个字符的字符串

参数名称：SetDefaultHomepage

需要：

  • OSX：1.3- 3.9
  • 不适用于 OSX：4.0
  • MX：4.3+

记住密码启用/禁用

用于启用/禁用默认 Android Web 浏览器的“记住密码”选项。

启用记住密码可以让浏览更加方便，不再需要设备用户在随后访问网站时输入密码。禁用记住密码通常会更加安全，尤其是在一个可能被共享的设备上，这可以防止未经授权的用户使用授权用户先前输入的密码登录。

注意：此功能仅在 Jelly Bean 设备上可用；它在 Android KitKat 和更高版本上已弃用。

参数名称：SetRememberPasswords

选项 名称 说明需要
0 不更改 此值（或 XML 中缺少此参数）导致不会更改记住密码选项；任何以前选择的设置都将保留。

OSX：1.3 - 3.9

不适用于 OSX：4.0

MX：4.3+
1 打开 启用“记住密码”选项，这会提示用户保存输入的任何密码。

OSX：1.3 - 3.9

不适用于 OSX：4.0

MX：4.3+
2 关闭 禁用“记住密码”选项，可阻止对密码的任何保存。

OSX：1.3 - 3.9

不适用于 OSX：4.0

MX：4.4+

保存表单数据启用/禁用

用于启用/禁用默认 Android Web 浏览器的“保存表单数据”选项。

启用保存表单数据可以让浏览更加方便，不再需要设备用户在随后访问网站时重新输入信息。禁用保存表单数据通常会更加安全，尤其是在一个可能被共享的设备上，因为这可以防止未经授权的用户查看或重新输入授权用户先前输入的敏感信息。

参数名称：SetSaveFormData

选项 名称 说明需要
0 不更改 此值（或 XML 中缺少此参数）导致不会更改保存表单数据选项；任何以前选择的设置都将保留。

OSX：1.3 - 3.9

不适用于 OSX：4.0

MX：4.3+
1 打开 启用“保存表单数据”选项，这将提示设备用户保存表单数据。

OSX：1.3 - 3.9

MX：4.3+
2 关闭 禁用“保存表单数据”选项，阻止保存保存任何表单数据。

OSX：1.3 - 3.9

不适用于 OSX：4.0

MX：4.3+

示例

查询

获取所有浏览器设置

输入

<wap-provisioningdoc><characteristic-query type = "BrowserMgr"/></wap-provisioningdoc>

输出

<wap-provisioningdoc><characteristic type="BrowserMgr" version="4.3"><parm name="SetDefaultHomepage" value="http://www.google.com"/><parm name="SetRememberPasswords" value="1"/><parm name="SetSaveFormData" value="1"/></characteristic></wap-provisioningdoc>

获取默认主页设置

输入

<wap-provisioningdoc><characteristic type = "BrowserMgr"><parm-query name="SetDefaultHomepage"/></characteristic></wap-provisioningdoc>

输出

<wap-provisioningdoc><characteristic type="BrowserMgr" version="4.3"><parm name="SetDefaultHomepage" value="http://www.google.com"/></characteristic></wap-provisioningdoc>

获取记住密码设置

输入

<wap-provisioningdoc><characteristic type = "BrowserMgr"><parm-query name="SetRememberPasswords"/></characteristic></wap-provisioningdoc>

输出

<wap-provisioningdoc><characteristic type="BrowserMgr" version="4.3"><parm name="SetRememberPasswords" value="1"/></characteristic></wap-provisioningdoc>

获取保存表单数据设置

输入

<wap-provisioningdoc><characteristic type = "BrowserMgr"><parm-query name="SetSaveFormData"/></characteristic></wap-provisioningdoc>

输出

<wap-provisioningdoc><characteristic type="BrowserMgr" version="4.3"><parm name="SetSaveFormData" value="1"/></characteristic></wap-provisioningdoc>
