关于 SmartDocs


“SmartDocs”栏可以自定义此页面，仅显示在特定 Zebra 设备上存在的功能。

  1. 使用下拉列表选择设备上的 OSX、MX 和/或 Android 版本，以及或者设备型号，它突出显示了设备中支持的功能。根据所做的选择，不受支持的功能会变灰。
  2. SmartDocs Nav 在查看单个 CSP 的功能时发挥作用。要使用它，请单击一个 CSP（例如“App Manager”），然后在 SmartDocs 栏中进行至少一项选择。不支持的功能将变为灰色。单击前进和后退 SmartNav 按钮可滚动到该 CSP 下一个或上一个支持的功能（SmartNav 在功能矩阵上不受支持）。
  3. 要清除所有选择，请按 SmartDocs 按钮。

在 Android“设置”面板或通过 ADB、EMDK 或 MX CSP 查询设备，可以找到设备的 OSX、MX 和 Android 版本信息。更多信息。

  • OSX 层提供了对基本操作系统的扩展，以实施 Android 未提供的功能。根 OSX 版本号始终与它扩展的 Android 版本的根号码匹配。更多信息。
  • MX 管理系统为设备上的特权和无特权 Android API 提供了一个统一的接口。某些功能需要对 MX 和 OSX 版本进行具体的配对。更多信息。
  • Android 版本有时足以确定功能集。其他情况则需要一个特定的 MX 版本加上 Android 的最低版本。
  • 设备型号可指明在设备上提供 MX 功能的哪个子集。

概述

Wi-Fi Manager (Wi-Fi) 管理设备的无线 LAN 设置和网络配置文件，包括连接到网络所需的设置。Wi-Fi CSP 还控制 OmniTrail，它是某些 Zebra 设备中提供的一组服务，可实现高度准确、安全和高能效的设备定位。OmniTrail 可以将设备定位到大约 5 英尺（1.5 米）以内。参见下面的 OmniTrail 要求。

Wi-Fi 启用/禁用

这是 Wi-Fi 无线电的开/关键。注意：必须启用 Wi-Fi 才能更改其他某些 Wi-Fi 设置，包括通过本文档后面介绍的“网络操作”参数控制的那些设置。尝试在 Wi-Fi 无线电禁用时更改此类设置将会在结果 XML 中返回错误。Zebra 建议从任意创建的配置文件开始更改 Wi-Fi 设置，使用一个命令启用 Wi-Fi 无线电。

小心：某些 Wi-Fi 参数因其他 CSP 使用的约定而异：

  • Wi-Fi CSP 中的“启用”和“禁用”设置控制用户或管理员是否可以“打开”和“关闭”功能，或者将其状态设置为“活动”或“非活动。”其他 CSP 中的此类术语指的是功能的“使用”设置，可能完全阻止访问功能。
  • 在 Wi-Fi 参数中使用“0”值是“禁用”某项功能，选择它的默认设置或者对于设置“不进行更改”或“无操作”。在其他大多数 CSP 中，“0”值用于选择“不进行更改”选项。

Wi-Fi 睡眠策略

指定设备进入睡眠模式时 Wi-Fi 无线电的状态。当 Wi-Fi 在睡眠期间保持启用状态时，现有的 Wi-Fi 连接将维持其在正常操作期间的状态。虽然这在某些情形下是首选，但是此模式可能显著减少电池命寿命。.

Android 设备默认情况下在闲置指定的一段时间后会进入睡眠模式。当处于此模式时，应用程序以低功耗状态运行。

选项 名称 说明需要
4 不更改 此（空）值（或 XML 中缺少此参数）导致不会更改设备上的当前睡眠策略；任何以前选择的设置都将保留。

MX：4.2+
0 AlwaysOn 导致 Wi-Fi 无线电在设备进入睡眠保持启用，为正在运行的应用程序提供无间断的 Wi-Fi 访问。

MX：4.2+
1 PluggedIn 导致只要设备接通了电源而不是靠电池电量运行时，无论是否为睡眠状态，都会使 Wi-Fi 无线电保持启用状态，为正在运行的应用程序提供无间断的 Wi-Fi 访问。

设置 2.4 GHz 信道

用于指定 2.4GHz 频段中供设备使用的信道。虽然此频段中使用的信道通常是通过与 Wi-Fi 基础设施协商后自动确定，但是也可以手动指定信道或信道集，以避免干扰其他设备，或者将某些信道专用于特定用途。警告：并非所有信道在所有国家/地区都可用。有关更多信息，请参见 WLAN 信道列表。

参数值输入规则：

  • 0 到 64 个字符的字符串，其中包含一组有效信道
  • 多个值必须用逗号分隔（例如“1,6,11,14”）
  • 还接受使用连字符(-) 指定的范围（例如“1,7-10”）

  • 指定空值（零长度）（或 XML 中缺少此参数）将无法设置可选服务器证书。

OmniTrail 间隔

用于指定发出定位信标的间隔（以毫秒为单位）。较为频繁的信标发射对于在室内或有严格界限的空间内确定设备位置通常更有效。

参数值输入规则：

  • 200 到 5000 的整数（以毫秒为单位），表示发射定位信标的所需间隔
