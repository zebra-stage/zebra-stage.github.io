关于 SmartDocs


“SmartDocs”栏可以自定义此页面，仅显示在特定 Zebra 设备上存在的功能。

  1. 使用下拉列表选择设备上的 OSX、MX 和/或 Android 版本，以及或者设备型号，它突出显示了设备中支持的功能。根据所做的选择，不受支持的功能会变灰。
  2. SmartDocs Nav 在查看单个 CSP 的功能时发挥作用。要使用它，请单击一个 CSP（例如“App Manager”），然后在 SmartDocs 栏中进行至少一项选择。不支持的功能将变为灰色。单击前进和后退 SmartNav 按钮可滚动到该 CSP 下一个或上一个支持的功能（SmartNav 在功能矩阵上不受支持）。
  3. 要清除所有选择，请按 SmartDocs 按钮。

在 Android“设置”面板或通过 ADB、EMDK 或 MX CSP 查询设备，可以找到设备的 OSX、MX 和 Android 版本信息。更多信息。

  • OSX 层提供了对基本操作系统的扩展，以实施 Android 未提供的功能。根 OSX 版本号始终与它扩展的 Android 版本的根号码匹配。更多信息。
  • MX 管理系统为设备上的特权和无特权 Android API 提供了一个统一的接口。某些功能需要对 MX 和 OSX 版本进行具体的配对。更多信息。
  • Android 版本有时足以确定功能集。其他情况则需要一个特定的 MX 版本加上 Android 的最低版本。
  • 设备型号可指明在设备上提供 MX 功能的哪个子集。

概述

在运行 Android 5.1 (Lollipop) 或更高版本的 Zebra 设备中，可使用 Remote Scanner Manager (RemoteScannerMgr) 监控、配置受支持的 Zebra 蓝牙扫描器的固件更新，将固件更新应用到这些扫描器。

支持 DS3678 扫描器需要 MX 6.3 或更高版本。

主要功能

  • 更新已连接的扫描器的固件（文件单独下载）
  • 在已配对的 RS6000 或 DS3678 扫描器上执行以下功能：
    • 应用 Scan123 配置文件（单独下载）
    • 重启扫描器
    • 启动设备定位 LED/声音序列
    • 断开扫描器
    • 取消配对扫描器
  • 执行要执行操作的设备的序列号

操作

用于指定在与设备配对并连接的远程蓝牙扫描器上启动管理功能的操作。如果执行固件更新或其他文件相关的功能，必须先使用 File Manager、StageNow（兼容的 MDM 解决方案）或其他方式获取文件。支持 Zebra DS3678 扫描器需要 MX 6.3 或更高版本。

参数名称：操作

选项 名称 说明需要
0 无操作 此值（或 XML 中缺少此参数）将导致不执行扫描器管理操作。

MX：5.2+
1 应用 123Scan 配置包（参见说明） 应用 123Scan (.scncfg) 配置包以配置扫描器。文件单独下载。仅支持 RS6000 和 DS3678；DS 需要 MX 6.3 或更高版本。

MX：5.2+
2 更新扫描器固件 应用在对应字段中指定的固件（.dat 文件单独下载）以更新扫描器固件。

MX：5.2+
3 重置扫描器（参见说明） 重启连接的扫描器，以清除它的内存和其他易失性设置。仅支持 RS6000 和 DS3678；DS 需要 MX 6.3 或更高版本。

MX：5.2+
4 寻呼扫描器（参见说明） 激活已配对设备上的设备定位序列，它包括蜂鸣声、LED 指示灯闪烁和振动，可帮助找到放错了地方的设备。仅支持 RS6000 和 DS3678；DS 需要 MX 6.3 或更高版本。

MX：5.2+
5 断开蓝牙扫描器（参见说明） 断开当前连接的蓝牙扫描器。仅支持 RS6000 和 DS3678；DS 需要 MX 6.3 或更高版本。

MX：5.2+
6 取消配对蓝牙扫描器（参见说明） 从设备中删除选定蓝牙扫描器的配对信息。支持 RS6000 和 DS3678；DS 需要 MX 6.3 或更高版本。

MX：5.2+

123Scan 配置文件名

用于指定 123Scan 配置文件的路径和文件名，该配置文件使用 Zebra 的 Windows 版 123Scan 配置实用程序生成。

参数值输入规则

  • 1 到 255 个字符的字符串，指定 123Scan 配置文件的路径和名称

在以下条件下显示：扫描器操作不是“无操作”

参数名称：ConfigFile

需要：

  • MX：5.2+

固件文件名

用于指定用来更新扫描器固件的路径和文件名。

参数值输入规则

  • 接受表示固件文件的路径和名称的字符串。
  • 最小长度为 1 个字符；最大长度为 255 个字符

在以下条件下显示：扫描器操作不是“无操作”

参数名称：DatFile

需要：

  • MX：5.2+

扫描器序列号

用于指定要执行操作的扫描器的序列号。此参数可用于验证通过序列号识别的扫描器就是当前连接的设备。仅当指定的序列号与当前连接的扫描器的序列号匹配时（或者，在此参数留空或者不在 XML　中时），才执行该操作。如果指定的序列号与连接的扫描器的序列号不匹配，则不会执行任何操作，并且在结果　XML　中返回错误。

参数值输入规则

  • 接受 1　到　255 个字符的字符串

在以下条件下显示：扫描器操作不是“无操作”

参数名称：SerialNumber

需要：

  • MX：5.2+
