Status Manager

Stagenow - 2.2

Overview

The Status Manager allows you to set notification options while MX is processing the XML stream.

Status Request Action

This parm allows you to specify the action to take for status requests.

Parm Name: RequestAction

Pivotal Parm: Yes

Versions: MX 4.2 +

Option Name Description Versions
1 Add/Replace Status Request Add or replace the a status request MX 4.2+
2 Remove Status Request Remove a status request MX 4.2+
3 Notify Status Set the notification state MX 4.2+

Request Status Filter

Shown if: The Status Request Action is "Add/Replace Status Request"

Parm Name: RequestFilter

Pivotal Parm: No

Versions: MX 4.2 +

Option Name Description Versions
0 Request ALL Status MX 4.2+
1 Request Completion Status MX 4.2+

Android Action Name

This parm allows you to specify the action name of the Android package that this status notification pertains to.

Parm value input rules:

  • String with a minimum size of 1

Shown if: The Status Request Action is "Add/Replace Status Request"

Parm Name: ActionName

Pivotal Parm: No

Versions: MX 4.2 +

Package Name

This parm allows you to specify the package name of the Android package that this status notification pertains to.

Parm value input rules:

  • String with a minimum size of 1

Shown if: The Status Request Action is "Add/Replace Status Request" or "Remove State Request"

Parm Name: PackageName

Pivotal Parm: No

Versions: MX 4.2 +

Class Name

This parm allows you to specify the class name of the Android package that this status notification pertains to.

Parm value input rules:

  • String with a minimum size of 1

Shown if: The Status Request Action is "Add/Replace Status Request" or "Remove State Request"

Parm Name: ClassName

Pivotal Parm: No

Versions: MX 4.2 +

Source ID

This parm allows you to specify identifier string of the status notification.

Parm value input rules:

  • String with a minimum size of 1

Shown if: The Status Request Action is "Notify Status"

Parm Name: NotifySourceId

Pivotal Parm: No

Versions: MX 4.2 +

Notify State

This parm allows you to specify the action to take for status requests.

Parm Name: NotifyState

Pivotal Parm: Yes

Versions: MX 4.2 +

Option Name Description Versions
0 None MX 4.2+
1 In progress Notify the percent of progress MX 4.2+
3 Complete Notify when complete MX 4.2+

Notify Body

This parm allows you to specify text to send in the status notification.

Parm value input rules:

  • String with a minimum size of 1

Shown if: The Status Request Action is "Notify Status"

Parm Name: NotifyBody

Pivotal Parm: No

Versions: MX 4.2 +
