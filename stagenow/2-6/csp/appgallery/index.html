About SmartDocs


The "SmartDocs" bar can customize this page to show only the features present on a particular Zebra device.

  1. Use drop-downs to select the OSX, MX and/or Android version on the device, and/or the device Type, which determines an MX feature subset. Unsupported features will turn gray based on selection(s).
  2. With at least one version selected when viewing a CSP (i.e. "App Manager"), click forward and back buttons to scroll to the next and previous supported feature of that CSP (SmartNav is supported only on CSP pages, not on the feature matrix).
  3. To clear all selections press the SmartDocs button.

OSX, MX and Android version information for a device can be found in the Android Settings panel or by querying the device through ADB, EMDK or the MX CSP. More info

  • The OSX layer provides extensions to the base operating system to implement functionality not offered by Android. The root OSX version number always matches the root number of the Android version that it extends. More info
  • The MX Management System provides a uniform interface into privileged and unprivileged Android APIs on the device. Some features require a specific pairing of MX and OSX versions. More info
  • The Android version is sometimes enough to determine a feature set. Others require a specific MX version plus a minimum version of Android.
  • The Device Group indicates which subset of MX features is present in the device.

Overview

AppGalleryMgr enables administrators to remotely configure the AppGallery client app on a device using Zebra StageNow or a third-party Mobile Device Management (MDM) solution with Zebra MDM Toolkit. AppGallery is an enterprise infrastructure provided by Zebra to organizations allowing them to deploy, distribute and maintain applications for internal use or for sale to the general public.

Main Functionality

  • Enable/disable the AppGallery client app on a device
  • Enable/disable client-app notifications
  • Limit client access to apps authorized by an organization
  • Specify a user name and password
  • Set client app to use default credentials
  • Persist settings following an enterprise device reset

AppGallery Enable/Disable

This is the on/off switch for the AppGallery client app on the device. Disabling AppGallery removes its icon from the Android Launcher but does not remove the app itself; it can be switched on and off as desired and will retain all prior settings, including user credentials. Note: Client settings persist following an enterprise reset.

Parm Name: State

Option Name Description Device Group Requires
0 Do Not Change This value (or the absence of this parm from the XML) will cause no changes; any previously selected setting will be retained. A

MX: 6.0+

1 Turn On Enables the AppGallery client app on the device. A

MX: 6.0+

2 Turn Off Disables the AppGallery client app on the device. A

MX: 6.0+

Use Default Credentials

Used to enable/disable the default log-in credentials built into the AppGallery client app. This allows access only to apps in the public marketplace. To configure a device for access to private apps and collections, see Set Credentials and Organization Key.

Shown if: AppGallery Client State is set to "Turn On"

Parm Name: UseDefaultCreds

Option Name Description Device Group Requires
0 Do Not Change This value (or the absence of this parm from the XML) will cause no changes; any previously selected setting will be retained. A

MX: 6.0+

1 Turn On Enables the AppGallery client app to accept default log-in credentials. A

MX: 6.0+

2 Turn Off Prevents the AppGallery client app from accepting default log-in credentials. A

MX: 6.0+

Set Credentials

Used to enable/disable the setting of log-in credentials for a device. Enable this parameter to provision a device with a new User Name and Password corresponding to an account on the AppGallery server. Such credentials are required for access to private app collections. User credentials persist following an enterprise reset.

Shown if: AppGallery Client State is set to "Turn On" and Use Default Credentials is set to "Disable"

Parm Name: SetCredentials

Option Name Description Device Group Requires
0 Do Not Change This value (or the absence of this parm from the XML) will cause no changes; any previously selected setting will be retained. A

MX: 6.0+

1 Turn On Enables an administrator to set log-in credentials for the device. A

MX: 6.0+

2 Turn Off Disables the setting of log-in credentials for the device. A

MX: 6.0+

User Name

Used to specify a User Name, which must correspond with a user account configured on the AppGallery server. The User Name (with corresponding Password and Organization Key) enables AppGallery on the device to access private apps and collections made available by the organization. AppGallery can otherwise access only public-facing marketplace apps.

Parm value input rules:

  • String containing the User Name as it appears in the Admin screen
  • The minimum length is 1 character

Shown if: Set Credentials is set to "Enable" and Use Default Credentials is set to "Disable"

Parm Name: Username

Requires:

  • MX: 6.0+

Password

Used to specify a user Password, which must correspond with a user account configured on the AppGallery server. The Password (with corresponding User Name and Organization Key) enables AppGallery on the device to access private apps and collections made available by the organization.

Parm value input rules:

  • String containing the Password as it appears in the Admin screen (hidden on the client)
  • The minimum length is 8 characters

Shown if: Set Credentials is set to "Enable" and Use Default Credentials is set to "Disable"

Parm Name: Password

Requires:

  • MX: 6.0+

Organization Key

Used to specify the Organization Key, which is found on the administrator's Profile page of the AppGallery server. Copy this value and paste it into the field; the value will remain hidden on the client. Use of this key will prevent the AppGallery client from viewing any apps or collections other than those designated by the administrator.

Parm value input rules:

  • String containing the Organization Key copied from the AppGallery administrator Profile page

Shown if: Set Credentials is set to "Enable" and Use Default Credentials is set to "Disable"

Parm Name: OrganizationKey

Requires:

  • MX: 6.0+

Notifications Enable/Disable

Used to enable/disable AppGallery client Notifications on the device. Notifications are user-configurable beeps, vibrations and/or pop-ups that alert the user of updates and other events relating to AppGallery apps currently in use or that they have been invited to use.

Shown if: AppGallery Client State is set to "Turn On"
