SmartDocs について


[SmartDocs] バーでは、特定の Zebra デバイスに存在する機能のみを表示するように、このページをカスタマイズできます。

  1. ドロップダウンを使用してデバイスの OSX、MX、Android のバージョンを選択するか、デバイスのモデルを選択すると、デバイスでサポートされている機能が強調表示されます。サポートされていない機能は、選択内容に基づいてグレー表示になります。
  2. SmartDocs Nav は、個々の CSP の機能を表示するときに機能します。使用するには、CSP (つまり「App Manager」) をクリックし、SmartDocs バーで少なくとも 1 つを選択します。サポートされていない機能はグレー表示になります。その CSP でサポートされている機能の間を前後にスクロールさせるには、SmartNav の [進む] ボタンと [前へ] ボタンをクリックします (機能マトリックスでは SmartNav はサポートされていません)。
  3. すべての選択をクリアするには、[SmartDocs] ボタンを押します。

デバイスの OSX、MX および Android のバージョン情報は、Android の設定パネルに表示されるか、ADB、EMDK、または MX CSP を使用してデバイスを照会することで確認できます。詳細情報。

  • OSX レイヤーは、Android で提供されていない機能を実装するために、基本オペレーティング システムの拡張機能を提供します。ルート OSX のバージョン番号は、拡張対象とする Android バージョンのルート番号と常に一致します。詳細情報。
  • MX 管理システムは、デバイス上の特権および非特権 Android API に対して、統一されたインタフェースを提供します。一部の機能では、MX バージョンと OSX バージョンの特定のペアリングが必要です。詳細情報。
  • 機能セットを決定するには、Android バージョンで十分な場合があります。また、特定の MX バージョンと Android の最小バージョンが必要になる場合もあります。
  • デバイス モデルは、MX 機能のどのサブセットがデバイスに存在するかを示します。

概要

Wireless Analyzer (以前の WorryFree Wi-Fi Manager) は、Zebra デバイス用の Zebra Wi-Fi 分析ツールと診断ツールを制御します。

重要:

  • Zebra Professional シリーズのデバイスの場合、Wireless Analyzer ツールを使用するには、Mobility DNA Enterprise ライセンスを購入する必要があります。
    MDNA ライセンスについてをご覧ください。
  • StageNow ユーザーの場合、Wireless Analyzer は次のデバイスでのみサポートされます。
    • StageNow 3.2 以降
    • MX 8.3 以降*

* 一部の機能には、新しい MX バージョンが必要です。MX 機能マトリックスを参照してください。

主な機能

  • 有効化/無効化:
    • Wi-Fi の動作の分析
    • Wi-Fi のサービス エリアの分析
    • パケット キャプチャ
    • パケット キャプチャへのユーザー アクセス用の一時パスワード
    • ping データ
    • オンデバイス クライアントの UI とパスワードの変更へのユーザー アクセス
    • コール許可制御
    • カバレッジ ビューのストリーミング
    • 自動到達可能性テスト
  • 8 文字以上のパスワード長 (MX 10.3 以降)
  • クライアント UI、設定、およびデータにアクセスするための古いパスワードと新しいパスワードを入力する
  • 以下のものを選択または構成する
    • 分析タイプ: 接続、音声、またはローミング
    • パッシブ (バックグラウンド) またはアクティブ (フォアグラウンド) 分析
    • 分析対象のネットワークの SSID
    • リモート サーバーと代替のデフォルト ゲートウェイの URL
    • エラーのみ、警告とエラー、またはすべてのイベントのロギング
    • キャプチャ ファイルとデバイスのストレージの最大サイズ
    • パケット キャプチャのファイル名
    • カバレッジのセッション名
    • カバレッジの SSID
    • スキャン間隔
    • 一時パスワードの期間
    • 分析セッションにかかる時間
    • デバイスに分析データを保管する時間
  • すべてのパケットまたは管理パケットのみをキャプチャする対象:
    • 接続の要求
    • 接続の応答
    • 再接続の要求
    • 再接続の応答
    • プローブの要求
    • プローブの応答
    • ビーコン
    • ATIM
    • 接続の解除
    • 認証
    • 認証の解除
    • アクション
  • プロトコルで音声パケットをフィルタリングする
