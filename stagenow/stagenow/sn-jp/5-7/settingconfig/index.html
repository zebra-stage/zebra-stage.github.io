設定

StageNow 5.7

はじめに

ステージング管理者は、プロファイルで使用および再利用可能な設定を作成して管理できます。

プロファイル作成時には、値を入力する構成画面が設定テンプレートとなります。各設定テンプレートで値を選択して保存すると、デバイスに特定の機能を構成する設定が作成されます。設定は保存しておいて、後から他のプロファイルを作成するときに使用できます。詳細については、「再使用するために保存」を参照してください。

ウィザード以外の場所で設定を作成することもできます。詳しくは「設定の作成」を参照してください。

設定タイプ

設定の作成に使用できるすべての設定タイプについては、「設定タイプ」セクションを参照してください。

すべての設定

左側のオプションから [すべての設定] を選択して、作成した設定のリストを開きます。このリストには、[再使用するために設定を保存] と指定したすべての設定が含まれています。プロファイル作成中に再使用目的で保存されていない設定は、このウィンドウには表示されません。

このウィンドウには、次のフィールドがあります。

  • タイプ - 設定のタイプ。

  • 名前 - 管理者が指定した設定の名前。設定を表示したり変更したりするには、名前をクリックします。1 つまたは複数の公開プロファイルで使用される設定を変更するには、ポップアップ ウィンドウで [公開取り消し] を選択する必要があります。そのような設定を変更すると、当該の設定を使用するすべてのプロファイルの動作に影響しますので注意してください。

  • プロファイルで使用中 - 設定を使用しているプロファイルの数。

  • 最終更新日時 - 管理者が最後に設定を更新した日時。

  • MX バージョン - 設定の作成時に使用される MX のバージョン。

  • アクション - 設定に対して実行する次のアクション。

    • コピー - 設定をコピーして、別の名前で同様の設定を作成します。

    • 削除 - 設定を削除します。1 つ以上のプロファイルで使用されている設定は削除できませんので注意してください。

設定の作成

プロファイルで使用する設定を作成するには、次の手順を実行します。

  1. 左側のオプションから [すべての設定] を選択して、[設定] ウィンドウを開きます。

  2. [設定の作成] を選択して、[新しい設定の作成] ウィンドウを開きます。

  3. ドロップダウン メニューからデバイスの MX バージョンを選択します。この選択は、将来の設定の構成時に保持されます。

  4. [設定タイプ] ドロップダウン リストから設定のタイプを選択します。

  5. 設定を完了するために必要な情報を選択および入力します。各設定テンプレートで使用可能なオプションについては、「設定タイプ」を参照してください。

  6. [保存] を選択します。これにより、設定が [設定] リストに追加されます。

再使用するために保存

[プロファイル] ウィザード内の [設定] 画面を完了したら、[再利用するために設定を保存] チェック ボックスをオンにして、同じウィザードや他のウィザードでこの後のプロファイル作成にこの設定を使用できるようにします。設定の名前を入力します。これは、多数のプロファイルが同じネットワークにアクセスする必要がある場合 (Wi-Fi 設定) など、同じ設定を使用するプロファイルを複数作成する場合に便利です。これにより、不必要な再入力を回避できるだけでなく、ミスを排除することができます。また設定の定義が変更された場合、管理者は 1 箇所で設定を変更すればよいので、多数の設定を変更する必要がなくなります。

それ以降のプロファイルの作成中に、[保存済み設定の再利用] タブを選択して、その設定タイプの既存 (保存済み) の設定を選択します。

設定のコピー

設定をコピーして、類似した設定を新しく作成できます。

設定のコピーを作成すると、コピーは元の設定とは別のものになるため、元の設定とコピーうちの一方を編集しても、もう一方の設定には影響しません。元の設定が 1 つ以上のプロファイルで参照されている場合、コピーは作成後に明示的に参照されない限り、参照されることはありません。

設定をコピーするには、次の手順を実行します。

  1. [すべての設定] を選択します。

  2. コピーする設定の横にあるコピー アイコンを選択します。

  3. 設定の新しい名前を入力して [保存] を選択します。

  4. 新しい設定の編集方法については、次のセクションを参照してください。

設定の編集

設定を編集するには、次の手順を実行します。

  1. [設定] リストから設定を選択します。設定が 1 つ以上の公開済みプロファイルで使用されている場合、当該のプロファイルが画面の下部に表示されます。任意の設定フィールドを選択すると、ポップアップが表示され、設定を編集するためには関連付けられたプロファイルを非公開にするように要求されます。

  2. 設定を編集するには、[OK] を選択し、[すべて公開取り消し] を選択します。

  3. [設定] ウィンドウで必要な変更を行い、[保存] を選択します。

設定がテスト済みプロファイルの一部である場合、関連付けられたプロファイルの状態が [テスト済み] から [完了] に変わります。

設定の削除

設定を削除するには、以下の手順を実行します。

  1. 設定の横にあるゴミ箱アイコンを選択します。1 つ以上のプロファイルで使用されている設定は削除できませんので注意してください。

  2. [OK] を選択して削除を続行します。

  3. 設定が正常に削除されたことを示すウィンドウが表示されます。[OK] を選択します。

設定のエクスポート

すべての設定をエクスポートするには、以下の手順を実行します。

  1. [すべての設定] を選択します。

  2. [すべてエクスポート] ボタンを選択します。

  3. すべての設定を含む zip ファイルのデフォルト名は AllSettings.zip です。設定を保存する場所を選択し、[保存] を選択します。設定が正常にエクスポートされたことを示すウィンドウが表示されます。

  4. [OK] を選択します。

設定のインポート

すべての設定をインポートするには、次の手順を実行します。

  1. [すべての設定] を選択します。

  2. [すべてインポート] ボタンを選択します。

  3. 参照ボタンを使用して zip ファイルを選択し、[開く] を選択します。

  4. [インポート] を選択します。設定が正常にインポートされたことを示すウィンドウが表示され、設定が設定リストに表示されます。
