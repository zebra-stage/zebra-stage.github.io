StageNow 3.0

StageNow 3.0 用户指南，修订版 A

StageNow MN-002910

2018 年 1 月

简介

本指南适用于 StageNow 3.0.x 及更高版本。

本指南为部署管理员提供有关使用 StageNow 3.0 为部署设备创建配置文件的说明。

支持的 Web 浏览器

  • Firefox 29 或更高版本

  • Google Chrome 35 或更高版本

  • Internet Explorer 9 或更高版本

本指南的各个章节

  • 安装 StageNow 提供了有关安装、卸载和升级 StageNow 部署解决方案的说明，包括系统要求。

  • 入门介绍 StageNow 解决方案，包括有关用户、使用案例和支持的设备的信息。

  • 配置文件提供有关 StageNow 向导的信息，此类向导允许部署管理员定义企业设备的软件配置和安装场景。

  • 配置文件向导介绍如何使用配置文件创建向导。

  • 设置为部署管理员提供有关配置和管理用于创建配置文件的设置的信息。

  • 设置类型列出创建设置时可用的参数和值。

  • 设备部署为部署操作员信息提供有关选择配置文件以配置目标设备和将配置文件材料部署到设备的信息。

  • 故障排除介绍 StageNow 工具中可能出现的错误，并提供可能的解决方案。

v2.10 新增功能

重要新闻

  • StageNow 3.0 不再支持 Jelly Bean 或 KitKat 设备的 OSupdatepackage。
  • 运行 Android Lollipop 及更高版本的设备的 StageNow 更新文件使用特定设备的最新 BSP 或安全增量补丁分发。
  • 擦除设备指南现在包括用于在企业重置之后绕过 Android 设置向导的说明。

新功能

  • 外部部署服务器支持

    • 部署和部署内容现在可以托管在 StageNow 工作站之外的服务器或系统上。
    • 外部部署服务器可以使用 FTP、FTPS、HTTP 或 HTTPS 协议

  • Zebra VC80 设备现在可以使用串行扫描器 (RS232) 部署

  • 部署 Zebra TC20/TC25 设备时，管理员现在可以防止禁用 Analytics Manager

  • MX 7.2 支持新增了以下设置类型和功能：

  • Audio Manager 新 CSP：

    • 控制设备上的音频是仅通过连接的手持话筒还是通过手持话筒和内置设备扬声器（仅 Zebra VC80x 上支持）播放。

  • Display Manager 新功能：

    • VC80 设备上添加了强制显示屏“保持唤醒”并持续开启的功能。

  • Power Manager 新功能：

    • 打开/关闭设备上的“假睡模式”节能功能。如果启用，只有特别指定的应用程序可以防止设备进入低功耗状态，以延长电池寿命。

  • Remote Scanner Manager 新功能：

    • 支持 RS-507 和 DS-3608 扫描器

  • Settings Manager 新功能：

    • 在运行 Android Nougat 的设备上启用/禁用应用程序通知控制

StageNow v2.10 现在支持以下操作系统和扩展：
