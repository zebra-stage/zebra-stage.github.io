关于 SmartDocs


“SmartDocs”栏可以自定义此页面，仅显示在特定 Zebra 设备上存在的功能。

  1. 使用下拉列表选择设备上的 OSX、MX 和/或 Android 版本，以及或者设备型号，它突出显示了设备中支持的功能。根据所做的选择，不受支持的功能会变灰。
  2. SmartDocs Nav 在查看单个 CSP 的功能时发挥作用。要使用它，请单击一个 CSP（例如“App Manager”），然后在 SmartDocs 栏中进行至少一项选择。不支持的功能将变为灰色。单击前进和后退 SmartNav 按钮可滚动到该 CSP 下一个或上一个支持的功能（SmartNav 在功能矩阵上不受支持）。
  3. 要清除所有选择，请按 SmartDocs 按钮。

在 Android“设置”面板或通过 ADB、EMDK 或 MX CSP 查询设备，可以找到设备的 OSX、MX 和 Android 版本信息。更多信息。

  • OSX 层提供了对基本操作系统的扩展，以实施 Android 未提供的功能。根 OSX 版本号始终与它扩展的 Android 版本的根号码匹配。更多信息。
  • MX 管理系统为设备上的特权和无特权 Android API 提供了一个统一的接口。某些功能需要对 MX 和 OSX 版本进行具体的配对。更多信息。
  • Android 版本有时足以确定功能集。其他情况则需要一个特定的 MX 版本加上 Android 的最低版本。
  • 设备型号可指明在设备上提供 MX 功能的哪个子集。

概述

在支持灵敏度设置的设备上，可使用 TouchMgr 针对不同的使用场景调节触摸屏灵敏度。在戴着手套工作或存在屏幕保护膜时，触摸面板设置可用来弥补使用手指或触笔的输入。

  • TouchMgr 仅在以下设备上受支持：
    • ET50
    • ET55
    • TC51
    • TC55
    • TC56
    • TC70
    • TC70x
    • TC75
    • TC75x

注意：触摸屏灵敏度设置因设备而异。

主要功能

  • 针对以下情况配置触摸面板灵敏度：
    • 裸露的手指
    • 触笔或裸裸露的手指
    • 戴有手套的手指或裸露的手指
    • 触笔或戴有手套的手指或裸露的手指
    • 在 TC51、TC56、TC70x 或 TC75x 上存在屏幕保护膜

设备类型

用于指定要应用触摸屏灵敏度设置的 Zebra 设备集。

参数名称：DeviceType

选项 名称 说明需要
0 任意 此值（或 XML 中缺少此参数）将对任意和所有设备应用触摸模式设置。

MX：6.2+
1 8960 对 TC55、TC70 或 TC75 设备应用设置。

MX：6.2+
3 8956 对 TC51、TC56、TC70x 或 TC75x 设备应用设置。

MX：6.2+
4 ET5x 对 ET50 和 ET55 设备应用设置。

MX：6.3+

触摸操作(任何)

用于根据预期使用场景所需的触摸屏灵敏度对任何受支持的设备应用触摸模式设置。当目标设备未被设备特定的参数覆盖，又已知支持触摸屏灵敏度设置时，可使用此参数代替设备特定的参数。在不支持触摸屏灵敏度设置的设备上使用此参数将在结果 XML 中返回错误。

在以下条件下显示：设备类型设置为“任何”

参数名称：TouchActionAny

选项 名称 说明需要
0 不更改 此值（或 XML 中缺少此参数）导致不会更改触摸模式；任何以前选择的设置都将保留。

MX：6.2+
1 触笔和手指 针对使用触笔或裸露手指的输入调节触摸屏灵敏度。

MX：6.2+
2 手套和手指 针对使用裸露的手指或戴手套的手指的输入调节触摸屏灵敏度。

MX：6.2+
3 仅手指 仅使用裸露手指调节输入的触摸屏灵敏度。

MX：6.2+
4 触笔、手套和手指 针对使用触笔、裸露的手指或戴手套的手指的输入调节触摸屏灵敏度。

MX：6.2+

TC 系列的触摸操作

用于根据预期使用场景所需的触摸屏灵敏度指定特定的一组设备的触摸模式。此参数设计用于 Zebra TC51、TC56、TC70x 和 TC75x（基于 8956）设备以及 TC55、TC70 和 TC75（基于 8960）设备。

在以下条件下显示：设备类型设置为“8956”或“8960”

参数名称：TouchAction

选项 名称 说明需要
0 不更改 此值（或 XML 中缺少此参数）导致不会更改触摸模式；任何以前选择的设置都将保留。

MX：4.2+
1 触笔和手指 针对使用触笔或裸露手指的输入调节触摸屏灵敏度。

MX：4.2+
2 手套和手指 针对使用裸露的手指或戴手套的手指的输入调节触摸屏灵敏度。

MX：4.2+

ET50、ET55 的触摸操作

用于根据预期使用场景所需的触摸屏灵敏度指定特定的一组设备的触摸模式。此参数设计用于 Zebra ET50 和 ET55 设备。

在以下条件下显示：设备类型设置为“ET5x”

选项 名称 说明需要

MX：6.3+
7 仅手指 使用裸露手指调节输入的触摸屏灵敏度。

屏幕保护膜（复选框）

控制是否调节屏幕灵敏度以弥补存在屏幕保护膜的影响。仅与 TC51、TC56、TC70x 和 TC75x 设备兼容。

参数名称：保护膜

选项 名称 说明需要
否 （未选中） 屏幕保护膜不存在；不会调节灵敏度进行补偿。

OSX：6.2+

MX：6.2+
是 （选中） 存在屏幕保护膜；将调节灵敏度（在支持该功能的设备上）以进行补偿。

OSX：6.2+

MX：6.2+
