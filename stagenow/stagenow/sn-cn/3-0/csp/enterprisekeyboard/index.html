关于 SmartDocs


“SmartDocs”栏可以自定义此页面，仅显示在特定 Zebra 设备上存在的功能。

  1. 使用下拉列表选择设备上的 OSX、MX 和/或 Android 版本，以及或者设备型号，它突出显示了设备中支持的功能。根据所做的选择，不受支持的功能会变灰。
  2. SmartDocs Nav 在查看单个 CSP 的功能时发挥作用。要使用它，请单击一个 CSP（例如“App Manager”），然后在 SmartDocs 栏中进行至少一项选择。不支持的功能将变为灰色。单击前进和后退 SmartNav 按钮可滚动到该 CSP 下一个或上一个支持的功能（SmartNav 在功能矩阵上不受支持）。
  3. 要清除所有选择，请按 SmartDocs 按钮。

在 Android“设置”面板或通过 ADB、EMDK 或 MX CSP 查询设备，可以找到设备的 OSX、MX 和 Android 版本信息。更多信息。

  • OSX 层提供了对基本操作系统的扩展，以实施 Android 未提供的功能。根 OSX 版本号始终与它扩展的 Android 版本的根号码匹配。更多信息。
  • MX 管理系统为设备上的特权和无特权 Android API 提供了一个统一的接口。某些功能需要对 MX 和 OSX 版本进行具体的配对。更多信息。
  • Android 版本有时足以确定功能集。其他情况则需要一个特定的 MX 版本加上 Android 的最低版本。
  • 设备型号可指明在设备上提供 MX 功能的哪个子集。

概述

Enterprise Keyboard Manager (EnterpriseKeyboard) 控制企业键盘 (Enterprise Keyboard) 的设置，作为 Zebra 的软输入设备，企业键盘为在运行 Android 的 Zebra 设备中手动输入数据提供了高效率的方法。Enterprise Keyboard 在 Android 炒股键盘的基础上构建，它增加了可编程按键，能够快速在语言和按键布局之间切换，并且可将数据直接扫描到应用程序。

企业键盘还利用 Android 的个人词典获得拼写建议和更正，允许客户和合作伙伴在设备中填充行业专用术语，帮助提高键盘输入的速度和准确度。个人词典可以手动填充，也可以从文件中读取，并且可以通过 Zebra 的移动性扩展 (MX) 和 StageNow 工具大规模部署。了解有关个人词典的更多信息。

另请参阅企业键盘用户指南。

主要功能

启用/禁用：

  • 自动大写
  • 自动更正
  • 阻止攻击性言语
  • 按键时振动、声音和按键弹出窗口
  • 建议下一个字词、拼写更正、个性化术语和联系人
  • 字母数字、数字、扫描和符号选项卡的显示
  • 语音输入键（麦克风图标）的显示
  • 使用“轻击”输入换档字符/长按字符

独特的功能：

  • 直接扫描条码
  • 轻扫、使用选项卡和/或按键切换键盘布局
  • 设置首选选项卡在调用键盘时显示
  • 在计算器、手机和其他布局之间轻松切换
  • 轻触一次即可切换语言
  • 重映射按键以创建自定义布局
  • 室内外轻松读取的颜色方案
  • 自动隐藏建议栏以获得最佳屏幕使用

包括的语言：

  • 英语（美国）

关于 Tab 键导航

企业键盘 Tab 键设计为将光标或 UI 焦点从一个文本字段或 UI 组件移到下一个文本字段或 UI 组件。Zebra 建议将应用程序构建为使用 Tab 键以此方式导航。在安装有 MX 6.2 之前版本和/或企业键盘 1.3 的设备上，Tab 键可能不会按预期移动光标。有关本机应用程序导航的帮助，请参阅 Android 开发人员文档的 Android Tab 键导航部分。

导航模式

控制在键盘布局之间导航的方式。“选项卡”模式允许通过直接滑动或轻触所需的选项卡来切换布局。除了在显示下一个字词建议（如果启用）时，选项卡栏始终可见。“按键”模式允许使用按键来切换布局。此类按键通常在布局的左下角，受到下列限制的约束。

注意：

  • 如果选项卡栏或建议栏被启用，则同时显示两个。
  • 如果“下一个字词建议”栏和语音输入键都被禁用，则不会显示选项卡栏。
  • 在 GMS 设备上，如果启用了语音并且禁用了下一个字词建议功能，则会在建议字词时在选项卡栏上方显示空白建议栏。
  • 如果启用了 TYPE_TEXT_FLAG_NO_SUGGESTIONS 输入类型，则选项卡栏和建议栏都不会显示。关于 Android 输入类型。
  • 启用“符号”选项卡将禁止重映射数字 P3 键。
  • 启用“字母”选项卡将禁止重映射数字 P4 键。
  • 启用“数字”选项卡将禁止重映射字母数字 P1 键。

参数名称：NavigationMode

选项 名称 说明需要
1 选项卡 启用基于选项卡的导航，以便在键盘布局之间切换。

MX：7.1+
2 按键 启用基于按键的导航，以便在键盘布局之间切换。

MX：7.1+
