设置

StageNow 3.0

简介

部署管理员可以创建和管理可在配置文件中使用和重复使用的设置。

创建配置文件期间，您在其中输入值的配置屏幕为设置模板。在每个设置模板中选择和保存的值将在设备上创建配置特定特征或功能的设置。在创建其他配置文件时，可保存该设置供以后使用。有关更多信息，请参阅保存以供重复使用。

您还可以在向导之外创建设置。请参阅创建设置。

设置类型

有关创建设置可用的所有设置类型，请参阅设置类型部分。

所有设置

从左侧选项中选择“所有设置”以打开已创建设置列表。请注意，此列表包括所有“保存以供重复使用”设置；创建配置文件期间未保存重复使用的设置不会显示在此窗口中。

此窗口包括以下字段：

  • 类型 - 设置的类型。

  • 名称 - 管理员提供的设置名称。单击设置以查看或修改它。要修改一个或多个已发布配置文件中使用的设置，您必须在弹出窗口中选择“取消发布”。请注意，修改此类设置会影响使用此设置的所有配置文件的行为。

  • 在配置文件中使用 - 使用该设置的配置文件的数量。

  • 上次更新时间 - 上次管理员更新设置的时间。

  • MX 版本 - 创建设置时所使用的 MX 版本。

  • 操作 - 提供以下操作以执行设置：

    • 复制 - 复制设置以创建带不同名称的类似设置。

    • 删除 - 删除设置。请注意，您无法删除在一个或多个配置文件中使用的设置。

创建设置

要创建在配置文件中使用的设置：

  1. 从左侧选项选择“所有设置”以打开“设置”窗口。

  2. 选择“创建设置”以打开“新建设置”窗口。

  3. 从下拉菜单中选择设备 MX 版本。配置未来设置时，该工具将保留此选择。

  4. 从“设置类型”下拉列表中选择设置类型。

  5. 选择并输入所需的信息以完成设置。有关在每一设置模板中可用的选项，请参阅设置类型。

  6. 选择“保存”。这会将设置添加到“设置”列表。

保存以供重复使用

在配置文件向导中完成设置屏幕后，选择“保存设置以供重复使用”复选框，以使此设置在后续相同或其他向导中创建配置文件时可用。输入设置的名称。此功能在创建使用相同设置的多个配置文件时非常有用，例如多个配置文件需要访问同一网络（Wi-Fi 设置）。此操作可避免不必要的重复条目，还可消除错误，以及如果设置的定义发生更改，管理员可在一个位置进行更改，而非多个位置。

在后续创建配置文件期间，选择“重复使用已保存的设置”选项卡以为该设置类型选择现有（已保存）的设置。

复制设置

您可以复制设置以创建新的类似设置。

请注意，当您创建设置副本时，该副本与原始设置无关，并且编辑原始设置或副本均不会影响另一个。如果原始设置由一个或多个配置文件引用，则副本不会，除非在其创建之后明确引用。

要复制设置：

  1. 选择“所有设置”。

  2. 选择设置旁边的“复制”图标进行复制。

  3. 输入设置的新名称，然后选择“保存”。

  4. 有关编辑新设置的说明，请参阅下一部分。

编辑设置

要编辑设置：

  1. 从“设置”列表中选择设置。如果在一个或多个已发布的配置文件中使用该设置，则在屏幕的底部列示这些配置文件。选择任何设置字段将调用请求您取消发布关联配置文件的弹出窗口以便编辑设置。

  2. 选择“确定”，然后单击“全部取消发布”以编辑设置。

  3. 在“设置”窗口中进行所需的更改，然后选择“保存”。

如果设置为已测试配置文件的一部分，则关联的配置文件将其状态从“已测试”更改为“完成”。

删除设置

要删除设置：

  1. 选择设置旁边的垃圾箱图标。请注意，您无法删除在一个或多个配置文件中使用的设置。

  2. 选择“确定”以继续删除操作。

  3. 窗口指示已成功删除该设置。选择“确定”。

导出设置

要导出所有设置：

  1. 选择“所有设置”。

  2. 选择“全部导出”按钮。

  3. 包含所有设置的 zip 文件默认名称为 AllSettings.zip。为设置选择位置，然后选择“保存”。窗口指示已成功导出设置。

  4. 选择“确定”。

导入设置

要导入所有设置：

  1. 选择“所有设置”。

  2. 选择“全部导入”按钮。

  3. 选择使用浏览按钮的 zip 文件，然后选择“打开”。

  4. 选择“导入”。窗口指示已成功导入设置，并且设置将显示在设置列表中。
