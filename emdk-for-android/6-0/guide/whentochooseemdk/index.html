DataWedge is recommended for the customers: ØWho wants to scan or capture the data without any writing code ØWho wants to scan or capture the data with minimal writing of code and minimal dynamic control over the data capture. DataWedge is not recommended for the customers: ØWho wants the full control of the data capture such as scanner selection, configuration, reading data and receiving decode data and status. These customers can use EMDK Barcode APIs to achieve this.
