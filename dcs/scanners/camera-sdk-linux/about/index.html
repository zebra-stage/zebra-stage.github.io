About Bioptic Color Camera SDK for Linux

Overview

This document provides installation and programming information that allow MP7 color camera enabled application development for Linux.

Supported Devices

For the compatible devices list, please visit following page.

https://www.zebra.com/us/en/support-downloads/software/developer-tools/bioptic-color-camera-sdk-for-linux.html

Supported Linux Distributions

Bioptic Color Camera SDK for Linux supports the following Linux Distributions.

  • Ubuntu 18.04 (64bit)
  • CentOS 7 (64bit)
