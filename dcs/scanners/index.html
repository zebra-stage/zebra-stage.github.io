Barcode Scanner Software Documentation



Barcode Scanner SDK for Windows

Zebra Scanner Software Developer Kit (SDK) for Windows - an architectural framework providing a single programming interface across multiple programming languages and across multiple system environments for all scanners communication variants.

Overview Setup Sample Apps Source Codes Appendix API

OPOS Drivers for Windows

Zebra OPOS Driver which enables bar code data communication between any scanner and an OPOS compliant POS application via either a USB or RS-232 connection.

Overview Setup Properties, Method and Events Sample Apps (Scanner) Sample Apps (Scale) DirectIO Appendix

JPOS Drivers for Windows

An application accesses the JavaPOS device through the JavaPOS Device Interface, which is specified by Java interfaces.

Overview Setup Properties, Methods, Events Sample Apps DirectIO Appendix

Scanner WMI Driver for Windows

Using a free provider plug-in from Zebra, WMI enables local and network remote management of a scanner.

Overview Setup Common Information Model (CIM) Sample Apps XML Schemas

Scanner Management Service (SMS) for Windows

The SMS enables your enterprise to remotely manage its Zebra scanners. In addition to accessing asset tracking information and statistics, the SMS can set parameters and update firmware.

Overview Setup Create SMS Package Deploy SMS Package

IoT Connector for Windows

IoT Connector is a simple logging tool that will log events related to scanners to a text file. It can be configured to write the logs to several places like console and a text file at the same time or only to one of them.

Overview

Active Focus Manager (AFM) for Windows

Active Focus Manager (AFM) keeps track of which window has active focus on the computer and configures the scanner to send data to meet this window's input needs.

Overview Setup

Scan-To-Connect Utility for Windows

The Scan-To-Connect (STC) Utility enables up to 7 Zebra Bluetooth scanners to pair to a PC / tablet, each in one simple step, without having to modify your app.

Overview Setup

Bioptic Color Camera SDK for Windows

The Bioptic Color Camera SDK for Windows provides the ability to manage camera connections, perform various operations, configure and retrieve information related to connected cameras.

Overview Getting Started SDK Bounding Box Detection Sample Apps Source Codes COM API

TWAIN Driver for Windows

The Zebra TWAIN driver for Windows which enables an imaging scanner to capture and transfer images to a TWAIN-compliant application

Overview Setup SDK Sample Apps

Barcode Scanner SDK for Linux

The Software Development Kit (SDK) for Linux enables a developer to create an application that can control a Zebra scanner. The SDK also includes a JPOS driver for Linux.

Overview Setup Test Utilities Sample Apps Appendix API

JPOS Drivers for Linux

An application accesses the JavaPOS device through the JavaPOS Device Interface, which is specified by Java interfaces.

Overview Setup Properties, Methods, Events Sample Apps DirectIO JPOS Remote Management Tool (Linux) Symbology Support

Scanner Management Solution (SMS) for Linux

The Scanner Management Service (SMS) enables your enterprise to remotely manage its Zebra scanners. In addition to accessing asset tracking information and statistics, the SMS can set parameters and update firmware.

Overview Setup

IoT Connector for Linux

IoT Connector is a simple logging tool that will log events related to scanners to a text file. It can be configured to write the logs to several places like console and a text file at the same time or only to one of them.

Overview

Bioptic Color Camera SDK for Linux

The Bioptic Color Camera SDK for Linux provides the ability to manage camera connections, perform various operations, configure and retrieve information related to connected cameras.

Overview Getting Started SDK Bounding Box Detection Monochrome Library API

Bioptic Color Camera SDK for Python (Linux)

The Bioptic Color Camera SDK for Python (Linux) provides installation and programming information that allow Bioptic Color and Monochrome camera enabled application development for Linux operating system using Python.

Overview Getting Started SDK API

Barcode Scanner SDK for Android

The Zebra Scanner SDK for Android enables cordless scanners to be connected and controlled by a tablet / smart phone without using a cradle, after pairing over Bluetooth.

Overview Getting Started Programmer's Guide Source Codes API

Barcode Scanner SDK for iOS

The Zebra Scanner Control app enables Zebra cordless scanners to be connected and controlled by a iPhone/iPad after Bluetooth pairing.

Overview Setup SDK

Xamarin Wrapper for Scanner SDK (Android and iOS)

Zebra Scanner SDK for Xamarin enables a developer to build native applications to connect and control Zebra Barcode scanners over a Bluetooth connection (no cradle involved) on iOS and Android devices from a single, shared C# codebase.

Overview Getting Started API

Barcode Scanner SDK for Chrome OS

The Scanner Software Development Kit (SDK) for Chrome allows developers to create Chrome apps. These apps can be run on any Chromebook and Chromium device. In addition, these apps can be run within a Windows Chrome browser version 69 or newer.

Overview Sample Apps API

.NET MAUI Wrapper for Scanner SDK (Android & iOS)

Zebra .NEW MAUI Wrapper for Scanner SDK enables a developer to build native applications to connect and control Zebra Barcode scanners on iOS and Android devices from a single, shared C# and XAML codebase.

Overview Getting Started Appendix
