About Xamarin Wrapper for Scanner SDK (Android and iOS)

Overview

Zebra Scanner SDK for Xamarin enables a developer to build native applications to connect and control Zebra Barcode scanners over a Bluetooth connection (no cradle involved) on iOS and Android devices from a single, shared C# codebase.

Functionality,

  • Discover, connect and disconnect readers.
  • Perform basic inventory (start, stop inventory with relevant notifications).
  • Barcode reading.

Version History

Version 1.0.15 – 04/2022

  1. Added support for Android API level 30.
  2. Added support for Barcode Scanner SDK for iOS v1.4.32 and Barcode Scanner SDK for Android v2.6.8.0.
  3. Bug fix – Resolved issue of firmware update via Xamarin wrapper in async.
  4. RFD40 (Standard, Premium, Premium+) support added (Bluetooth only).

Version 1.0.13 – 10/2020

  1. Added support for Barcode Scanner SDK for iOS v1.4.23 and Barcode Scanner SDK for Android v2.5.16.0.
  2. Added support for Bluetooth Classic STC paring on Android SDK.

Version 1.0.10 – 10/2020

  1. Bug fix – scanners get disconnected when the “Discover Bluetooth Scanners” reenabled.

Version 1.0.9 – 08/2020

  1. Added CS6080 and RS5100 support to the Android and iOS SDKs.

Version 1.0.4 – 01/2020

  1. Bioptic scale support added to Barcode Scanner SDK for Xamarin but only for the Android OS. The iOS operating system does not support the Bioptic scale.

Version 1.0.1 – 10/2019

  1. Initial release of Xamarin wrapper for Barcode Scanner SDK supporting both iOS and Android operating systems.
    • iOS v13 support.
    • Android v10 support.

Version 1.0.0 – 08/2019

  1. Initial release.
