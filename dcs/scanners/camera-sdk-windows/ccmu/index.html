Color Camera Management Utility (CCMU) for Windows

Overview

Color Camera Management Utility (CCMU) is a tool that can utilize Color Camera SDK (software development kit) to execute command line interface-based commands for Camera Management.

Utilizing the color camera management utility requires user to copy the CCMU.exe into the following path where the bin files of the Color Camera SDK are installed.

Default location: C:\Program Files (x86)\Zebra Technologies\Barcode Scanners\Bioptic Camera\SDK\Release\bin\x86

The following commands can be executed using the CCMU tool.

  1. Retrieve Configuration - Retrieve configuration from camera and save to file (*.bcccfg)
  2. Load Configuration - Load the given bcccfg configuration file to the supported camera. (*.bcccfg)
  3. Firmware Update - Update the camera firmware with the given firmware file (*.DAT)
  4. Get Asset Information - Query asset information from the first camera connected.
  5. Reboot - Reboot the first color camera connected to the host. (v1.0.13.2 and above)

Utility Commands

  1. Open the command prompt and navigate to the folder which contains the CCMU.exe.

    Example: "cd C:\Program Files (x86)\Zebra Technologies\Barcode Scanners\Bioptic Camera\SDK\Release\bin\x86"

  2. The following are examples of executing the above-mentioned commands.

    1. Retrieve Configuration

      The following command will retrieve the configuration from the color camera and save it as ret01.bcccfg file on the same folder path as CCMU.exe.

      --time, will log the actions in progress with timestamp.

      
ccmu.exe --time retrieveconfiguration ./ret.bcccfg

      NOTE: This requires a command prompt with administrator privileges to create the retrieved config file in the "C:\Program Files (x86)\Zebra Technologies\Barcode Scanners\Bioptic Camera\SDK\Release\bin\x86" folder path.

    2. Load Configuration

      Load configuration to the Color Camera. The config file's folder path needs to be passed along with .bcccfg extension.

      --time, will log the actions in progress with timestamp.

      
ccmu.exe --time loadconfiguration ret.bcccfg

    3. Firmware Update

      Update the color camera, the DAT file needs to be included for the command. DAT file needs to be in the same folder path as RCMU exe to execute the following command. If the firmware file is stored elsewhere, provide the relative path in the command.

      --time, will log the actions in progress with timestamp.

      
ccmu.exe --time updatefirmware R01.DAT

    4. Get Asset Information

      Obtain Asset information from the color camera. 

      
ccmu.exe assetinfo

    5. Reboot

      Reboot color camera. After a successful reboot, the serial number of the rebooted camera will be informed on the CLI. 

      
ccmu.exe reboot

  3. Use the following command to check the utility help.

    
ccmu.exe -help-all
