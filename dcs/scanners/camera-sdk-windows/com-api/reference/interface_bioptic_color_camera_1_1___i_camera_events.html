Firmware download event progress event callback. In order for this call back to get invoked, it is required to register for the event by calling RegisterForFirmwareDownloadProgressEvents(TRUE).

Parameters
serialSerial number of the camera being updated.
modelModel number of the camera being updated.
total_recordsThe Total number of records in the firmware.
current_recordThe most recent record number transfered to camera.
operation_codeOperation code of the current firmware download operation. 0 - Session Start, 1 - Download start, 2 - Download in progress, 3 - Download end, 4 - Session end, 5 - Error.
