Class: BarcodeScannerAdapter

BarcodeScannerAdapter

Provides means to communicate with a physical barcode scanner device connected to a host. The BarcodeScannerAdapter at present facilitates communicating with the physical barcode scanners over USB.

(async) addAuxiliaryDeviceAddedHandler(callback) → {Promise}

Returns a Promise that registers the given callback with the topology changed events (auxiliary device attachment) of the underlying barcode scanner, upon fulfillment of the Promise. The topology changed events will be enabled if they have not yet been already enabled. The callback will be invoked upon attachment of a new auxiliary device to the underlying barcode scanner.
Parameters:
Name Type Description
callback function The callback with a single parameter, which will be invoked upon attachment of a new auxiliary device. The argument to the callback would be a ContextAddress.
Throws:
An Error. See https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Error
Type
Error
Returns:
A Promise that registers the given callback with the topology changed events (auxiliary device attachment) of the underlying barcode scanner, upon fulfillment of the Promise. Or else throws an Error instance if the callback argument does not resolves to a function.
Type
Promise
Example
await scannerAdapter.addAuxiliaryDeviceAddedHandler((contextAddress)=> console.log(contextAddress.hopNumber););

Name Type Attributes Description

contextAddress ContextAddress <optional>
 An instance of the ContextAddress type which represents an auxiliary device attached to the underlying barcode scanner. Default value of this parameter is 0 which is the root device.
Throws:
An error. See https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Error
Type
Error
Returns:
A Promise that resolves to a ScaleReadWeight instance upon receiving a stable weight from the scanner. Or else, throws an exception in the event receiving a non-stable weight from the scanner, with a ScaleReadWeight instance as the exception argument. Promise
Example
