(async) createBarcodeScannerAdapterFor(deviceInfo) → {Promise}

Instantiates a BarcodeScannerAdapter that represents the barcode scanner given by the deviceInfo provided. If the barcode scanner provided by deviceInfo is no present in the host, a null reference is returned.
Parameters:
Name Type Description
deviceInfo DeviceInfo DeviceInfo instance that of the barcode scanner
Returns:
A Promise that resolves to a reference to the BarcodeScannerAdapter that represents the physical barcode scanner if it is found in the host. Or else resolves to a null reference.
Type
Promise
Example
let deviceManager = DeviceManager.createDeviceManager();
let discoveredDevices = await deviceManager.discoverDevices();
let scannerAdapter = await deviceManager.createBarcodeScannerAdapterFor(discoveredDevices[0]);

(async) discoverDevices(deviceModes) → {Promise}

Discovers data capture devices from Zebra Technologies connected to the host machine
Parameters:
Name Type Description
deviceModes deviceModes A bit mask for interested device types. Use 1 for USB SNAPI devices
Returns:
A Promise that resolves to a array of DeviceInfo instances. Or else, throws an Error instance with the reason for not being able discover devices.
Type
Promise
Example
let deviceManager = DeviceManager.createDeviceManager();
let discoveredDevices = await deviceManager.discoverDevices();
console.log("Device count= " + discoveredDevices.length);

getDiscoveredDevices()

Returns an array of already discovered Zebra devices
Returns:
Array of DeviceInfo
Example
let deviceManager = DeviceManager.createDeviceManager();
let discoveredDevices = await deviceManager.getDiscoveredDevices();
console.log("Device count= " + discoveredDevices.length);
