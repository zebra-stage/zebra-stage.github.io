Zebra Scanner SDK for Chrome

Overview

Zebra Scanner SDK for Chrome provides a comprehensive set of APIs to develop Chrome apps for Zebra barcode scanners.

Supported USB Host types

  1. USB SNAPI (Symbol Native API).

Prerequisites

  • Chrome 69 or higher
  • ECMAScript 2017 (ES8)

Installation

Zebra Scanner SDK for Chrome comes in a zip package which you can extract and use in your development environment.

Usage

  1. Copy all .js files available in SDK_for_ChromeOS_vY.Y.Y.Y folder into your project folder
  2. Include the provided java scripts in the order given below in your application 
     <script src="js/logger.js"></script>
 <script src="js/hiddeviceadapter2.js"></script>
 <script src="js/snapiInputOutReports.js"></script>
 <script src="js/rmdDataTypes.js"></script>
 <script src="js/rmdCommandResponse.js"></script>
 <script src="js/remoteManagementDiagnostics.js"></script>
 <script src="js/barcodeScannerAdapter.js"></script>
 <script src="js/deviceManager.js"></script>
