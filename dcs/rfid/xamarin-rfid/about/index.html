About Xamarin Wrapper for RFID SDK (iOS)

Overview

Zebra RFID SDK for Xamarin enables a developer to build native applications to connect and control Zebra Barcode scanners over Bluetooth on iOS devices from a single, shared C# codebase.

Functionality,

  • Discover, connect and disconnect readers.
  • Perform basic inventory (start, stop inventory with relevant notifications).
  • Barcode reading.

Supported Devices

  • RFD8500 RFID Reader
  • RFD8500i UHF RFID Sled

Supported Operating System

Xamarin Wrapper for RFID SDK support the following iOS operating systems.

iOS 12.X, 13.X and 14
