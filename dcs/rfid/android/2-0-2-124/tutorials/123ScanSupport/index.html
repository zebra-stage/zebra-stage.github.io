123Scan Support

Applicable Devices :RFD40 premium plus and RFD90

Overview

This Tutorial provides a walk-through of the steps to do basic data formatting for the barcode scan data with RFD40 premium plus or RFD90 devices

Details

123Scan tool can be used to create parameter barcode to perform basic data formatting. Below highlighted data formatting options are the once which is supported. Advanced Data formatting is not supported

To apply the parameter barcode that has been generated, user needs to disable passthrough mode as first step and then apply generated parameter barcode followed by enabling parameter barcode again.

The below parameter barcode would disable passthrough mode.

Steps to create parameter barcode

  1. Launch 123Scan software application
  2. Create new configuration file.
  3. My scanner is not connected.
  4. Choose the PL5000 Imager,
  5. Select PL500-COMMON MOdels

  6. Select modify data

    7. Options highlighted by the square box are the once that is supported by RFD40 Premium plus device and by RFD90

  7. Once the required options is selected you can select print barcode and generate the parameter barcode

    8. User can save or print this parameter barcode.

  8. Barcode to enable back passthrough mode
