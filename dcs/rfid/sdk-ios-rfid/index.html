About RFID SDK for iOS

Overview

iOS RFID SDK is an API that an iOS developer can use to develop applications that use Motorola/Zebra Bluetooth RFID readers. The API is capable of managing readers and receiving RFID data. API is provided as an "XCFramework" and there is a Demo application that a developer can refer to on how to use that XCFramework.

Supported Devices

  • RFD40
  • RFD90
  • RFD40 Premium Plus
  • RFD40 Premium
  • RFD8500 RFID Reader
  • RFD8500 ASIC
