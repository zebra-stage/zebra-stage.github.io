Properties, Methods and Events.

Overview

This page provides a table listing the supported features(properties, methods, and events) of the Zebra RFID JPOS driver.

For more information about all the features in the UPOS specification, refer to Chapter 33 "RFID Scanner" in Unified POS Retail Peripheral Architecture.
