JPOS Drivers for Windows.

Overview

UPOS (Unified Point of Service) is a standard specification used in retail and point of sale (POS) systems that define the comman programming interface for various types of POS devices. UPOS is part of the broader JavaPOS and OPOS standards, which are often used in the retail environment to interface with the hardware devices like barcode scanner, receipt printers, cash drawer, and RFID readers.

Please refer this link for more information about Unified POS Retail Peripheral Architecture.

JavaPos (JPOS) is a free, open-source software framwork that provides a standard interface for point of sale (POS) devices (including RFID reader). JPOS is based on the International Organization for Standardization transaction card originated messages standard (ISO-8583).

Zebra RFID JPOS driver package includes the Libs with Service CLass Library and other JPOS dependent libraries with that user can develop powerfull jpos application for the FXP20 device. And package also includes Jpos.xml for Jpos configuration file contain the logical device name, jpos-rfidscanner-test.jar : Contains sample app to demonstarte, javaPosTest.bat : Batch file to run the sample app.
