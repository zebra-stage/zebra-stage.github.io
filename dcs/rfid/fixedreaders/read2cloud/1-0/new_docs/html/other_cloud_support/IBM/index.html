Connect Fixed Readers to IBM Watson IoT Platform¶

This section serves to guide the user to connect Zebra fixed readers FX9600, FX7500 and ATR7000 with IBM Watson IoT Platform service.

Important

prerequisite:

  • FX9600 and FX7500 readers firmware version should be >= 3.10.X

  • ATR7000 reader firmware version should be >= 3.17.X

Warning

Reader uses 8883 port to connect with IBM Watson IoT Platform. Port should be open for this tutorial to work.
