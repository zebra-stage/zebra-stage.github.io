Configure device using Web Interface¶

To enroll the reader to Zebra Data Services, follow the steps below:

Add ZDS Endpoint¶

  1. Open a web browser to connect to the reader using the host name or IP address.

  1. Click Communication > Zebra IoT Connector > Configuration.

  2. Click on Add Endpoint button to add new endpoint.

  3. Select ZDS for Endpoint Type.

  4. Configure Endpoint Name and Endpoint Description fileds.

  5. The following parameters are required to enroll the reader. Values for these parameters are obtained after creating and registering your Cloud application and need to be keyed into this page.

    1. Claim Code - Secret code for enrolling the device (Enrolling the Reader).

  6. Click on Add.

  7. Once added the endpoint will be seen under Endpoint Configurations.

Interface Configuration¶

Important

Zebra Data Services endpoint currenlty doesn’t support Management Event Interface.

  1. Open a web browser to connect to the reader using the host name or IP address.

  2. Click Communication > Zebra IoT Connector > Configuration.

  3. Select ZDS endpoint for Management Interface, Control Interface, and Tag Data Interface.

  4. Click on Update button to update interface configuration.

Start ZIoTC Service¶

  1. Open a web browser to connect to the reader using the host name or IP address.

  2. Click Communication > Zebra IoT Connector > Connection.

  3. Click on Connect button.

  4. Once connected the current connection status of the interfaces will be displayed under Connection Status section.
