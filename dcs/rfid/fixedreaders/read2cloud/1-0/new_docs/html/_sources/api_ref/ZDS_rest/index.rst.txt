===================================
Zebra Data Services (ZDS) REST APIs
===================================

When deployed with ``Data Services``, ZDS exposes a REST API interface that can be used to manage, configure, and read tag data from RFID readers. For details on the REST API, please visit `Device Management for RFID <https://developer.zebra.com/apis/rfid-reader-management-0>`_.
