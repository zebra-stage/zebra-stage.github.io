====================
Enrolling the Reader
====================

Enrollment is the first step whereby the reader is registered as an IoT device with the Zebra Data Services. After enrollment, the reader is connected to cloud application by an IoT agent service on the reader that enables control, management, and data channels for cloud app to communicate with the reader.

To enroll the reader to Zebra Data Services, Register and generate the Claim Code from the Zebra Data Services platform. Go to `https://developer.zebra.com/docs/enroll-fx-reader <https://developer.zebra.com/docs/enroll-fx-reader>`_ for more information on how to generate the claim token.
