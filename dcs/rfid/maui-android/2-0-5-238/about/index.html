About RFID SDK For MAUI Android

RFID SDK For MAUI 2.0.5.238

Overview

The Unified Zebra RFID SDK for MAUI is .dll library that adds a comprehensive set of APIs to easily create powerful applications for Zebra RFID Handheld readers. RFID SDK for MAUI includes class library, sample apps and source code to enable developers to easily build apps that take full advantage of the power of Zebra devices.

Supported Devices

Mobile Computers

  • TC53e-RFID
  • EM45

HandHeld Readers

  • RFD40XX
  • RFD90XX
  • MC33XXR
  • RFD8500
  • TC22R NEW

Android Versions 8 and Above

  • RFD40XX – Oreo and onwards
  • RFD90XX – Oreo and onwards
  • MC33XXR - Oreo and onwards
  • RFD8500 - Oreo and onwards
