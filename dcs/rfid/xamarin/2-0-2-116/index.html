About RFID SDK For Android

RFID SDK For Android 2.0.2.116

Overview

RFID SDK for Android is Android aar library that adds a comprehensive set of APIs to easily create powerful applications for Zebra RFID Handheld readers. RFI SDK for Android includes class library, sample apps and source code to enable developers to easily build apps that take full advantage of the power of Zebra devices.

API-related Usage

  1. A profile name argument (the first argument of processProfile API) is mandatory.
  2. If the profile name is given in the extraData (third argument of processProfile API) and if it does not match the profile name (first argument of processProfile API), the profile will NOT be modified.

Samples & Programmer's Guide

There are sample projects that are included as part of the RFID SDK For Android. You can import the project into your IDE and run on your device.
