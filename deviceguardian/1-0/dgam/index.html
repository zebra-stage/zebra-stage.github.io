Asset Management

Device Guardian 1.0

Overview

Device Guardian Asset Management (DGAM) enhances device management and access with streamlined operations. A physical display provides instant visibility into the Device Guardian solution, allowing easy monitoring of device availability, tracking which worker is using which device, and ensuring devices are fully charged before placed into service. Optimize operations with local visibility, improve workflow efficiencies, and swiftly recover missing devices.

The intuitive dashboard displays real-time data:

  • Device count at-a-glance
  • Available devices
  • Devices in use
  • Missing devices - Rapidly locate by identifying the current user, prior user, reason for the missing status, and initiate recovery with Device Guardian
  • Worker-device assignments, enhancing accountability

Setup

Device Guardian Asset Management requires DGAM-specific licenses, in addition to Device Guardian licenses, and necessitates the installation and configuration of Device Guardian - see Device Guardian Setup for details. This section offers further guidance on setting up DGAM.

Device Permissions

For both mobile and kiosk devices, DGAM requires granting the following permissions:

  • Camera Permission - Needed to scan the registration QR code, essential for registering the device with a kiosk.
  • Location Permission - Required to determine the device’s geographical position, crucial for location-based services or GPS-related functionalities.
  • Music and Audio Permission - Enables access and control of the device’s audio features, including playing or managing audio within the application, such alarms.
  • Notifications Permission - Allows the application to send notifications to the user, ensuring effective communication of important updates or alerts.
  • Files and Media Permission - Specific to Android 11, required to read from or write to the device’s storage, allowing access or modification of files and media content while adhering to Android 11’s scoped storage policies.

DataWedge Profile Integration

Zebra's DataWedge application facilitates seamless barcode scanning and data entry on Zebra devices. For DGAM, a specific DataWedge profile must be imported.

Steps to Import DataWedge Profile:

  1. Download the DataWedge Profile: Obtain the DataWedge database file.
  2. Copy the File to Your Device: Transfer the DataWedge profile to the device’s internal storage.
  3. Import the Profile into DataWedge: Open the DataWedge app on your device (installed by default).
    • From the top-right menu, select Settings.
    • Select Import Profile. Browse to the file location from step 2. The profile is imported.
            
  4. Verify Import: Return to the main screen of DataWedge to ensure the profile named "DGAM” is listed.

Device Registration

Register devices to the designated local kiosk to ensure dashboard data is populated. This is accomplished by scanning a barcode displayed on the kiosk with the device.

Steps to register a device to the kiosk:

  1. On the mobile device, open the Device Guardian app.
  2. Tap the top-right menu and select Register to Kiosk.
  3. A screen appears enabling you to scan a barcode.
  4. On the kiosk, tap the menu and select Register Device to Kiosk.
  5. Scan the either the QR Code or Barcode to complete the registration of the device to the specific kiosk.

Dashboard

The kiosk dashboard features three prominent tiles that display device counts:

  • Available - Represents devices currently placed on charge.
  • In Use - Represents devices authenticated with a valid PIN code using Identity Guardian.
  • Missing - Represents devices removed from charging.

Kiosk Dashboard

Tapping any tile provides detailed information on the corresponding devices:

  • Device Name
  • Battery Level
  • Serial Number
  • Alias Name
  • Asset Type (Device)
  • Last User
  • Last Access Point
  • Last Updated

Device In Use

Dashboard actions:

  • Data Sorting - Sort data on the dashboard by tapping visible column names, facilitating easier to organization and analysis.
  • Data Filtering - Apply filter options based on specific columns, such as Device Name and Battery Level, to quickly focus on relevant data points. Tap Add Filter and select the desired columns for filtering.
