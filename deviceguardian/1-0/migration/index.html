Migration

Device Guardian + Access Management 1.0

All references to Device Guardian also apply to Device Guardian Access Management within this guide, unless otherwise specified.

Overview

Device Guardian has been rebranded from Device Tracker, featuring updated user interfaces for both the web portal and Android client. Existing Device Tracker users who wish to migrate to Device Guardian can follow the instructions provided in this migration guide.

Key Changes

The following outlines the enhancements and key points of the upgrade:

  • Device Compatibility: Device Guardian supports devices running Android 11 and higher. Mixed deployments of Android 8, 10 and 11 or higher are not supported. All devices must be upgraded to Android 11 or higher with the compatible LifeGuard Update; see the Zebra support portal.
  • Client Release: The new Device Guardian client has been released with package name com.zebra.mdna.dg and version 1.0.
  • Configuration Management: Device Guardian client settings are now managed through Enterprise Mobility Managment (EMM) configurations. Configuration via StageNow Barcode and MX XML are not supported with Device Guardian 1.0.
  • Device Tracker Compatibility: Device Tracker client versions earlier than v6.3 are not compatible with the new Device Guardian Server.
  • Identity Guardian Compatibility: Device Guardian is compatible with Identity Guardian v2.0 or later.
  • New APIs: APIs are now available to retrieve and share data between applications, including Identity Guardian.
  • Device Tracker Deprecation: No new features for Device Tracker will be released; only bug fixes will be provided until December 31, 2025.

No changes:

  • Existing purchased license counts remain unchanged.

Transition Plan & Actions

Cloud Upgrade:

  • Zebra will upgrade the server from Device Tracker to Device Guardian Cloud, and you will be notified when the upgrade takes place.
  • Device Guardian Cloud v1.0 remains compatible with Device Tracker client v6.3.
  • License count discrepancies (previously showing a 15% overage) are corrected in Device Guardian to reflect actual purchased quantities.

Device Upgrade:

  • Upgrade devices to Android 11 or higher.
  • Update Identity Guardian to v2.0 or later, if in use.
  • Ensure all requirements are met and prepare to deploy installation and configuration via your EMM.
  • Uninstall Device Tracker.
  • Deploy the Device Guardian client and configurations to your devices using your EMM. For instructions, see the Installation Guide.
