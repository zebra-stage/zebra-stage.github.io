安装 StageNow

StageNow 3.0

简介

本部分提供有关安装、卸载和升级 StageNow 部署解决方案的说明，包括系统要求。

StageNow 工作站要求

以下是主机运行 StageNow 工作站工具的最低要求：

  • 操作系统：微软 Windows 7（64 位）

  • 操作内存：最小 2 GB

  • 硬盘驱动器存储：最小 6 GB（如果尚未安装 .Net Framework）

  • 屏幕分辨率：推荐 1366 x 768（16:9 纵横比）

  • .Net Framework 4.5

  • .Net 4.5.1 包含在 StageNow 安装向导中。如果需要，也可从 http://www.microsoft.com 网站获取

  • StageNow 工作站工具安装程序，也可从 http://www.zebra.com/support 网站获取

  • 支持部署材料的 PDF 读取器，建议使用 Adobe Acrobat Reader。

  • Java Runtime (JRE) 将使用音频部署

  • 无线网卡将使用 Wi-Fi 热点功能

安装 StageNow

安装 StageNow 工作站工具还可以自动安装微软 .NET Framework 4.5、部署数据库和部署服务器。

如果您有安装在主机 PC 上的 StageNow 工作站工具的旧版本，请参阅 升级 StageNow 工作站工具。

要安装工作站工具：

  1. 访问 StageNow 下载页面，并下载安装程序包 staging_solution.[版本号].exe。

  2. 双击安装程序包文件。

  3. 如果您尚未具有 .Net Framework 4.5，则安装过程将提示您安装它。

  4. 选择“安装”。框架安装完成后，将出现说明向导正在准备安装 StageNow 的窗口。

  5. 选择“下一步”以继续操作。

  6. 接受许可协议条款，然后选择“下一步”。

  7. 输入“用户名”和“公司名称”，并指定哪些人可以使用“工作站工具”。

  8. 选择“下一步”。

  9. 输入管理员密码，然后选择“下一步”。
    注意：所有 StageNow 配置文件和数据均与此密码相关联。

  10. 选择“下一步”以接受默认文件夹，或选择首先“浏览”以选择另一安装文件夹。

  11. 选择“下一步”以接受当前设置，或选择“返回”以进行更改。向导指示它正在为安装配置软件，然后指示安装即将完成。

  12. 如果需要，选择复选框以在台式机上创建快捷方式，和/或以启动“StageNow 工作站工具”，然后选择“下一步”。

  13. 因为部署材料以 PDF 文件形式呈现，向导将提示您下载 PDF 读取器；或者，如果您已安装读取器，请跳过此操作。选择适当的选项，然后选择“完成”。

StageNow 工作站工具现已准备就绪。

升级 StageNow

如果主机 PC 上存在旧版本的 StageNow 工作站工具，可将其进行升级。

重要说明：升级之前，完成任何在主机 PC 上部分完成的配置文件。如果不这样做，可能会导致不可预测的或不需要的行为。

要升级 StageNow 工作站工具：

  1. 下载安装程序包 staging_solution.[版本号].exe。

  2. 双击安装程序包文件。InstallShield 检测现有版本，并显示以下消息。

  3. 选择“是”以继续升级，或选择“否”以保留现有版本和中止升级。如果您选择“是”，则下一个消息询问是否要保留或放弃现有 StageNow 数据（设置、配置文件等）

  4. 选择所需选项以继续升级。注意：如果您选择保留现有数据，则安装过程中不会请求管理员密码，因为它保留了现有密码。

  5. 有关安装说明的其余部分，请参阅安装 StageNow 工作站工具。

卸载 StageNow

要卸载 StageNow 工作站工具：

  1. 选择“控制面板”>“程序”>“程序和功能”。

  2. 双击 StageNow。将打开安装向导。

  3. 遵循向导中的步骤以完成卸载。

  4. 选择“完成”以完成卸载。

设备上的部署软件

StageNow 设备上的客户端软件支持所有运行 Android KitKat 和更高版本的 Zebra 设备。这些设备通常随附预安装的 StageNow 客户端，并且支持所有 StageNow 设备配置选项。

一些旧的 KitKat 设备不包括 StageNow 客户端，而是随附“快速部署”(RD) 客户端。对于部署，这些设备可以扫描 StageNow 生成的“RD 条码”，这会导致 RD 客户端连接到网络并为 KitKat 设备自动安装 StageNow 客户端 和 MX 5.0。然后，可以使用 StageNow 客户端扫描用于配置的 StageNow 配置文件条码。由于操作系统的限制，此类设备可能仍会提供有限的 StageNow 功能。

了解如何：

  • 生成 RD 条码
  • 确定特定设备上支持的功能
