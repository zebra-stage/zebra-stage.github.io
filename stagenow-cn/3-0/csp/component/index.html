关于 SmartDocs


“SmartDocs”栏可以自定义此页面，仅显示在特定 Zebra 设备上存在的功能。

  1. 使用下拉列表选择设备上的 OSX、MX 和/或 Android 版本，以及或者设备型号，它突出显示了设备中支持的功能。根据所做的选择，不受支持的功能会变灰。
  2. SmartDocs Nav 在查看单个 CSP 的功能时发挥作用。要使用它，请单击一个 CSP（例如“App Manager”），然后在 SmartDocs 栏中进行至少一项选择。不支持的功能将变为灰色。单击前进和后退 SmartNav 按钮可滚动到该 CSP 下一个或上一个支持的功能（SmartNav 在功能矩阵上不受支持）。
  3. 要清除所有选择，请按 SmartDocs 按钮。

在 Android“设置”面板或通过 ADB、EMDK 或 MX CSP 查询设备，可以找到设备的 OSX、MX 和 Android 版本信息。更多信息。

  • OSX 层提供了对基本操作系统的扩展，以实施 Android 未提供的功能。根 OSX 版本号始终与它扩展的 Android 版本的根号码匹配。更多信息。
  • MX 管理系统为设备上的特权和无特权 Android API 提供了一个统一的接口。某些功能需要对 MX 和 OSX 版本进行具体的配对。更多信息。
  • Android 版本有时足以确定功能集。其他情况则需要一个特定的 MX 版本加上 Android 的最低版本。
  • 设备型号可指明在设备上提供 MX 功能的哪个子集。

概述

ComponentMgr 用于配置设备上子系统（如以太网）的状态和使用。

“状态”定义子系统是打开（活动）还是关闭（不活动）。“使用”则启用或禁用子系统，并控制其状态是否可以由设备用户通过系统设置菜单或使用 ComponentMgr 以编程方式更改（打开或关闭）。如果试图使用 ComponentMgr 控制其“使用”当前已被禁用的子系统的状态，将会在结果 XML 文档中返回错误。

注意：并非每个设备都支持每一个子系统。例如，一些设备不支持以太网。如果您尝试使用 ComponentMgr 控制在给定设备上不支持的子系统的状态或使用，则会在结果 XML 文档中返回错误。

主要功能

  • 打开或关闭以太网
  • 启用或禁用以太网使用

以太网使用

此参数允许您控制是否可以更改以太网选项的状态。

如果以太网的使用被禁用，随后又启用，以太网的状态（“开”或“关”）将与它被禁用之前相同。

参数名称：EthernetUsage

选项 名称 说明需要
0 不更改 此值（或 XML 中缺少此参数）将导致不更改是否可以使用以太网。

OSX：4.4+

MX：4.4+
1 启用 此值将导致“以太网选项”的状态被解除锁定，允许通过设备用户或通过 ComponentMgr 更改。

OSX：4.4+

MX：4.4+
2 禁用 此值将导致“以太网选项”的状态被锁定，从而阻止设备用户或 ComponentMgr 更改它。

OSX：4.4+

MX：4.4+

以太网状态

此参数允许将您“以太网选项”的状态更改为“开”或“关”。

参数名称：EthernetState

选项 名称 说明需要
0 不更改 此值（或 XML 中缺少此参数）将导致不更改是打开还是关闭“以太网选项”。

OSX：4.4+

MX：4.4+
1 打开 此值将导致“以太网选项”被打开，从允许在设备上使用以太网连接。

OSX：4.4+

MX：4.4+
2 关闭 此值将导致“以太网选项”被关闭，从而禁止在设备上使用以太网连接。

OSX：4.4+

MX：4.4+

示例

将“屏幕锁定”设置为“显示超时 1 分钟后”

<wap-provisioningdoc><characteristic type="DevAdmin" version="4.3" ><parm name="ScreenLockTimeoutInterval" value="60"/></characteristic></wap-provisioningdoc>

允许安装未知来源的应用程序

<wap-provisioningdoc><characteristic type="DevAdmin" version="4.3" ><parm name="UnknownSourcesStatus" value="1"/></characteristic></wap-provisioningdoc>

<wap-provisioningdoc><characteristic type="DevAdmin" version="4.3" ><characteristic type="AppAsDevAdmin"><parm name="DevAdminAction" value="1"/><characteristic type="DevAdminDetails"><parm name="DevAdminPkg" value="PackageName1"/><parm name="DevAdminClass" value="ClassName1"/></characteristic></characteristic></characteristic><characteristic type="DevAdmin" version="4.3" ><characteristic type="AppAsDevAdmin"><parm name="DevAdminAction" value="1"/><characteristic type="DevAdminDetails"><parm name="DevAdminPkg" value="PackageName2"/><parm name="DevAdminClass" value="ClassName2"/></characteristic></characteristic></characteristic></wap-provisioningdoc>

<wap-provisioningdoc><characteristic type="DevAdmin"><parm-query name="ScreenLockTimeoutInterval"/></characteristic></wap-provisioningdoc>

输出

<wap-provisioningdoc><characteristic type="DevAdmin" version="4.3"><parm name="ScreenLockTimeoutInterval" value="1"/></characteristic></wap-provisioningdoc>

输入

<wap-provisioningdoc><characteristic type="DevAdmin"><parm-query name="UnknownSourcesStatus"/></characteristic></wap-provisioningdoc>

输出

<wap-provisioningdoc><characteristic type="DevAdmin" version="4.3"><parm name="UnknownSourcesStatus" value="1"/></characteristic></wap-provisioningdoc>

示例

启用“以太网使用”并打开以太网

输入

<wap-provisioningdoc><characteristic type="ComponentMgr" version="4.4"><parm name="EthernetUsage" value="1"/><parm name="EthernetState" value="1"/></characteristic></wap-provisioningdoc>

错误输出

以下 XML 是尝试在不支持以太网功能的设备上设置以太网功能时返回的错误。

<wap-provisioningdoc><characteristic-error desc="exception" type="ComponentMgr" version="4.4" ><parm-error name="EthernetUsage" desc="Error in enabling Ethernet UI" value="1" /><parm-error name="EthernetState" desc="Failed to Turn On. Ethernet is disabled by admin" value="1" /></characteristic-error></wap-provisioningdoc>
