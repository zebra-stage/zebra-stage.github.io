关于 SmartDocs


“SmartDocs”栏可以自定义此页面，仅显示在特定 Zebra 设备上存在的功能。

  1. 使用下拉列表选择设备上的 OSX、MX 和/或 Android 版本，以及或者设备型号，它突出显示了设备中支持的功能。根据所做的选择，不受支持的功能会变灰。
  2. SmartDocs Nav 在查看单个 CSP 的功能时发挥作用。要使用它，请单击一个 CSP（例如“App Manager”），然后在 SmartDocs 栏中进行至少一项选择。不支持的功能将变为灰色。单击前进和后退 SmartNav 按钮可滚动到该 CSP 下一个或上一个支持的功能（SmartNav 在功能矩阵上不受支持）。
  3. 要清除所有选择，请按 SmartDocs 按钮。

在 Android“设置”面板或通过 ADB、EMDK 或 MX CSP 查询设备，可以找到设备的 OSX、MX 和 Android 版本信息。更多信息。

  • OSX 层提供了对基本操作系统的扩展，以实施 Android 未提供的功能。根 OSX 版本号始终与它扩展的 Android 版本的根号码匹配。更多信息。
  • MX 管理系统为设备上的特权和无特权 Android API 提供了一个统一的接口。某些功能需要对 MX 和 OSX 版本进行具体的配对。更多信息。
  • Android 版本有时足以确定功能集。其他情况则需要一个特定的 MX 版本加上 Android 的最低版本。
  • 设备型号可指明在设备上提供 MX 功能的哪个子集。

概述

“可加载”的个人词典 CSP (PersonalDictionary) 用于管理对 Zebra 的企业键盘使用的自定义字词库的更改。这包括添加和删除单个术语（可以是字词或短语），为术语分配快捷方式，批量加载术语以及导出和/或擦除所有已保存的数据。

个人词典可以与企业键盘的自动更正和拼写建议功能结合使用，允许个人词典中保存的术语显示为所键入术语的拼写更正。同样，快捷方式功能允许为术语分配一个或多个字母可自动替换为对应术语的快捷方式。对于所用术语较长或复杂的行业，可提前存储行业术语，帮助提高键盘输入的速度和准确度。例如，如果是配置一个医疗设备，可以将短语“gastroesophageal reflux disease”添加到词典，并分配快捷方式“gerd”，这样每当输入“gerd”时，就会自动替换为较长的术语。

有关更多信息，请参阅企业键盘用户指南。

主要功能

  • 添加术语（字词或短语）
  • 从文件导入术语
  • 将所有存储的术语导出到文件
  • 删除单个术语
  • 删除文件中指定的术语列表
  • 允许/防止重复术语
  • 为术语分配快捷方式
  • 从词典中删除所有术语
  • 基于区域设置维护不同的词典

操作

控制个人词典中术语（字词和/或短语）的添加和删除。可一次一个或成批从设备上存储的文件中添加或删除术语，使用 AddWordsFromFile 和 DeleteWordsFromFile 参数指定。

参数名称：操作

选项 名称 说明需要
1 添加字词 将指定的术语添加到词典中。

MX：6.0+
2 删除字词 从词典中删除指定的术语。

MX：6.0+
3 删除所有字词 从词典中删除所有术语。警告 - 此操作清除所有保存的数据。

MX：6.0+
4 从文件中添加字词 从设备上的文件添加字词到词典。

MX：6.3+
5 从文件中删除字词 从词典中删除与设备上的文件中指定的术语匹配的术语。

MX：6.3+
6 导出字词到文件 将当前词典中的术语保存到设备上的文件。

MX：6.3+

添加字词

用于输入要添加到个人词典的字词或短语。重复使用此参数，在个人词典中预先填充一组字词和/或短语。

参数值输入规则：

  • 接受 1-64 个字符的字符串
  • 一次接受一个字词/短语

在以下条件下显示：操作是“添加字词”

参数名称：AddWord

需要：

  • MX：6.0+

删除字词

用于从个人词典中删除字词或短语。

参数值输入规则：

  • 1 到 64 个字符的字符串

在以下条件下显示：操作是“删除字词”

参数名称：DeleteWord

需要：

  • MX：6.0+

从文件中添加字词

用于指定设备上包含要添加到个人词典的字词（或多字词的术语）的文件的路径和名称。为避免术语重复，请参阅下面的“重复”部分。

参数值输入规则：

  • 接受 1-64 个字符的字符串
  • 必须包含完整路径和文件名（也就是 /storage/sdcard1/NewTerms.txt）
  • 必须是纯文本文件
  • 每个字词（或多字词的术语）必须用单独的一行列出

在以下条件下显示：操作是“从文件中添加字词”

参数名称：AddWordsFromFile

需要：

  • MX：6.3+

重复（复选框）

定义在使用“从文件添加字词”参数时是否允许在个人词典中附加重复的术语。当此框选中时（值=1），无论重复与否，指定文件中的所有术语都附加（添加）到个人词典。当此框未选中时（值=0），在词典中存在的重复术语（与文件中的术语具有相同的拼写和字母大小写）将被替换为文件中的这些术语。

重复检查将区分大小写。这允许某些缩写与它们的通用语言对应词共存。例如，缩写“AD”在医疗职业中有多种用途，但是它小写的对应词“ad”通常用来指广告。这两个术语可能与“Ad”和“aD”变体共存于词典中。一段时间后，此属性可能导致词典包含对应术语的多个重复项。例如，术语“Crohn's disease”和“Crohn's Disease”（大写“D”）具有相同的含义，但是被重复检查功能视为不同。

注意：重复检查延长导入时间。

在以下条件下显示：操作是“从文件中添加字词”

参数名称：重复

选项 名称 说明需要
0 （未选中） 只有当个人词典中没有拼写和大小写相同的术语时，才会将术语添加到个人词典。

MX：6.3+
1 （选中） 所有术语将附加到个人词典，不考虑是否重复。

MX：6.3+

删除文件中的字词

用于指定包含要从个人词典中删除的字词或多字词术语的文件的完整路径和名称。

参数值输入规则：

  • 接受 1-64 个字符的字符串
  • 必须包含完整路径和文件名（也就是 /storage/sdcard1/DeleteTerms.txt）
  • 必须是纯文本文件
  • 每个字词（或多字词的术语）必须用单独的一行列出

在以下条件下显示：操作是“删除文件中的字词”

参数名称：DeleteWordsFromFile

需要：

  • MX：6.3+

导出字词到文件

用于指定要包含从个人词典中导出的术语的文件的完整路径和名称。

参数值输入规则：

  • 接受 1-64 个字符的字符串
  • 必须包含有效路径和文件名（也就是 /storage/sdcard1/ExportedTerms.txt）

在以下条件下显示：操作是“导出字词到文件”

参数名称：ExportWordsToFile

需要：

  • MX：6.3+

快捷方式

用于添加、更改或删除当前位于个人词典中或正要添加到个人词典的术语的快捷方式。在设备键盘上输入时，快捷方式自动替换为它所链接到的术语。这对于存储行业术语可能很有用，可帮助提高键盘输入的速度和准确度。例如，如果要配置医疗职业使用的词典，可以为短语“gastroesophageal reflux disease”指定“gerd”快捷方式。

参数值输入规则：

  • 接受 1-64 个字符的字符串

在以下条件下显示：操作是“添加字词”或“删除字词”

参数名称：快捷方式

需要：

  • MX：6.0+

使用当前区域设置（复选框）

用于确定要据以执行操作的区域设置，如从个人词典中添加或删除字词或短语。选中时（值=1），将使用当前选定的区域设置。未选中时（值=0），可以指定一个区域设置。

参数名称：UseCurrentLocale

选项 名称 说明需要
0 指定区域设置 允许要据以执行操作的区域设置。

MX：6.0+
1 使用当前区域设置 将根据当前选定的区域设置（默认值）执行操作。

MX：6.0+

区域设置

用于输入要据以执行操作（添加字词）的特定区域设置。一个区域设置是语言以及使用该语言的区域的组合。语言使用两个小写字母进行标识；区域使用两个大写字母进行标识 。区域设置使用下划线（“_”）字符将两对字符连起来。.例如，“en_US”是为“美国英语”指定的区域设置，“fr_CA”是为“加拿大法语”指定的区域设置。

有关受支持的区域设置的列表，请参阅 UI Manager。

参数值输入规则：

  • 2 到 10个字符的字符串
  • 必须对应于设备上已安装的区域设置

在以下条件下显示：“使用当前区域设置”为“未选中”（值=0）

参数名称：区域设置

需要：

  • MX：6.0+
