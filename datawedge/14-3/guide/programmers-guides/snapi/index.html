Zebra USB Scanner (SNAPI) Programmer's Guide

DataWedge 14.3

Overview

The Zebra USB Scanner SNAPI (Symbol Native Application Programming Interface) Programmer’s Guide provides developers with instructions on implementing and utilizing the Zebra USB Scanner in SNAPI mode. This mode enables seamless communication between Zebra USB scanners and host devices, enabling advanced data capture capabilities for barcode decoding and image capture via the connected USB scanner. This guide covers setup, configuration, and integration of SNAPI with Zebra Android devices.

Requirements

The following requirements must be met:

  • A device that supports Document Capture; refer to the Feature Matrix
  • A Mobility DNA license to use the Document Capture feature of NG SimulScan
  • Android 13 or higher
  • A camera or supported 2D imager on the device
  • DataWedge v8.0.37 or higher, configured to access NG SimulScan

Configure USB Scanner

To enable SNAPI mode, the Zebra USB Scanner must be configured as follows:

  1. Connect the Scanner - Connect the Zebra USB Scanner to a Zebra Android device via any USB port.
  2. Switch to SNAPI Mode - Scan the SNAPI barcode to change the scanner’s mode: The scanner automatically restarts in SNAPI mode.

Configure DataWedge Profile

A DataWedge profile is required to allow the Zebra USB Scanner to send decoded data to the associated app. The following options must be enabled:

  • SNAPI Capture Mode
  • Intent Output
  • Use Content Providers

If a DataWedge profile has not been created, see Create a New Profile.

Enable SNAPI Capture

SNAPI Capture Mode can be configured either manually or programmatically.

Manually:

  1. Open the DataWedge profile. Under Barcode Input, go to Configure scanner settings > Reader params. Select SNAPI capture mode.
         
  2. Select the appropriate mode:
    • Barcode Decode (Default) - Enables the scanner to decode barcodes, providing standard functionality for Zebra USB Scanners when connected via SNAPI.
    • Image Capture - Allows the scanner to capture images, a feature available only in SNAPI mode and supported by Zebra USB Scanners (SNAPI) equipped with 2D imaging capabilities.

Programmatically: To configure SNAPI mode programmatically, use the Set Config API to set the snapi_capture_mode parameter to the desired mode.

Enable Intent Output

Follow the instructions in Intent Output Setup to:

  • Enable Intent Output - Allows decoded data to be delivered to your application, enabling seamless processing and integration.
  • Enable Use Content Providers - Activates DataWedge's content provider to acquire, process, and output scanned data from the SNAPI Scanner. Follow the Content Provider Programmer's Guide to retrieve and process the acquired data so for application use.
