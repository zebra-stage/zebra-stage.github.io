Document Capture Programmer's Guide

DataWedge 14.3

Overview

Document Capture, a feature of NextGen (NG) SimulScan, is a data capture solution designed to extract information from documents, forms, and labels. It operates by scanning barcodes or capturing images of entire or partial documents. Using NG SimulScan templates, it specifies the data to be captured from barcodes or images and delivers this information to the application for further processing.

Requirements

  • Refer to the Feature Matrtix for devices that support Document Capture
  • A camera or supported 2D imager
  • DataWedge v8.0.37 or higher, configured to access NG SimulScan
  • A Mobility DNA license is required to use the Document Capture feature of NG SimulScan

Document Capture

Document Capture Implementation:

1. Generate the Template

Templates are important for defining the data to be captured and processed. Each template is tailored to a specific document type, such as a shipping label or manufacturing form. It outlines both the Form region (entire document) and the Field regions (specific elements like barcodes, text, or images) for data capture.

For instructions to create the template, see Template Builder.

2. Deploy Template to Device

Deploy the created template to the device (e.g., /sdcard).

3. Transfer Template to Secure Location

Use StageNow with DataWedge Mgr CSP to transfer the template to a secure location for import by DataWedge. Choose one of the following methods to implement this change:

  • Generate a StageNow barcode and scan it with the device
  • Use an Enterprise Mobile Management (EMM) system to deploy the .XML file created via StageNow

For instructions, refer to Import NextGen SimulScan Templates.

4. Create the DataWedge Profile

Create the DataWedge profile and associate it to the application that is receiving the captured data.

5. Configure the Profile

Configure the DataWedge profile for Document Capture:

  1. In the NG SimulScan Configuration, set the Scanning mode to Document Capture.
  2. Select the imported template in Document Selection. The selection is populated after template deployment.

6. Use Content Provider

Use DataWedge's content provider to acquire, process, and output the data from document capture. Enable the Use Content Providers option under Intent Output in the DataWedge profile.

For instructions on retrieving and processing the data for application output, refer to the Content Provider Programmer's Guide .
