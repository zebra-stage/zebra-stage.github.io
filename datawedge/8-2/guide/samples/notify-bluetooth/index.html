Overview

The DataWedge Notify Sample demonstrates how to use the DataWedge Notify API to communicate with the connected Bluetooth scanner to play a notification sound(s) and/or display a colored LED after a scan.

This sample application is intended only for educational purposes, demonstrating the use of DataWedge intent API(s). It is provided as-is without any guarantee.
