MultiActivity

フォアグラウンドのアプリのアクティビティに基づいて、さまざまなプロファイルを使用して複数のアクティビティにわたってスキャンする方法を示します。

製品

DataWedge - 11.0

デバイス

Zebra Android Marshmallow (v6.0) 以降のサポートされているすべてのデバイス

概要

このサンプル アプリでは、フォアグラウンドのアプリのアクティビティに基づいて、さまざまなプロファイルを使用して複数のアクティビティにわたってスキャンする方法を示します。アプリには 2 つのアクティビティがあり、それぞれに個別の DataWedge プロファイルが作成されます。各プロファイルは、それぞれのアクティビティに関連付けられます。Profile1 は MainActivity に関連付けられています。これにより、Code128 が有効になり、EAN13 が無効になります。Profile2 は SecondActivity に関連付けられています。これにより、Code128 が無効になり、EAN13 が有効になります。アクティビティがフォアグラウンドになると、対応する DataWedge プロファイルがロードされ、それぞれのバーコードをスキャンできます。ボタンをタップして、両方のアクティビティ間を移動します。各アクティビティには、アクティブ プロファイル名が表示されます。バーコードをスキャンすると、スキャン データとバーコードのシンボル体系が表示されます。

一般的な使用事例:

  • マルチアクティビティ アプリケーションのフォアグラウンド アクティビティに基づいて、さまざまなバーコードをスキャンする。
  • フォアグラウンドでのアクティビティに基づいて、データ操作などのさまざまなスキャナ構成を指定する。

このサンプル アプリは、以前のブログ記事「Using DataWedge with Multiple Android Activities」に基づいています

注: このアプリケーションは、デモンストレーションのみを目的としています。このデモは、保証なしで現状のまま提供され、個々のニーズに合わせて変更できます。サンプル アプリ画面の外観は、サンプル アプリのバージョン、Android のバージョン、および画面サイズによって異なります。

使用している API

DataWedge API アプリの機能
構成の設定 既存のプロファイルをチェックし、存在しない場合は作成して設定を構成します。プロファイル「MultiActivity_Profile1」の場合は、これを MainActivity に関連付けて、Code128 を有効にして EAN13 を無効にします。プロファイル「MultiActivity_Profile2」の場合は、これを SecondActivity に関連付けて、Code128 を無効にして EAN13 を有効にします。
アクティブ プロファイルの取得 現在使用されているプロファイルの名前を取得します。

要件

  • このサンプルは、DataWedge バージョン 7.6 (作成時に存在していたバージョン) に基づいて作成されています。機能に影響を与える可能性のある、今後のバージョンでの変更については、DataWedge API ドキュメントを確認してください。API がサポートされている場合は、古いバージョンの DataWedge でも機能するはずです。
  • Android API 23 (Marshmallow) 以降
  • DataWedge (すべての Zebra デバイスに組み込み済み)

このサンプルの使用

  1. サンプル アプリをダウンロード、ビルド、および起動します。プロファイルが作成されたことを示すトースト通知メッセージが表示されます。

  2. DataWedge を開いて、2 つのプロファイル「MultiActivity_Profile1」と「MultiActivity_Profile2」が作成されていることを確認します。

  3. サンプル アプリをもう一度開きます。アクティブ プロファイル「MultiActivity_Profile1」を示すメイン アクティビティが表示されます。メイン アクティビティで、EAN13 バーコードをスキャンします。プロファイルでバーコードが有効になっていないため、バーコードをスキャンできません。

  4. メイン アクティビティで、Code128 バーコードをスキャンします。スキャン データとデコーダが表示されます。

  5. [2 番目のアクティビティに移動] をタップします。2 番目のアクティビティが表示され、アクティブ プロファイル「MultiActivity_Profile2」が示されます。

  6. 2 番目のアクティビティで、Code128 バーコードをスキャンします。プロファイルでバーコードが有効になっていないため、バーコードをスキャンできません。

  7. 2 番目のアクティビティで、EAN13 バーコードをスキャンします。スキャン データとデコーダが表示されます。
