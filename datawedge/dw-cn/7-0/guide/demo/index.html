DataWedge 7.0

DataWedge 演示应用程序

使用下列各项可演示和测试 DataWedge 功能：

  • DWDemo - 所有 Zebra 设备随附，在下面说明
  • DataWedge API 示例应用程序 - 可下载的应用程序演示如何使用 DataWedge API。
  • “厨房水槽”应用程序 - 可下载且可自定义的应用程序，在下载页面上说明
  • “示例 Intent”应用程序 - 可下载的应用程序显示如何通过 Intent 接收扫描的条码数据。

DWDemo 显示应用程序如何使用 DataWedge 服务获取数据。名为“DWDemo”的 DataWedge 配置文件与 DataWedge 一起安装并与演示应用程序相关联。默认情况下，该配置文件处于禁用状态，可根据需要修改以用于测试和演示。启用演示配置文件之后（请参阅以下内容），按下应用程序的“扫描”按钮或设备触发器会启动条码扫描，并在屏幕上显示已解码的数据。如果一个设备已连接，则 DWDemo 应用程序支持使用成像器、摄像头、蓝牙设备或磁条读取器 (MSR) 进行扫描。

本指南需要 DataWedge 配置文件的工作知识。有关配置文件设置的更多信息，请参阅 DataWedge 设置指南。

DWDemo 应用程序会出现在“Android 应用程序抽屉”中。

设备选择器

DWDemo 应用程序提供可用扫描输入设备的菜单，使得可以轻松地从一个设备切换到另一个设备以进行测试。还可在 DWDemo 配置文件中配置这个和其他 DataWedge 设置和首选项，如下所示。有关配置文件设置的信息，请参阅 DataWedge 设置指南。


设备控件（从左到右）：

  • 闪电 - 打开和关闭“摄像头闪光灯”（选择“摄像头”时处于活动状态）；控制其他设备的照明模式。
  • 输入设备 - 允许选择“摄像头”、“扫描器”或“蓝牙”设备（如果未连接“蓝牙”设备，则选择“蓝牙”将尝试配对）。
  • 摄像头输入类型 - 在图像和条码数据捕获之间切换（选择“摄像头”时处于活动状态；因设备而异）。
  • DWDemo“汉堡”菜单 - 将调用“关于”面板和 DWDemo 配置文件（请参见以下内容）。
