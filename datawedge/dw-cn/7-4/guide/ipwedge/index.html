IPWedge

DataWedge 7.4

概述

IPWedge 是一种小型的 Windows 应用程序，该应用程序使用 DataWedge IP 输出插件监控网络端口并接收设备发送的已扫描数据。接收 PC 将数据作为按键插入到前台应用程序或 Windows 剪贴板，实质上是将设备用作无线扫描器。

设置

前提条件

使用 IPWedge 需要满足以下要求：

  • 使用 Windows XP、Vista 或 7 的 PC
  • 安装了 Microsoft .NET Framework 3.5 Service Pack 1（或更高版本）
  • 运行 Android 4.4 KitKat（或更高版本）的 Zebra 设备
  • DataWedge for Android 1.5（或更高版本）

设置 IPWedge

1.访问 Zebra 支持门户，在搜索框中输入“IPwedge”，并下载适合目标设备的版本。

2.在要将已扫描数据发送到的系统上安装 .zip 文件。

3.运行 IPWedge 应用程序。将出现类似于下面图像的屏幕。记下 IP 地址和端口号（红色框中）。IPWedge“系统托盘”菜单。

注意：此默认端口号与设备上 IP 输出插件的相同。

4.从 Windows“开始”菜单打开 IPWedge 应用程序，或在“系统托盘”中轻触 IPWedge 图标，然后选择“选项”。

将显示类似于下面图像的 IPWedge“选项”面板。 常规选项。

按键选项。

剪贴板选项。

5.在“常规”、“按键”和“剪贴板”选项卡中对“选项”做出任何所需更改。在每个选项卡上单击“确定”以保存设置。

PC 现已准备就绪，可从运行 DataWedge（带 IP 输出插件，其 IP 地址和端口设置与 PC 的相匹配）的设备接收数据。有关设备设置的说明，请参阅 DataWedge 设置指南中的“IP 输出”。
